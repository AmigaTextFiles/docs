

 ===================================
 === INSTALLATION SUR DISQUE DUR ===
 ===================================


 - Créer un répertoire 'Jarre'
 - Copier les tiroirs Images,Samples,Textes,MIDI dans 'Jarre'
 - Copier le Jarre.Guide_Démo (+ .info) dans 'Jarre'
 - Ajouter à la User-Startup les lignes suivantes

  ;BEGIN Jarre Guide
  Assign Jarre: 'Path qui mene à votre tiroir Jarre'
  ;END Jarre Guide

 - Installer la font Freddy dans votre Fonts:
 - Assurez vous d'avoir 'Multiview dans votre 'Workbench:Utilities/'
 - Assurez vous d'avoir 'Vt' (View-Tek) dans votre 'Workbench:Utilities/'
 - Placer le programme 'Player' (du tiroir tools) dans votre tiroir C:

 Apres cela vous n'aurez plus qu'à double cliquer sur Jarre.Guide_Démo !!
