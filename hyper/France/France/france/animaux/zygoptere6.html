L'agrion élégant : (Ischnura elegans)

         Longueur :  
        
        Gestation : 

        Longévité :
Description :



Espèce à abdomen noir, sauf l'avant dernier segment abdominal (le huitième) est bleu. Les yeux sont tachés de bleu chez le mâle comme chez la femelle, sauf pour la forme de type rose. Les ailes sont transparentes, sauf le ptérostigma qui est noir. Peut-être confondue avec Erythroma najas et Erythroma viridulum.
