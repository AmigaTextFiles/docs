 (sorti en 1985)

Microprocesseur : 68000 à 8 MHz

Composant graphique : Shiffer (nombre maximum de couleurs : 512 )
Affichage texte : 25 lignes/40 colonnes, 25 lignes/80 colonnes, 50 lignes/80 colonnes
Affichage graphique : 320 x 200 (16 couleurs), 640 x 200 (4 couleurs), 640 x 400 (monochrome)
Composant sonore : YM 2149, Son stéréo (3 voix + Bruit)
ports de jeux : 2 (1 manette/souris et 1 manette)
Mémoire de masse : lecteur de disquette 3"1/2 (360 Ko).Absent sur les 1ère versions)
port RS232 : 1
port parallèle : 1
Connecteur moniteur mono/TV : 1
Connecteur DD atari : 1
prises Midi : 2
port cartouche : 1 (128ko)
