Comment intégrer un bouton appelant directement miami ?
1- choisissez réglages dans le menu réglages.
2- cliquez 2 fois sur interfaces, un sous-menu apparaît.
3- choisissez barre d'outils
4- Attrapez le bouton custom dans le haut de l'écran et insérez le en le lâchant sur la barre d'icônes en bas de l'écran.
5- changez le nom de l'étiquette (custom) exemple: connexion.
6- changez le type d'action,choisissez Arexx et indiquez le chemin d'accès de Yam sur votre disque dur.suivi de /rexx/miamionline.yam
exemple: work:internet/yam/rexx/miamionline.yam
7- cliquez sur OK.
8- Sauvez les réglages dans le menu réglages.

Comment intégrer un bouton appelant Yam ?
Même opération que ci-dessus jusqu'au 4.puis,
5- Changez le nom de l'étiquette (custom) exemple: Courrier
6- Changez le type d'action, choisissez command et indiquez le chemin d'accès de Yam sur votre disque dur.
exemple: Work:/internet/yam/yam

Comment lancer miami (le programme de connection) automatiquement,dès le démarrage de yam ?
Cliquez sur l'icône configuration,cliquez sur arexx,et sur la fonction démarrage,de là,dans script,indiquer où se trouve miami sur votre disque dur.
