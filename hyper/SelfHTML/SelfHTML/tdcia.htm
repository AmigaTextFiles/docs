Beschreibung: so wird's gemacht

Zweispaltiger Text

Jedes Computer-Programm, jede Anwendung, auch Ihre Textverarbeitung, Ihr Dateimanager und Ihr WWW-Browser, besteht aus einer kontrollierten Abfolge von Anweisungen, die vom Computer ausgeführt werden. Die Anweisungen, die ein Computer versteht, sind für Menschen jedoch ziemlich ungenießbar. Moderne Programmiersprachen sind daher sogenannte Hochsprachen, die aus menschenlesbarem Klartext bestehen. Die darin formulierten Anweisungen werden erst beim Compilieren oder beim Interpretieren in computergerechte Befehle übersetzt. Wie in vielen anderen Bereichen ist Englisch die übliche Sprache für Programmier-Hochsprachen. Aber keine Sorge, Sie müssen keinen Englischkurs besuchen, bevor Sie eine Sprache wie JavaScript lernen - das Englisch, auf das sich solche Sprachen beschränken, besteht aus einigen wenigen Wörtern, die Sie schnell lernen, falls Sie sie noch nicht kennen. Daneben gibt es viele Namen, die Sie selbst vergeben dürfen, und dabei können Sie selbstverständlich auch deutsche Wörter verwenden.
