Lauftext definieren

Anzeigebeispiel: So sieht's aus

Sie können erreichen, daß ein Text mitten im übrigen Text automatisch horizontal scrollt, so wie in dem von Windows bekannten Bildschirmschoner "Marquee". Diese Angaben werden bislang jedoch nur vom MS Internet Explorer ab Version 2.0 interpretiert und gehören nicht zum HTML-Standard.

Beispiel:

<marquee>Dieser Text wird horizontal wie am Laufband gescrollt</marquee>

Erläuterung:

Mit <marquee> leiten Sie den zu scrollenden Text ein. Dahinter folgt der Text, der gescrollt werden soll. Am Ende des zu scrollenden Textes notieren Sie ein abschließendes </marquee>.

 

Laufrichtung bestimmen

Anzeigebeispiel: So sieht's aus

Per Voreinstellung werden Marquee-Texte vom Internet Explorer von rechts nach links bewegt. Sie können die Laufrichtung umkehren.

Beispiel:

<marquee direction=right>Dieser Text wird von links nach rechts gescrollt</marquee>

Erläuterung:

Durch das Attribut direction=right im einleitenden Tag von <marquee ... > erzwingen Sie die Laufrichtung von links nach rechts (direction = Richtung, right = rechts). Mit dem Attribut direction=left stellen Sie die Laufrichtung auf rechts nach links (Voreinstellung).

 

Laufgeschwindigkeit einstellen

Anzeigebeispiel: So sieht's aus

Der Internet Explorer verwendet intern eine bestimmte Laufgeschwindigkeit. Sie können eine eigene Geschwindigkeit bestimmen. Die Geschwindigkeit können Sie durch zwei Parameter beeinflussen: 1. durch die Anzahl der Pixel zwischen einem Bewegungszustand und dem nächsten, und 2. durch die Angabe der Verzögerung zwischen zwei Bewegungszuständen.

Beispiel:

<marquee scrollamount=50 scrolldelay=5>Dieser Text wird ziemlich schnell gescrollt</marquee>

Erläuterung:

Durch das Attribut scrollamount= im einleitenden Tag von <marquee ... > bestimmen Sie die Pixelanzahl zwischen den Scroll-Zuständen. Durch das Attribut scrolldelay= legen Sie die Verzögerung zwischen zwei Scroll-Zuständen in Millisekunden fest (amount = Zustand, delay = Verzögerung).

 

Hin und her wechselnder Lauftext

Anzeigebeispiel: So sieht's aus

Sie können bestimmen, daß der sich der Marquee-Text abwechselnd von rechts nach links und von links nach rechts bewegt. Dabei wird die Laufrichtung des Textes umgekehrt, sobald der Text einen der beiden Fenster- bzw. Seitenränder berührt.

Beispiel:

<marquee behavior=alternate>Dieser Text wird hin- und herbewegt</marquee>

Erläuterung:

Durch das Attribut behavior=alternate im einleitenden Tag von <marquee ... > erzwingen Sie den Wechsel-Effekt (behavior = Verhalten, alternate = wechselnd).

 

Lauftext zusätzlich hervorheben

Anzeigebeispiel: So sieht's aus

Sie können den als Marquee-Effekt definierten Textabschnitt zusätzlich hervorheben, indem Sie eine eigene Hintergrundfarbe für diesen Textabschnitt definieren. Der Scroll-Effekt findet dann optisch gesehen in einem andersfarbigen Kasten statt.

Beispiel:

<font color=#FFFF00><marquee bgcolor=#990099>
Dieser Scroll-Text erscheint gelb auf knallrotem Hintergrund
</marquee></font>

Erläuterung:

Durch das Atrribut bgcolor= im einleitenden Tag von <marquee ... > erzwingen Sie, daß der gescrollte Text in einem andersfarbigen Rahmen erscheint (bgcolor = background color = Hintergrundfarbe). Für die Farbauswahl gelten die Regeln zum  Definieren von Farben in HTML.

Beachten Sie:

Wenn Sie eine Hintergrundfarbe für den Lauftext, sollten Sie auch eine dazu kontrastierende Vordergrundfarbe für den Text bestimmen. Dies erreichen Sie durch Bestimmen der  Schriftfarbe mit <font color=>. Das <font>-Tag sollte dabei außerhalb des <marquee>-Tags stehen, andernfalls interpretiert der MS Internet Explorer es falsch.

Die definierten Farben sollten ferner zu den dateiweiten  Hinter- und Vordergrundfarben passen.

 

Umgebenden Text ausrichten

Anzeigebeispiel: So sieht's aus

Sie können bestimmen, daß Text, der im gleichen Absatz wie der Marquee-Text steht, im Verhältnis zu dem Marquee-Text ausgerichtet wird.

Beispiel:

<marquee align=top>Marquee-Text</marquee> und oben daran ausgerichteter Text
<marquee align=middle>Marquee-Text</marquee> und mittig daran ausgerichteter Text
<marquee align=bottom>Marquee-Text</marquee> und unten daran ausgerichteter Text
<marquee height=20 hspace=10>Marquee-Text</marquee>

Erläuterung:

Durch das Atrribut align=top im einleitenden Tag von <marquee ... > erreichen Sie, daß der umgebende Text obenbündig zu dem Marquee-Text ausgerichtet wird. Mit align=middle wird der umgebende Text mittig ausgerichtet, mit align=bottom untenbündig (align = Ausrichtung, top = oben, middle = mittig, bottom = unten).

Durch die Zusatzangabe height= im einleitenden Tag von <marquee ... > bestimmen Sie die Höhe des Marquee-Bereichs (height = Höhe). Diese Angabe ist vor allem interessant, wenn Sie für den Marquee-Bereich eine eigene Hintergrundfarbe bestimmen. Mit der Höhe des Marquee-Bereichs legen Sie dann die Höhe des andersfarbigen Kastens fest, in dem der Marquee-Text gescrollt wird. Die Angabe erfolgt entweder absolut in Pixeln (wie im Beispiel), oder in Prozent relativ zur Höhe des Anzeigefensters. Um einen Prozentwert anzugeben, hängen Sie hinter die Wertangabe einfach ein Prozentzeichen % (z.B. height=10%).

Mit hspace= bestimmen Sie den Abstand zwischen Marquee-Text und dem "normalen" Text, der unmittelbar davor bzw. dahinter folgt (hspace = horizontal space = horizontaler Abstand). Diese Angabe ist nur von Bedeutung, wenn Sie den Marquee-Bereich mitten im Text definieren. Wenn Sie den Marquee-Bereich in einem eigenen Absatz definieren, ist die Angabe bedeutungslos. Die Angabe des Abstands zum Text davor bzw. dahinter erfolgt in Pixeln. Der angegebene Abstand gilt sowohl für den Text davor als auch für den Text dahinter.
