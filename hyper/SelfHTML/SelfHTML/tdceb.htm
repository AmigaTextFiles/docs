Hintergrundbild


Beschreibung: so wird's gemacht

Textabsatz mit eigenem Hintergrundbild (Wallpaper). In herkömmlichem HTML waren solche Hintergrundbilder nur auf ganze Dateien, später auch auf Tabellenelemente anwendbar. Achten Sie bei Verwendung von Wallpapers immer auch darauf, daß der darauf erscheinende Text eine gute Kontrastfarbe zum Hintergrund hat.

Eine Überschrift

Ein Textabsatz. Um ein einheitliches Hintergrundbild für mehrere verschiedene Elemente zu definieren, schließen Sie die gewünschten Elemente einfach in die HTML-Tags <div>...</div> ein und definieren Sie für das <div>-Tag die gewünschten Style-Sheet-Angaben.
