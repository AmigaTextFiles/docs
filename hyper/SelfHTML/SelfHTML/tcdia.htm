HTML-Tags für logische Textauszeichnung


Beschreibung: so wird's gemacht

<em>....</em> bedeutet "betont (emphatisch)". Und so wie dieser Text sieht's bei Ihnen aus.
<strong>....</strong> bedeutet "stark betont". Und so wie dieser Text sieht's bei Ihnen aus.
<code>....</code> bedeutet "dies ist Quellcode". Und so wie dieser Text sieht's bei Ihnen aus.
<samp>....</samp> bedeutet "dies ist ein Beispiel". Und so wie dieser Text sieht's bei Ihnen aus.
<kbd>....</kbd> bedeutet "dies ist eine Tastatureingabe". Und so wie dieser Text sieht's bei Ihnen aus.
<var>....</var> bedeutet "dies ist eine Variable". Und so wie dieser Text sieht's bei Ihnen aus.
<cite>....</cite> bedeutet "dies ist ein Zitat aus einer anderen Quelle". Und so wie dieser Text sieht's bei Ihnen aus.
<dfn>....</dfn> bedeutet "dies ist eine Begriffsdefinition". Und so wie dieser Text sieht's bei Ihnen aus.
<acronym>....</acronym> bedeutet "dies ist eine Abkürzung". Und so wie dieser Text sieht's bei Ihnen aus.
<q cite=URL>....</q> bedeutet "Dies ist ein Zitat von URL". Und so wie dieser Text sieht's bei Ihnen aus.
