Browser-interne Verweise definieren

Die folgenden Spezial-Referenzen werden nur von Netscape interpretiert. Es handelt sich zum Teil um nützliche, zum Teil um weniger nützliche Verweise einige Verweise führen zu internen, in der Browser-Software gespeicherten Zielen, andere zu Zielen im WWW.

Verweise zu software-internen Zielen

 javascript: - ruft eine interne Browser-Seite von Netscape auf, in der man JavaScripts Zeile für Zeile interpretieren und auf Richtigkeit prüfen lassen kann. Als Befehl in HTML:
<a href="javascript:">Verweistext</a>

 livescript: - ruft eine interne Browser-Seite von Netscape auf, in der man LiveScripts (Vorgänger-Version von JavaScript) Zeile für Zeile interpretieren und auf Richtigkeit prüfen lassen kann. Als Befehl in HTML:
<a href="livescript:">Verweistext</a>

 mocha: - ruft eine interne Browser-Seite von Netscape auf, in der man Mocha-Code Zeile für Zeile interpretieren und auf Richtigkeit prüfen lassen kann. Als Befehl in HTML:
<a href="mocha:">Verweistext</a>

 about: - ruft die interne Browser-Seite mit Versionshinweisen auf. Als Befehl in HTML:
<a href="about:">Verweistext</a>

 about:blank - ruft eine leere interne Browser-Seite auf (auch beim MS Ineternet Explorer 4). Als Befehl in HTML:
<a href="about:blank">Verweistext</a>

 about:document - ruft eine interne Browser-Seite auf, die Informationen zum aktuell angezeigten Dokument enthält. Als Befehl in HTML:
<a href="about:document">Verweistext</a>

 about:fonts - ruft eine interne Browser-Seite auf, die Informationen zu downloadbaren Schriftarten enthält (nur ab Netscape 4.x). Als Befehl in HTML:
<a href="about:fonts">Verweistext</a>

 about:global - ruft eine interne Browser-Seite auf, die den lokalen History-Cache-Speicher (ab Netscape 4.x mit anklickbaren Verweisen) darstellt. Als Befehl in HTML:
<a href="about:global">Verweistext</a>

 about:mozilla - ruft eine nicht ganz so ernstzunehmende, interne Browser-Seite von Netscape auf. Als Befehl in HTML:
<a href="about:mozilla">Verweistext</a>

 about:logo - ruft eine interne Browser-Seite auf, die das Netscape-Logo in voller Größe darstellt. Als Befehl in HTML:
<a href="about:logo">Verweistext</a>

 about:plugins - ruft eine interne Browser-Seite auf, in der angezeigt wird, welche Netscape-Plugins (z.B. Live-Chat) beim Anwender installiert sind. Wenn keine Pugins installiert sind, bleibt die Seite leer. Als Befehl in HTML:
<a href="about:plugins">Verweistext</a>

 about:license - erzeugt eine interne Browser-Seite, in der die Lizenzvereinbarungen angezeigt werden, die beim Erstaufruf von Netscape einmalig erscheinen. Als Befehl in HTML:
<a href="about:license">Verweistext</a>

 about:rsalogo - ruft eine interne Browser-Seite auf, die das RSA-Logo für den von Netscape unterstützten Security-Standard darstellt. Als Befehl in HTML:
<a href="about:rsalogo">Verweistext</a>

 about:javalogo - (ab Netscape 4.x) ruft eine interne Browser-Seite auf, die das Java-Logo darstellt. Als Befehl in HTML:
<a href="about:javalogo">Verweistext</a>

 about:coslogo - (ab Netscape 4.x) ruft eine interne Browser-Seite auf, die das Logo des Cosmo-Players (VRML) darstellt. Als Befehl in HTML:
<a href="about:coslogo">Verweistext</a>

 about:visilogo - (ab Netscape 4.x) ruft eine interne Browser-Seite auf, die das Logo von Visigenic darstellt. Als Befehl in HTML:
<a href="about:visilogo">Verweistext</a>

 about:odilogo - (ab Netscape 4.x) ruft eine interne Browser-Seite auf, die das Logo von Object Design darstellt. Als Befehl in HTML:
<a href="about:odilogo">Verweistext</a>

 about:mclogo - (ab Netscape 4.x) ruft eine interne Browser-Seite auf, die das Logo von Marimba Castanet darstellt. Als Befehl in HTML:
<a href="about:mclogo">Verweistext</a>

 about:ncclogo - (ab Netscape 4.x) ruft eine interne Browser-Seite auf, die das Logo von Netcast darstellt. Als Befehl in HTML:
<a href="about:mclogo">Verweistext</a>
