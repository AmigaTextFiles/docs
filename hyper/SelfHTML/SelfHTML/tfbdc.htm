Unabhängige Formate mit JSSS Style-Sheets definieren


Beschreibung: so wird's gemacht

Dies ist ein Text, der in die Tags <p>...</p> eingeschlossen ist. Im einleitenden <p>-Tag wurde jedoch das Attribut id="rot" notiert. Zuvor wurde mit einer JSSS-Definition ein Format "rot" definiert, und zwar mit der Angabe:
ids.rot.color = "red";

Dies ist ein Text, der in die Tags <p>...</p> eingeschlossen ist. Alle Elemente innerhalb dieses Absatzes, die mit <tt>...</tt> formatiert sind, haben jedoch im einleitenden <tt>-Tag das Attribut id="hervorgehoben" erhalten. Zuvor wurde mit einer JSSS-Definition ein Format "hervorgehoben" definiert, und zwar so:

 with(ids.hervorgehoben)
  {
    backgroundColor = "#CCFFFF";
    fontWeight = "bold";
   }
