Verweis zu E-Mail-Adresse setzen

Anzeigebeispiel: So sieht's aus

Sie können auf jede beliebige E-Mail-Adresse im Internet-Format einen Verweis setzen. Eine Internet-gerechte E-Mail-Adresse erkennen Sie an dem Zeichen @ in der Mitte der Adresse. Wenn der Anwender auf den Verweis klickt, kann er eine E-Mail an den betreffenden Empfänger absetzen. Normalerweise benutzen Anbieter von WWW-Seiten diese Möglichkeit, um Besuchern die Möglichkeit zu bieten, eine E-Mail an die eigene Adresse senden, zum Beispiel wegen Feedback zum Angebot. Sie können aber auch Verweise zu anderen E-Mail-Adressen anbieten.

Beispiel:

<a href="mailto:muenz@csi.com">Mail an den Autor, muenz@csi.com</a>

Erläuterung:

Für das Setzen des Verweises gilt das  Schema für Verweise in HTML. Das Verweisziel ist die E-Mail-Adresse.

Beachten Sie:

Verweise zu E-Mail-Adressen beginnen immer mit mailto: (ohne // dahinter!).

Der Verweis ist bei Anwendern nur ausführbar, wenn der WWW-Browser das Erstellen und Absenden von E-Mails unterstützt (z.B. Netscape), oder wenn bei solchen Verweisen automatisch ein E-Mail-Programm gestartet wird (dies ist z.B. beim MS Internet Explorer nötig).

Nennen Sie im Verweistext möglichst noch einmal die E-Mail-Adresse explizit, damit Anwender, bei denen der E-Mail-Verweis nicht ausführbar ist, auf Wunsch separat eine E-Mail senden können.

 

E-Mail-Verweise an mehrere Empfänger

Anzeigebeispiel: So sieht's aus

Sie können einen E-Mail-Verweis notieren, bei dessen Anklicken der Anwender eine E-Mail versenden kann, die an zwei oder mehrere Empfänger verschickt wird. Dabei können Sie sichtbare Kopienempfänger (cc-Mail) oder unsichtbare Kopienempfänger (bcc-Mail) angeben.

Beispiele:

<a href="mailto:muenz@csi.com?cc=s.muenz@euromail.com">Mail an den Autor</a>

<a href="mailto:muenz@csi.com?bcc=s.muenz@euromail.com;muenz@teamone.de">
Mail an den Autor
</a>

Erläuterung:

Formulieren Sie zunächst einen einfachen   E-Mail-Verweis. Notieren Sie direkt hinter der Hauptempfängeradresse (in den obigen Beispielen: muenz@csi.com) ein Fragezeichen und wieder direkt dahinter cc= (für sichtbare Kopienempfänger) oder bcc= für unsichtbare Kopienempfänger. Dahinter folgen eine oder mehrere Adressen. Falls Sie mehrere Kopienempfänger angeben, trennen Sie die Mail-Adressen der Empfänger jeweils durch ein Semikolon, so wie im zweiten der obigen Beispiele.

Beachten Sie:

Diese Möglichkeiten gehören nicht zum HTML-Standard, sind jedoch vom W3-Konsortium als software-abhängiges Feature geduldet.

 

E-Mail-Verweise mit vordefiniertem "subject"

Anzeigebeispiel: So sieht's aus

Jede E-Mail hat ein "Subject", d.h. einen Titel. Bei einem E-Mail-Verweis können Sie einen E-Mail-Titel vordefinieren. Das ist z.B. sinnvoll, wenn der E-Mail-Verweis eine bestimmte Funktion haben soll, etwa die, Feedback zum eigenen Angebot einzuholen.

Beispiel:

<a href="mailto:muenz@csi.com?subject=Feedback zu SELFHTML">Mail an den Autor</a>

Erläuterung:

Formulieren Sie zunächst einen einfachen   E-Mail-Verweis. Notieren Sie direkt hinter der Empfängeradresse (in den obigen Beispielen: muenz@csi.com) ein Fragezeichen und wieder direkt dahinter subject= zum Vordefinieren des Subjects. Dahinter geben Sie den gewünschten Text für den Titel der E-Mail an.

Beachten Sie:

Diese Möglichkeiten gehören nicht zum HTML-Standard, sind jedoch vom W3-Konsortium als software-abhängiges Feature geduldet.

 

E-Mail-Verweise mit vordefiniertem "body"

Anzeigebeispiel: So sieht's aus

Bei einem E-Mail-Verweis können Sie einen Text für den Mail-Inhalt vordefinieren. Das ist z.B. sinnvoll, wenn der E-Mail-Verweis eine bestimmte Funktion haben soll, etwa die, Feedback zum eigenen Angebot einzuholen.

Beispiel:

<a href="mailto:muenz@csi.com?body=Mein Feedback:%0Abitte hier eingeben">Mail an den Autor</a>

Erläuterung:

Formulieren Sie zunächst einen einfachen   E-Mail-Verweis. Notieren Sie direkt hinter der Empfängeradresse (in den obigen Beispielen: muenz@csi.com) ein Fragezeichen und wieder direkt dahinter body= zum Vordefinieren des Mail-Inhalts. Dahinter geben Sie den gewünschten Text für den Inhalt der E-Mail an.

Beachten Sie:

Wenn Sie innerhalb des Mail-Textes einen Zeilenumbruch wünschen, müssen Sie an der betreffenden Stelle die Zeichenfolge %0A notieren, so wie im obigen Beispiel.

Diese Möglichkeiten gehören nicht zum HTML-Standard, sind jedoch vom W3-Konsortium als software-abhängiges Feature geduldet.

 

Vordefinierte Angaben kombinieren

Anzeigebeispiel: So sieht's aus

Sie können Vordefinitionen kombinieren, also beispielsweise mehrere Empfänger und ein vordefiniertes Subject angeben.

Beispiel:

<a href="mailto:muenz@csi.com?cc=s.muenz@euromail.com&subject=Hallo Stefan&body=Hier schreiben">
Mail</a>

Erläuterung:

Formulieren Sie zunächst einen einfachen  E-Mail-Verweis. Notieren Sie direkt hinter der Empfängeradresse (in den obigen Beispielen: muenz@csi.com) ein Fragezeichen und wieder direkt dahinter die erste gewünschte vordefinierte Angabe, also cc=, bcc=, subject= oder body=. Dahinter folgt ein entsprechender Wert, also die gewünschte Mail-Adresse für einen zweiten Empfänger, das Subject oder der gewünschte vordefinierte Text der E-Mail. Vor jeder weiteren vordefinierten Angabe müssen Sie dann ein kaufmännisches Und & (also kein weiteres Fragezeichen!) notieren.

Die Reihenfolge der Angaben ist egal. Für die mögliche Anzahl so kombinierter Angaben gibt es auch keine vorgeschriebene Grenze. Angaben zu subject= und body= sollten natürlich nur einmal vorkommen.
