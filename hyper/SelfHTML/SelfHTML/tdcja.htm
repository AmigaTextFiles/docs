Beschreibung: so wird's gemacht

Diese Seite erscheint an Ihrem Bildschirm genau so wie alle anderen Seiten dieser Dokumentation. Wenn Ihr Browser jedoch die Style-Sheet-Angaben zum Seitenlayout kennt, versuchen Sie mal, diese Datei über den normalen Druckbefehl Ihres WWW-Browsers auszudrucken. Im Kopf dieser HTML-Datei wurde mit der Angabe @page { size:landscape; } festgelegt, daß der Ausdruck querkant erfolgen soll.
