Grafik auf einem anderen WWW-Server referenzieren


Beschreibung: so wird's gemacht

Viele Anwender lesen dieses Dokument offline. Zum Referenzieren einer Grafikdatei auf einem anderen Internet-Server ist eine Online-Verbindung erforderlich. Wenn Sie die Verbindung aufgebaut haben oder aufbauen möchten,  besuchen Sie die Extra-Datei.
