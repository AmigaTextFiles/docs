Quickbar verwenden

Die SELFHTML-Quickbar ist eine kleine Leiste im oberen Bereich des Anzeigefensters, die dauerhaft eingeblendet bleibt. Über die Quickbar können Sie aus Aufklapplisten schnell einen Befehl oder ein Stichwort auswählen, nach dem Sie suchen. Sie springen dann gleich zu dem Dokumentabschnitt, in dem der betreffende Befehl beschrieben bzw. das Stichwort behandelt wird.

Die Quickbar ist Frame- und JavaScript-basiert. Sie funktioniert mit Netscape ab Version 3.x und mit dem MS Internet Explorer ab Version 4.x. Mit MS Internet Explorer 3.x gibt es Probleme, die jedoch in Kauf genommen wurden. Andere Browser wurden nicht getestet. Falls die Quickbar bei Ihnen nicht funktioniert, kann es daran liegen, daß Sie in Ihrem Browser JavaScript nicht aktiviert haben. Wenn die Auswahl aus den Aufklapplisten Fehlermeldungen erzeugt, kommt Ihr Browser mit der Quickbar nicht zurecht. Benutzen Sie in diesem Fall die Verzeichnisse, die über die Einstiegsseite von SELFHTML als normale Seiten aufrufbar sind, das sind hier das  Befehlsverzeichnis und das  Stichwortverzeichnis.

Wählen Sie aus der links oben angezeigten Aufklappliste zunächst einen Eintrag aus. Je nachdem, was Sie ausgewählt haben, wird rechts davon eine andere Aufklappliste angezeigt. Wenn Sie aus so einer zweiten Aufklappliste einen Eintrag auswählen, springen Sie direkt zu der entsprechenden Stelle im Dokument.
