Rahmen um Objekte


Beschreibung: so wird's gemacht

Die eingebettete Datei über diesem Text ist eine MIDI-Datei. Das ist ein Format für Musik, die mit Synthesizern über eine entsprechende PC-Schnittstelle direkt in solche Dateien gespielt und nachbearbeitet werden kann. MIDI-Dateien haben den Vorteil, daß sie im Verhältnis zu der Länge der Musik vergleichsweise klein sind.

Wenn Sie ein Plugin für Midi-Dateien haben, stellt der WWW-Browser einen Midi-Player im Anzeigefenster dar. Um den Player wurde ein Rahmen von 5 Pixel definiert.

Wenn bei Ihnen nichts angezeigt wird, kann das daran liegen, daß Ihr WWW-Browser das <object>-Tag noch nicht oder noch nicht vollständig interpretiert. Probieren Sie in diesem Fall das  Einbinden von Multimedia nach herkömmlicher Netscape-Syntax.
