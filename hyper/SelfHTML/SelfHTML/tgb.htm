CGI und Perl: Voraussetzungen/Allgemeines

Sinnvolle Vorkenntnisse zum Verständnis:  HTML sowie  CGI und HTML.
Dieser Abschnitt behandelt aus Sicht eines normalen PC-Benutzers, was an Software erforderlich ist, um CGI-Scripts lokal am eigenen Rechner zum Laufen zu bringen, und was zu beachten ist, um CGI-Scripts an einem "richtigen" WWW-Server zum Laufen zu bringen. Ferner behandelt dieser Abschnitt wichtige Zusammenhänge, die Sie zum Verständnis der Materie benötigen.
Der Abschnitt besteht aus folgenden Seiten:

  WWW-Server lokal auf einem PC einrichten
  Perl lokal auf einem PC einrichten
  CGI-Scripts auf öffentlichem WWW-Server installieren
  CGI-Umgebungsvariablen
  Server Side Includes in HTML
  HTML-Formulare und CGI
