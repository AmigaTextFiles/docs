Ergänzungen und Alternativen zu HTML

Sinnvolle Vorkenntnisse zum Verständnis dieses Abschnitts:  Internet, WWW und HTML.
Dieser Abschnitt geht auf verschiedene Sprachen und Technologien ein, die mehr oder weniger eng mit HTML zusammenhängen und sich zum Teil auch als Alternativen zu HTML verstehen.
Der Abschnitt besteht aus folgenden Seiten:

  CSS Style-Sheets
  JavaScript
  CGI (Common Gateway Interface)
  Java
  ActiveX
  VRML (Virtual Reality Modeling Language)
  XML (Extensible Markup Language)
  Plugins (beliebige Browser-Erweiterungen)
