Klassen mit JSSS Style-Sheets definieren


Beschreibung: so wird's gemacht

Dies ist ein Text, der in die Tags <blockquote>...</blockquote> eingeschlossen ist. Alles ganz normal.
Dies ist ebenfalls Text, der in die Tags <blockquote>...</blockquote> eingeschlossen ist. Hier wurde im einleitenden <blockquote>-Tag jedoch das Attribut class="rottext" notiert. Zuvor wurde mit einer JSSS-Definition für <blockquote>-Tags die Klasse "rottext" definiert, und zwar mit der Angabe:
classes.rottext.BLOCKQUOTE.color = "red";

Dies ist ein Textabsatz, der in die Tags <p>...</p> eingeschlossen ist. Alles ganz normal.

Dies ist ebenfalls ein Textabsatz. Er wurde jedoch mit class="extra" der Klasse "exta" zugewiesen. Diese Klasse wurde zuvor mit JSSS definiert, und zwar so:

 with(classes.extra.all)
  {
    backgroundColor = "#CCFFFF";
    color = "#000066";
    fontSize = "14pt";
    fontFamily = "Impact";
    borderStyle = "none";
    borderWidth = "thin";
   }
