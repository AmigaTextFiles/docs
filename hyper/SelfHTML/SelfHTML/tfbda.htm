HTML-Tags mit JSSS Style-Sheets formatieren


Beschreibung: so wird's gemacht

Dies ist ein Text, der in die Tags <blockquote>...</blockquote> eingeschlossen ist. Für das <blockquote>-Tag wurde mit Hilfe von JSS Style-Sheets die Farbe Blau definiert, und zwar mit der Angabe:
tags.BLOCKQUOTE.color = "blue";

Dies ist eine Überschrift 4. Ordnung. Dafür wurde folgende Angabe notiert:
with(tags.h4)  backgroundColor = "#CCFFFF";

Dies ist ein Bereich, der in die Tags <div>...</div> eingeschlossen ist. Für das <div>-Tag wurde definiert:
with(tags.div)
  {
    color = "red";
    fontSize = "16pt";
    marginLeft = "3cm";
    marginTop = "1.5cm";
   }
