Seitenränder definieren

Anzeigebeispiel: So sieht's aus

Sie können die Abstände des Fensterinhalts von den Fensterrändern links bzw. rechts und oben bzw. unten bestimmen. Die Angaben hierzu werden bislang jedoch nur vom MS Internet Explorer ab Version 2.0 interpretiert.

Bitte benutzen Sie die hier beschriebenen Befehle nur in begründeten Ausnahmefällen - sie werden nur noch der Vollständigkeit halber dokumentiert. Benutzen Sie ansonsten auf jeden Fall CSS Style-Sheets zum Definieren von Seitenrändern. Weiter unten auf dieser Seite finden Sie ein Beispiel für  Seitenränder mit CSS Style-Sheets

Beispiel:

<body leftmargin=30 topmargin=50>

Erläuterung:

Die Angabe für Seitenränder erfolgt im einleitenden <body>-Tag der HTML-Datei (siehe hierzu  Grundgerüst einer HTML-Datei). Durch die Zusatzangabe leftmargin= bestimmen Sie den Abstand zwischen linkem bzw. rechtem Fensterrand und Inhalt der Datei (leftmargin = linker Rand). Mit topmargin= bestimmen Sie den Abstand zwischen oberem bzw. unterem Fensterrand und dem Inhalt der Datei (topmargin = oberer Rand). Beide Angaben erfolgen in Pixeln. Linke und rechte Ränder sind immer gleich groß, ebenso obere und untere.

Beachten Sie:

Die definierten Seitenränder gelten jeweils für die HTML-Datei, in der sie definiert werden. Wenn Sie ein Dokument aus mehreren HTML-Dateien erstellen, das einheitliche Seitenränder haben soll, müssen Sie die Seitenränder in jeder HTML-Datei neu definieren. Natürlich können Sie auch für alle Dateien unterschiedliche Seitenränder definieren.

Sie können zu beiden Werten Angaben machen, aber auch nur zu einem der beiden.

Seitenränder mit CSS Style-Sheets:

Hier noch ein Beispiel, wie Sie Seitenränder auch mit Hilfe von Style-Sheets erreichen können:

Beispiel:

<head>
<style type="text/css">
body { margin-left:30px; margin-right:30px; margin-top:50px; margin-bottom:50px }
</style>
</head>

Erläuterung:

Mit den obigen Angaben im Dateikopf erreichen Sie die gleiche Wirkung wie die zuvor beschriebenen HTML-Befehle. Diese Angaben sind CSS Style-Angaben, die von Netscape seit Version 4.0 und vom MS Internet Explorer seit Version 3.0 interpretiert werden. Außerdem sind diese Angaben vom W3-Konsortium standardisiert.
