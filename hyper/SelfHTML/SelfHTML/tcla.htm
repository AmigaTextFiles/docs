Zentrale Style-Sheet-Bereiche definieren

Anzeigebeispiel: So sieht's aus

Um zentrale Style-Sheets (Formatiermöglichkeiten für HTML-Tags) zu definieren, können Sie im Kopf einer HTML-Datei einen oder mehrere Style-Sheet-Bereiche definieren.

Beispiel:

<head>
<title>Text des Titels</title>
<style type="text/css">
<!--
 body { margin:2cm }
 h1 { font-size:24pt }
//-->
</style>
<style type="text/javascript">
<!--
with(tags.H2)
 {
  color = "red";
  fontSize = "16pt";
  marginTop = "2cm";
 }
//-->
</style>
</head>

Erläuterung:

Mit <style> leiten Sie einen Bereich zum Definieren von Formaten ein (style = Stil). Innerhalb des einleitenden <style>-Tags können Sie mit dem Attribut type= angeben, welche Style-Sheet-Sprache Sie innerhalb des Bereichs zum Definieren der Formate benutzen möchten. Die gängigste Angabe ist dabei type="text/css". Mit dieser Angabe definieren Sie  CSS Style-Sheets als Sprache für die Formatdefinitionen. Die Angabe sollte in Anführungszeichen stehen.

Sie können aber auch andere Sprachen benutzen. Falls Sie mehrere Style-Sprachen innerhalb der gleichen HTML-Datei einsetzen möchten, definieren Sie einfach für jede Sprache einen <style>-Bereich. Im obigen Beispiel etwa werden zwei solcher Bereiche definiert. Der erste Bereich enthält CSS-Formatdefinitionen, der zweite Formatdefinitionen in der netscape-spezifischen Sprache  JSSS Style-Sheets.

Mit </style> beenden Sie einen Abschnitt für Formatdefinitionen.

Beachten Sie:

Es ist empfehlenswert, den Inhalt von Style-Sheet-Bereichen in einen mehrzeiligen  Kommentar (<!-- bzw. //-->) zu setzen, so wie im obigen Beispiel. Dadurch verhindern Sie, daß ältere Browser die Inhalte irrtümlich als Text anzeigen.

Auf die Style-Sheet-Angaben wird hier nicht näher eingegangen. Lesen Sie dazu die entsprechenden Abschnitte innerhalb dieser Dokumentation.

 

Lokale Style-Sheet-Bereiche definieren

Anzeigebeispiel: So sieht's aus

Für Schnellformatierung im Text mit Hilfe von CSS Style-Sheets gibt es ein eigenes HTML-Tag.

Beispiel:

Dies ist ein <span style="font-size:18pt">formatierter Text</span>

Erläuterung:

Mit <span>...</span> können Sie einen Bereich für eine Formatierung definieren. Das Tag für sich genommen hat keine eigene Bedeutung und bewirkt keinen optisch sichtbaren Effekt. Zum Leben erwacht es erst, indem Sie im einleitenden <span>-Tag das  Universalattribut style= notieren. Dahinter folgen, in Anführungszeichen stehend, eine oder beliebig viele  Style-Sheet-Angaben.
