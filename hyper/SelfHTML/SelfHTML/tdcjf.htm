Seitenumbruch nach einem Element


Beschreibung: so wird's gemacht

Diese Seite erscheint an Ihrem Bildschirm genau so wie alle anderen Seiten dieser Dokumentation. Wenn Ihr Browser jedoch die Style-Sheet-Angaben zum Seitenlayout kennt, versuchen Sie mal, diese Datei über den normalen Druckbefehl Ihres WWW-Browsers auszudrucken. Der Absatz, den Sie gerade lesen, enthält einen Befehl, der das nachfolgende Element auf eine neue Seite zwingt.

Überschrift für neue Seite

Wenn Ihr Browser die Angabe kennt, fügt er vor der voranstehenden Überschrift einen erzwungenen Seitenumbruch ein.
