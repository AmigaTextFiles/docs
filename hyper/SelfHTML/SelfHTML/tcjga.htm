Multimedia-Datei einbinden


Beschreibung: so wird's gemacht

In diese Datei ist im folgenden eine VRML-Datei (dreidimensionale Welten) eingebettet. Wenn bei Ihnen gar nichts oder eine Fehlermeldung angezeigt wird, haben Sie kein Programm oder Plugin zum Anzeigen von VRML-Dateien. Wenn Sie ein Plugin, haben, zum Beispiel für Netscape, wird die VRML-Datei direkt im Anzeigefenster des Netscape-Browsers angezeigt. Es stellt einen Zeppelin dar, den Sie sich von allen Seiten anschauen können. Ziehen Sie dazu die Maus bei gedrückter linker Maustaste über die Anzeigefläche, und benutzen Sie die eventuell angezeigten Buttons für andere Navigations- und Ansichtsformen.
