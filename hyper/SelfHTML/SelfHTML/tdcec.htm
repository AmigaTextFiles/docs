Wiederholungs-Effekt


Beschreibung: so wird's gemacht

Dies ist ein einziger Textabsatz mit einigen Zeilenumbrüchen.
Für diesen Textabsatz wurde
eine Hintergrundgrafik definiert.
Dabei wurde jedoch die Angabe
background-repeat:repeat-x
hinzugefügt. Das Ergebnis ist, daß der
Wallpaper-Effekt eingeschränkt ist.
Das Hintergrundbild wird nur einmal für eine "Zeile"
wiederholt, danach jedoch nicht mehr.

Dies ist ein einziger Textabsatz mit einigen Zeilenumbrüchen.
Für diesen Textabsatz wurde
eine Hintergrundgrafik definiert.
Dabei wurde jedoch die Angabe
background-repeat:repeat-y
hinzugefügt. Das Ergebnis ist, daß der
Wallpaper-Effekt eingeschränkt ist.
Das Hintergrundbild wird nur einmal für eine "Spalte"
wiederholt, danach jedoch nicht mehr.

Dies ist ein einziger Textabsatz mit einigen Zeilenumbrüchen.
Für diesen Textabsatz wurde
eine Hintergrundgrafik definiert.
Dabei wurde jedoch die Angabe
background-repeat:no-repeat
hinzugefügt. Das Ergebnis ist, daß das Hintergrundbild nur einmal angezeigt,
aber nicht wiederholt wird. Kein Wallpaper-Effekt also.
Beachten Sie dazu auch ein  erweitertes Beispiel.
