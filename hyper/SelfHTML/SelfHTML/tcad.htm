Deutsche Umlaute und scharfes S

Wenn Sie in Ihrer HTML-Datei sonst keine Angaben zum verwendeten Zeichensatz machen, sollten Sie im Hinblick auf das Internet und die internationale Verwendung deutsche Umlaute durch spezielle, dafür vorgesehenen HTML-Zeichenfolgen ersetzen. Das gilt für den gesamten Inhalt einer HTML-Datei.

Ersetzen Sie das Zeichen ä durch die Zeichenfolge &auml;
Ersetzen Sie das Zeichen Ä durch die Zeichenfolge &Auml;
Ersetzen Sie das Zeichen ö durch die Zeichenfolge &ouml;
Ersetzen Sie das Zeichen Ö durch die Zeichenfolge &Ouml;
Ersetzen Sie das Zeichen ü durch die Zeichenfolge &uuml;
Ersetzen Sie das Zeichen Ü durch die Zeichenfolge &Uuml;
Ersetzen Sie das Zeichen ß durch die Zeichenfolge &szlig;

Beispiel:

In M&uuml;nchen steht ein Hofbr&auml;uhaus.
Dort gibt es Bier aus Ma&szlig;kr&uuml;gen.

Erläuterung:

Ersetzen Sie die deutschen Sonderzeichen wie im obigen Beispiel durch die entsprechenden Zeichenfolgen - keine Sorge, die WWW-Browser verstehen das und zeigen die Zeichen korrekt an.

Beachten Sie:

Es gibt eine Menge weiterer Sonderzeichen, die durch solche Umschreibungen dargestellt werden sollten. Rufen Sie zur Information auf jeden Fall einmal den Abschnitt  Benannte Zeichen in HTML auf. Dort finden Sie Listen mit erlaubten Umschreibungen.

Der genannte Abschnitt befindet sich innerhalb dieser Dokumentation im Kapitel über  Internationalisierung. Dieses Thema wird angesichts der weltweiten Verbreitung von Daten im Internet immer wichtiger. Um tiefer in die Materie einzutauchen, ist die Lektüre des gesamten Internationalisierungskapitels unbedingt zu empfehlen.

Ab HTML 4.0 sind Sie nicht mehr gezwungen, Sonderzeichen auf die oben beschriebene Weise zu umschreiben. Stattdessen können Sie  Zeichensätze als Meta-Angabe notieren. Wenn Sie dann in der Software, mit der Sie die HTML-Dateien erstellen, eben diesen Zeichensatz verwenden, hat ein WWW-Browser alle Informationen, die zur korrekten Anzeige der Datei erforderlich sind.

 

HTML-eigene Zeichen maskieren

Wenn in Ihrem Text Zeichen vorkommen, die in HTML eine bestimmte Bedeutung haben, müssen Sie diese Zeichen maskieren. Die folgenden Zeichen müssen Sie wie folgt maskieren:

Ersetzen Sie das Zeichen < durch die Zeichenfolge &lt;
Ersetzen Sie das Zeichen > durch die Zeichenfolge &gt;
Ersetzen Sie das Zeichen & durch die Zeichenfolge &amp;
Ersetzen Sie das Zeichen " durch die Zeichenfolge &quot;

Beispiel:

Das ist ein &lt;HTML-Tag&gt;
Gmbh &amp; Co. KG
&quot;Text steht in Anf&uuml;hrungszeichen&quot;

Erläuterung:

Ersetzen Sie die HTML-eigenen Zeichen wie im obigen Beispiel durch die entsprechenden Zeichenfolgen.
