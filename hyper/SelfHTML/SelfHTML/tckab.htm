Inline-Layer definieren


Beschreibung: so wird's gemacht

Das ist Layer-Text mitten im Text. Das Wort "Layer-Text" ist dabei in einen Inline-Layer eingebunden. Damit das besser sichtbar wird, hat der Inline-Layer eine eigene  Hintergrundfarbe erhalten. Außerdem wurde die Angabe top=3 notiert. Das bewirkt, daß der Inline-Layer 3 Pixel tiefer dargestellt wird als normal.
