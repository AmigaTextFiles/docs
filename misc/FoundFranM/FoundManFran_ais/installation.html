Système Requis:

Fondation requière un Amiga équipé en AGA avec 2Méga (p.e. A1200). Le jeu a été développé pour les Amigas basés sur le 68030 mais un A1200 est suffisant pour faire fonctionner le jeu. La version RTG requière une carte graphique supportée par CyberGraphX ou Picasso96 et au moins 8méga de mémoire rapide (Fast memory). 4méga de ram Vidéo sont recommandés pour les écrans hautes résolutions. Un processeur rapide est requis pour faire fonctionner la version RTG.

La mémoire supplémentaire est aussi très utile puisque qu'elle réduit la quantité d'accès au disque durant le jeu. Les utilisateurs avec seulement 2méga de mémoire trouveront que le jeu accède très fréquemment au disque. Quand le jeu doit utiliser sa propre cache système il est recommandé que vous utilisiez un programme de cache dédicacé pour un support et une flexibilité meilleur, pas pour la vitesse.

Installation:

Pour installer le jeux vous devez copier le programme "FoundationPrefs" à un emplacement qui convient sur votre disque dur. Il est recommandé de créer un nouveau répertoire et d'y placer le programme "FoundationPrefs". Quand vous lancez "FoundationPrefs" il créé certains répertoires importants et installe le minimum de fichiers requis pour démarrer le jeu. "FoundationPrefs" contient quelques options d'installation qui vous permettront d'installer plus de fichiers de Foundation si vous préférez utiliser l'accès rapide aux fichiers procuré par le disque dur.
