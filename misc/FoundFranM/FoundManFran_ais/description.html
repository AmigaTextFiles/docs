Description du jeu:

Les objectifs de chaque jeu varient mais souvent ils nécessitent de compléter différentes tâches. Ces tâches peuvent être, détruire l'équipe ennemie ou secourir certain peuples. Les méthodes utilisées pour compléter une mission changeront avec chaque tâche mais vous serez souvent amené à créer un village, entraîner quelques soldats, développer de nouvelles technologies et prendre part à des batailles.

La petite carte

Dans le coin supérieur gauche de l'écran principal vous pouvez voir la petit carte. Celle-ci montre la position de votre vue actuel sur le terrain et offre une méthode rapide pour déplacer cette vue à d'autres endroits. Elle vous montre aussi les zones qui ont été explorées et les zones colorées montrent l'emplacement des constructions. En cliquant la petite carte avec le bouton droit vous ferez apparaître la fenêtre de la grande carte.

La section de la carte contient des gadgets supplémentaires. Le gadget de vitesse de jeu, vous permet de passer entre les différentes vitesses de jeu. Le gadget de message s'éclairera quand vous aurez certains messages importants à lire. Ce gadget peut être utilisé pour ouvrir la fenêtre de message quand un message est en attente. Le gadget d'espaces de construction vous montrera tout les points de constructions disponibles. Le gadget suivant est le gadget de notification d'un emplacement, qui est utilisé pour vous renseigner sur un emplacement important chaque fois que vous êtes averti d'un évènement important. Finalement il reste le gadget d'idée qui s'éclaire quand une nouvelle invention a été créée, vous permettant d'examiner la nouvelle idée.

La grande carte

C'est la grande Carte.

Le panneau des ressources

Situé dans le bas de l'écran, cette zone donne une vue actuel de vos ressources. La figure montre la quantité de ressources ou la population que vous avez dans votre quartier général, les entrepôts et les autres types de construction pour l'entreposage.

La vue de la zone principal

C'est ici que toutes les actions peuvent être vues. C'est ici que vous pouvez pointer et sélectionner vos sujets et constructions. Cette vue est utilisée pour presque tout vos contrôles du jeu. Vous pouvez redimensionner la vue du jeu en utilisant les touches + et - du clavier numérique. Rendre la taille de la vue plus petite accélérera le jeu sur les machines plus lentes.

Le panneau de contrôle

C'est le panneau ou toutes les statistiques et les contrôles du jeu peuvent être accédés. Si vous avez sélectionné une construction ou un de vos sujets, vous pouvez donner des ordres ou altérer les réglages de vos constructions. Depuis le panneau d'icônes vous pouvez aussi définir votre commerce et beaucoup d'autres contrôles du jeu. Le panneau de contrôle avec la zone de visualisation est l'endroit où l'ensemble du jeu est contrôlé.

Les boutons de localisation rapide

Parfois vous pourrez avoir besoin de vous rendre plusieurs fois à un emplacement particulier sur la carte. Par exemple vous pourriez vouloir faire une vérification sur un fort enfermé dans le territoire ennemi et en même temps continuer à jouer normalement. Dans ce cas vous devez positionner votre vue pour qu'elle affiche le fort et presser Shift-F1 pour marquer l'emplacement dans la "Localisation rapide 1". A partir de cet instant, vous pouvez rappeler cette vue en pressant la touche "F1".

Un système similaire est disponible pour sélectionner des groupes de sujets. Si vous avez un petit nombre de sujets sélectionné et que vous pressez Shift-1, ce groupe sera marqué dans la "Localisation rapide 1". A partir de cet instant, vous pouvez resélectionner ce groupe en pressant la touche "1".

Il y a aussi un raccourci souris pour ce système, utilisant le bouton droit pour définir la "Localisation rapide" et le bouton de gauche pour l'utiliser. Si un bouton de localisation rapide n'est pas encore utilisé, le bouton gauche de la souris peut être utilisé pour l'initialiser.
