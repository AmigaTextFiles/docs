Les Touches dans le Jeu:

Shift F1 - F5 = Mémorise la position sur la carte
F1 - F5 = Retourne à la position sur la carte
Shift 1 - 5 = Mémorise le groupe de Sujets actuellement sélectionné
1 - 5 = Retourne au groupe de Sujets
P = Pause
O = Sélectionne tout les sujets en dehors des bords
G = Sélectionne tout les sujets
H = Ramène la vue sur le Quartier Général
E = Sélectionne tout les sujets visibles ou sélectionne une personne ennemie
N = Boucle entre les sujets sélectionnés
C = Centre sur le personnage actuel
S = Touche de sauvegarde rapide
L = Touche de chargement rapide
X = Disperser les sujets sélectionnés
- = Réduire la taille de la vue
+ = Elargir la taille de la vue
Enter = Mode Message
