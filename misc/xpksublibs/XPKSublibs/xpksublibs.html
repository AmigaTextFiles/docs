KEY














Source UNKNOWN =

Source was never known. It appears only on lists compiled by others or on file collections or other packages.














It is possible that the sublib is so old it predates Aminet or Aminet being popular as software repository. It may be on old collections such as Fish Disks or from BBS.



























Source LOST =

It was either on Aminet but has subsequently been removed or accidentally deleted/overwritten.














or it was on developer’s own site which then was closed and was lost.













Pre-Cholok versions and dates are shown (bracketed).














Cholok * =

xpk sublib has been re-engineered, optimised and fixed of minor bugs by Cholok and/or Cosmos.









































Cholok FIX! =

sublib has been fixed of critical bugs, which caused memory corruption and/or

data corruption (whether on some modes or all), or it simply didn’t work before

Cholok’s fix. Using a non-Cholok-updated version of the sublib is NOT recommended.
