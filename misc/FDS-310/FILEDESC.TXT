%DES: Default
USG File Description Standard, release 3!
 This is the third release of the USG
standard for file descriptions.
 The standard contains features like
multilanguage descriptions, support for
different charsets, use in ANY kind of file.
It has keywords that describes requirements,
language, registration info, versions,
author info, and MORE!
%DES: Svenska
USG File Description Standard, release 3!
 Det här är den tredje upplagan av USG
standarden för att beskriva filer.
 Standarden innehåller saker som
flerspråkiga beskrivningar, stöd för olika
teckenuppsättningar, användning i ALLA typer
av filer.  Den har nyckelord som beskriver
systemkrav, språk, registrerings
information, version, info om upphovsman,
och MYCKET mer!
