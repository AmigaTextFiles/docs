Wo gibt es Amiga Fonts

  1. Als erstes sollte man im AMINET nach fonts suchen. Dies ist das größte Archiv mit Amiga Software und dort findet man eine ganze Menge an Fonts. Das Aminet hat viele Mirrors, die auf der ganzen Welt verstreut sind. Hier ist eine Auswahl:

    @enumerate @item ftp.wustl.edu, @item ftp.luth.se, @item ftp.eunet.ch, @item ftp.uni-paderborn.de, @item ftp.doc.ic.ac.uk. @end enumerate

    Man muß sich als user ftp einloggen und in das folgende Verzeichnis wechseln

    /pub/aminet/text/font

  2. Ein weiterer guter ftp server ist der CICA-Server:

    @enumerate @item ftp.cica.indiana.edu @end enumerate

    Zu diesem Server gibt es ebenfalls zahlreiche Mirrors.

  3. Ein weiterer Platz zum Suchen wäre der folgende WWW Server:

    @enumerate @item http://jasper.ora.com/Internet-Font-Archive.html @end enumerate

  4. Zudem befinden sich noch sehr viele Zeichensätze auf der Fresh Fonts I CD-ROM. Diese CD ist erhältlich bei

    @enumerate @item Fred Fish / Amiga Library Services (orders@amigalib.com) @item Stefan Ossowski / Schatztruhe GmbH @end enumerate

    Die CD ist kostenlos, wenn man eine weitere CD bei dieser Firma kauft.

    Die HTML-Seiten der CD befinden sich unter folgender Adresse:

    @enumerate

    @item http://macke.gris.informatik.uni-tuebingen.de:4711/~damor/

    @end enumerate
