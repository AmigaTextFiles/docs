Kann ich AmigaBasic auf dem A1200 verwenden?

AmigaBasic wurde (als einziges Amiga-Programm) von Microsoft entwickelt und ist entsprechend fehlerhaft.

Man kann es auf dem A1200 trotzdem zum Laufen bringen, wenn man folgendes beachtet:

  1. NoFastMem starten (also FastMem ausschalten).
  2. Im Audio Prefs den Ton abstellen.
  3. Möglichst SUBs vermeiden und stattdessen GOSUBs verwenden, dann ist die Kompatibilität zu neuen Prozessoren höher.
  4. Es gibt einen Patch, den man zu Beginn von AmigaBasic starten sollte: 
      ---------------------------------- CUT HERE -------------------------------
   `" AmigaBASIC patch to let AmigaBASIC work on A1200 and other newer machines.
   `" Start at the beginning of AmigaBASIC or invoke AmigaBASIC with this program

   OPEN "AMIGABasic" AS 1 LEN=1
   FIELD #1,1 AS d$
   i&=&HF3*256+&H87 : PRINT i&
   GET #1,i& : a$=HEX$(ASC(d$))
   PRINT a$
   IF a$="79" THEN
       LSET d$=CHR$(&H78)
       PUT #1,i&
   END IF
   CLOSE 1
  ---------------------------------- CUT HERE -------------------------------
