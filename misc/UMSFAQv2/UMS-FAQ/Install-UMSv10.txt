Installing Universal Message System v10.9...

>In which disk or drawer should Universal Message System be installed?
*ANS: Work:UMS

>In which disk or drawer should the message base be placed?
*ANS: Work:UMS/mb/

>Which tools do you want to install?
*ANS: TICK ALL OF THEM

>Which message readers do you want to install?
*ANS:CLICK ON  IntuiNews ONLY

>Which languages do you want to use for IntuiNews?
*ANS:  english UNLESS YOU WANT ANOTHER ;)

>Select a destination for the english online documentation
*ANS: HELP:english/

>Which Bulletin Board Systems do you want to install?
*ANS: DO NOT SELECT ANY OF THESE

>In which networks do you want to participate?
*ANS: CLICK ON 'UUCP' ONLY

>Please enter the name of your newsfeed (without domain)
*ANS: Demon

>Does  your newsfeed rely on information in the UUCP mail envelopes (RFC976
format) instead of parsing to RFC822 header?
*ANS: NO

>Do you want "binary" instead of "8bit" as MIME contents encoding type?
*ANS: NO

>Do  you  want  bad  jobs,  which  had only dupe errors to be automatically
deleted?
*ANS: YES

>Please choose exporting method for mail and news
*ANS: batch news

>Please select compression method for news
*ANS: none

>Please enter maximal number of bytes for one batch
*ans: 65536

>Please enter your node's name
*ans: tlc1
*  Don't  enter 'tlc1' put your nodename there (1st bit after @ in your net
addr )

>Please enter your node's domain name
*ANS: tlc1.demon.co.uk
* Again, don't enter the above sub everything after the @ in your net-addr

>Is your node registered in the UUCP maps?
*ANS: No

>Do you want to install developer material (ARexx, C, Modula-2, Oberon)?
*ANS: Yes
* You don't HAVE to but there are some useful docs there to Read :)

>Please enter your full name
*ANS: Clive Thomas
* ie YOUR real name!


>Please enter your short name
*ANS clive
* this is the bit BEFORE the @ in your net-addr

>Please enter your password
*ANS: Hit Right-Amiga key and the x key together then hit return so it's
BLANK then proceed.

>In which disk or drawer should the backup of your message base be placed?
*ANS: Work:UMS/bak/

Exiting script...
