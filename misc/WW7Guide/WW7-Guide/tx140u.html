ARexx-Befehle im Menü 'Format'

FONT (NAME/S,SIZE/N,ITALIC/S,BOLD/S,PLAIN/S, UNDERLINED/S,DOUBLEUNDERLINED/S, STRIKETHROUGH/S,SMALLCAPS/S,WIDTH/K/N, OBLIQUE/K/N)

Erlaubt die Auswahl von Fonts und/oder Formaten. Werden keine Parameter angegeben, so erscheint das entsprechende Dialogfenster.

Der Parameter WIDTH (Breite) erlaubt die Angabe von Werten zwischen 10% und 999%, OBLIQUE zwischen "-45" und 45, wobei negative Werte grundsätzlich in Anführungszeichen zu setzen sind.

TABS
Öffnet das Dialogfenster `Tabulatoren`

SETTAB type position (/A,/N)
Setzt einen Tabulator für den oder die aktuellen Absätze.

'type' kann entweder LEFT, RIGHT, CENTRE oder DECIMAL sein.

'position' kennzeichnet den Abstand vom linken Rand des Dokuments, gemessen in den aktuellen Skaleneinheiten.

Beispiel: 'SETTAB LEFT 2.5'

DOCUMENT
DOCUMENT width height leftmargin rightmargin topmargin bottommargin columns gap landscape facingpages doublepagespread mirrormargins

Werden keine Parameter angegeben, öffnet sich das Dialogfenster 'Dokument'.

Wenn Parameter angegeben werden, müssen diese einem der beiden oben gezeigten Formate entsprechen: Entweder für eine benannte Seitengröße (z.B. A4) oder eine individuelle Seitengröße. Alle Parameter sind optional.

'pagesizetype': Entweder A3, A4, A5, USLetter, USLegal oder WideTractor.
'width' ist die Seitenbreite im voreingestellten Maß, z.B. in Zentimetern.
'height' ist die Seitenhöhe im voreingestellten Maß.
'leftmargin' ist der linke Rand im voreingestellten Maß.
. 'rightmargin' ist der rechte Rand im voreingestellten Maß.
'topmargin' ist der obere Rand im voreingestellten Maß.
'bottommargin' ist der untere Rand im voreingestellten Maß.
'columns' ist die Spaltenanzahl, 1-32.
'gap' ist der Spaltenabstand im voreingestellten Maß.
'landscape' ist entweder LANDSCAPEON oder LANDSCAPEOFF.
'facingpages' ist entweder FACINGPAGESON oder FACINGPAGESOFF.
'doublepagespread' ist entweder DOUBLEPAGESPREADON oder DOUBLEPAGESPREADOFF.
'mirrormargins' ist entweder MIRRORMARGINSON oder MIRRORMARGINSOFF.

STYLESHEETS num (/N)
Öffnet das Dialogfenster `Absatzformate`, wenn kein Parameter angegeben wird.

Wird eine Zahl vorgegeben, wird dem aktuellen Absatz bzw. den markierten Absätzen das dieser Zahl entsprechende Format (0=Kein Format, 1=Normal, usw.) zugeordnet.

BULLETFORMAT
Öffnet das Dialogfenster `Listenpunktformat`.
