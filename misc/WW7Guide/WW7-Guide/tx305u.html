Andersformatiges Dokument öffnen

Dokumente, die in einem von Wordworth unterstützten Fremdformat vorliegen, lassen sich genauso laden wie ein normales Wordworth-Dokument.

Mit Wordworth können Sie Texte laden, die in einem der folgenden Formate vorliegen:

  • Rich Text Format (RTF)
  • Wordperfect für Dos
  • Word für Dos
  • Works für Windows
  • Windows Write
  • Wordstar
  • Final Copy II / Final Writer
  • ASCII

Beachten Sie, daß beim Laden von fremdformatigen Texten zunächst möglicherweise falsche Schriftarten benutzt werden.

Damit Wordworth beim Laden von Texten im RTF-Format die richtigen Schriften benutzt, müssen Sie evtl. die Datei "FontMapOpen" nachbearbeiten, die Sie im Verzeichnis "WWFiles" finden.

Anhand dieser Datei ermittelt Wordworth, welche Schriftarten auf dem Quellsystem denjenigen auf dem Zielsystem zuzuordnen sind. Dies erweist sich oft als problematisch, da die Fonts häufig leicht unterschiedliche Namen aufweisen.

Die Datei "FontMapSave", welche die Anpassung von Schriften beim Abspeichern in das RTF-Format vornimmt, entspricht im wesentlichen der Datei "FontMapOpen", besitzt allerdings umgekehrte Funktionalität; wenn beim Laden die Schriftart "Arial" in "Shannon Book" gewandelt wurde, wird dieser Vorgang beim Speichern wieder rückgängig gemacht.

HINWEIS:
Die besten Ergebnisse lassen sich üblicherweise mit dem "RTF-Format" erzielen. Das RTF-Format ist eine sehr weit ver breitete Standardimplementation, die von fast jedem Textverarbeitungsprogramm auf dem Markt unterstützt wird.
