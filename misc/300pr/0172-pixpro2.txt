Axiom Software releases Pixel 3D Professional Version 2


August 1, 1994 - Axiom Software is pleased to announce that 
Pixel 3D Professional Version 2 has just been released and is now 
available. 

Pixel 3D Professional Version 2 (PixPro2) has been entirely rewritten 
from the ground up and includes many new powerful features.  PixPro2 now 
sports full Amiga AGA support, full PostScript font and file support, 
artificial intelligent bitmap to 3D conversion, greatly simplified bitmap 
smoothing, auto-convertion of 24 bit files through ADPro, 3D point editing, 
3D polygon painting, complete AREXX support, 3D router bit beveling, 
multiple object loading, complete hierarchy support and editing, 
support for seventeen 3D file formats including LightWave scenes, 
Imagine hierarchies, 3D Studio binary and ASCII, Wavefront, and the most 
comprehensive DXF AutoCAD support the Amiga has even seen!

PixPro2's retail price is $299.95.   All Pixel 3D Professional V1 owners 
may upgrade to PixPro2 through either their local dealer or through 
Axiom Software for $99.99.  All dealers interested in becoming an 
Axiom Software approved dealer should contact Andy Hills at  
1-800-49-AXIOM (1-800-492-9466).
