HiSOFT C++ brings ANSI C and AT&T 3.0 C++ programming to your Amiga. No
need to battle with the CLI, HiSOFT C++ boasts a powerful,
closely-integrated development environment with drag-and-drop features
throughout, leaving you free to concentrate on writing your applications.
No need to leaf through hundreds of printed pages, the HiSOFT C++ manual is
conveniently read on line using the included HotHelp Reader. The ANSI C
include files are also included as a HotHelp project.
