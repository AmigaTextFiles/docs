3.3   Animation laden

Hier können Sie mit Xi-Paint erstellte Animationen als auch beliebige andere Animationen laden. Diese müssen allerdings die Dateiendung .000 bis .999 haben. Wählen Sie das niedrigste Bild aus, welches Sie bearbeiten wollen.

Beachten Sie bitte, daß Xi-Paint nur im IFF-DEEP abgespeicherte Bilder bearbeiten kann. Wählen Sie, falls dem nicht so ist, im Hauptfenster für Animationen das Gadget "Generiere Minipic" aus. Ausserdem sollte als Lademodul dasjenige ausgewählt sein, welches das Bildformat beherrscht.
