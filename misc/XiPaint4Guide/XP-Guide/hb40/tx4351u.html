3.5.1   Vorschau

Im Vorschaufenster können Sie ungefähr erkennen, wie Ihre Animation aussieht. Sie haben die Möglichkeit, mit Start die Animation abzuspielen, mit Stop zu beenden.

Auf den meisten Rechnern ist diese Animation zu schnell - dann sollte man Delay auf einen Wert von 1 bzw. 2 stellen. Dann wird nach jedem Bild eine Verzögerung von 1/50 Sekunde bzw. 2/50 Sekunde eingehalten.

Mit dem Schalter PingPong können sie die Animation von Vorne nach hinten und dann wieder nach vorne abspielen lassen. Ist dieser Schalter nicht aktiv, so wird die Animation immer wieder von Vorne begonnen.
