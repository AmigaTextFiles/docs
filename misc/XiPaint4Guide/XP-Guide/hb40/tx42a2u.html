2.10.2   Bild

 

Magic Fill:
Stellt das neue Feature zum Füllen von Farbbereichen ein. Ist dieser Schalter ausgeschaltet, so wird nur eine solide Farbe als Füllumrandung genutzt.


Magic Delta:
Stellt den Bereich zum Füllen von variablen Farbbereichen ein. Je größer der Wert, umso mehr wird gefüllt.
