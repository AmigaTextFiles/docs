1.2.10   Retina

Dieser ist als Update bei MacroSystem erhältlich. Mit diesem neuen Treiber können Sie dann alle Programme dieser CD benutzen. Als Zwischenlösung bietet sich die AA 256-Farben Version unter der Workbench-Emulation an.

Es existieren zwei Versionen, einer für direkte Retina-Screens in Auflösungen von 24-Bit, 16-Bit und 8-Bit (gerastert).

Die zweite Version benutzt die 24-Bit-Workbench-Emulation der Retina-Software. Damit können sie auf einem normalen IntuitionScreen (Public) in 24-Bit oder 26-Bit malen. Die Auswahl der Auflösung erfolgt über den ASL-Screenmode-Requester.
