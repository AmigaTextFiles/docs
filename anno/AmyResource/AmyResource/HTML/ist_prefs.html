Prefs

This drawer contains various preference files to properly set up your working environment so that the icons will be showed with the correct colours and resolution.

If you are already using the "MagicWB" colour scheme, you will not need to execute these files.
If you are using some other kinds of icons you are greatly suggested to select the appropriate resolution by clicking the "Colours" icon.

A complete preferences management tool is also included for your personal use. The ARPrefs program will let you choose the best way to display or play various kind of files ( i.e. mods, images, web pages etc... ). Many powerful tools have already been selected by the Amy Resource staff, so that the most appropriate software will always be activated when requested.
