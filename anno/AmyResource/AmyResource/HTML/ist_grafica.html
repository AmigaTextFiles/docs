Graphics

This drawer contains the "artistic spirit" of the Amy Resource CD.
Many animations have been included together with textures and clip-art.
Everything has been carefully selected from many Public Domain sources and you will be able to find the best artwork recently released by various authors.
All of the images, textures and clip-arts are provided in the format originally selected by their authors, but some have been converted in other formats and resolutions for ECS owners and for other kind of using ( web, etc.... ).

  • To easily explore the contents of these drawers, you can use the "PicView" program ( a complete catalogue is available for each drawer )

  • By clicking the single icons, you will be able to see the original image or animation using the programs selected with the ARPrefs tool.
