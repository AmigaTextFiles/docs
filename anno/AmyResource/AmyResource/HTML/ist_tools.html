Tools

This drawer contains players, viewers and browsers needed to explore this CD-ROM.

We've managed to obtain two special versions of "IBrowse" and "AWeb", created only to browse local files (so these programs cannot load directly Internet documents, as they haven't HTTP routines).

CD-ROM tools' documentation could be read even directly from the Web browser, thanks to the complete translation of the documentation in HTML format (please consult: MUI Documentation, Viewtek Documentation, Visage Documentation and PicView Documentation).

Last, but not least, there's a preferences system that could be accessed by the "ARPRefs" programs: in this way, you will be able to configure a tool for every single filetype.
