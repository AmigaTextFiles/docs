Using the right-side gadgets, it's possible to use a file requester to choose the tool to use to view/play/browse the filetype.

Using the cycle-gadget it's possible to choose between special versions of IBrowse and AWeb present on CD-ROM or to insert a personal browser.
If you want to use a browser different from the above mentioned, please remember that the HTML Magazine makes use of frames and tables, so the browser you want to use must support these features.

Save: this button saves permanently the prefs created in a file stored in the ENVARC:/ENV: directory.

Use: this button saves temporary the prefs created in a file stored in the ENV: directory (turning off or resetting the computer results in a loss of the prefs file).
Use this button to test the configuration before making it definitive.

Cancel: this button resets the changes made on the configuration, restoring the previous prefs file.
