
Rainboot 3
----------

The ultimate multimedia boot, Rainboot3, is also on the CD-ROM
in the new, exclusive version 3.1. Rainboot v3.1 allows
the user to show complete boot shows during the bootup. Rainboot
supports nearly everything you can think of: Animations, sounds,
copper rainbows, password protection, brushes, different fonts,
modules, fades, CyberGraphX and Picasso96 support, progressbars,
retrieving and displaying information about the used Amiga
system, graphical effects, scrolling of pictures and much much
more. Rainboot is a unique program, which has neither been
beaten in features and effectivity on Amiga nor on PC.


New features of Rainboot 3
--------------------------

- user controls can be added to Rainboot configurations now, that
  means that one can create real multimedia programs with Rainboot.
  You can catch mouseclicks and -moves in Rainboot configurations
  and execute several operations then

- Rainboot can now use more than one animbrush. You can use now
  as many animbrushes as your memory allows

- more than one Protracker module can be used

- more than one cycle can be defined and used

- MED modules can be used

- and many other small new things

Rainboot 3 is fully compatible to older versions of Rainboot;
you can use all Rainboot configurations with Rainboot 3.
