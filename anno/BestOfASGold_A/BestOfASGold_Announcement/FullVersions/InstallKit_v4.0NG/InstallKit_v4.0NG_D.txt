
CD32 Games Install Kit v4.0 NG
------------------------------

Das CD32 Games Install Kit v4.0 Next Generation wurde exklusiv
für diese CD-ROM entwickelt. Mit der Version 4.0 des CD32 Games
Install Kits lassen sich jetzt über 170 CD32 Spiele auf
Festplatte installieren oder auch direkt starten. Außerdem
besteht auch die Möglichkeit, mit der Hilfe eines generellen
CD32 Installationsalgorhythmus, nicht vordefinierte Spiele
zu installieren. Das Bootsystem Install2001 liegt nun in der
lang erwarteten Version 1.0 vor, welche mit vielen Neuerungen
aufwartet. Zusätzlich bietet das CD32 Games Install Kit die
Möglichkeit, einige CD32 Spiele auch ohne AGA Chipsatz zu
installieren/starten. Mit im Paket sind auch einige weitere
interessante Programme für CD32 Spiele. 


Neue Features des CD32 Games Install Kits v4.0 NG
-------------------------------------------------

- Viele neue Spiele können installiert werden

- Install2001 wurde neuprogrammiert und unterstützt mehr
  Emulationen

- Bessere Benutzeroberfläche

- Neue Einstellmöglichkeiten und Verbesserungen in der
  CD-ROM Unterstützung

- Und viele andere neue Funktionen
