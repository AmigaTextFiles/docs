This installer requires the original Infestation disk from Psygnosis.

fixed:

fixed more self modifying code.

Features:

- JST and WHDLoad loaders
- System friendly imager (requires RawDic 1.6)
- Press <CTRL> to quit at any time.
- Use the USERDATA=2 tooltype to skip intro (JST).
- Save game to HD.

Save Game Instructions:

To save games you must first select (3) Format disk in drive 0.  Then you can
save with (4).  Selection 3 saves the 1st track of the save disk to the HD.
