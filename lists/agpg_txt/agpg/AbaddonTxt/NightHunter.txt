Short:	 	HD Installer for Night Hunter v1.0
Author:		Keith Krellwitz (abaddon@wwa.com)
Uploader:	Keith Krellwitz (abaddon@wwa.com)
Type:	 	game/patch

This installer requires the original Night Hunter disk from UBI-Soft.

That install requires RawDIC V1.6 to be in the path.
	
Features

 - JST and WHDLoad loaders
 - System friendly imager using RawDic
 - At any time press 'F10' to quit.
 - High scores saved to HD
 - Saved games saved to HD

This installer is NOT FOR COMMERCIAL USE and may not be included on any
CD compilation without my written permission.  The only exception to this
is Aminet.
