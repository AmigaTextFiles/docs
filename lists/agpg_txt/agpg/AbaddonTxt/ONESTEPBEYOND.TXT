Short:	 	HD Installer for One Step Beyond v1.1
Author:		Keith Krellwitz (abaddon@wwa.com)
Uploader:	Keith Krellwitz (abaddon@wwa.com)
Type:	 	game/patch

This installer requires the original One Step Beyond disks by Ocean, there are two.

Thanks to Chris Vella for sending the original.

Features:

- JST and WHDLoad loaders (fixed missed access fault)
- Iconify/Snapshot supported <TAB> (Reg'ed users) (JST)
- Full loading of the game from HD
- Convenient installer script using system-friendly disk ripper
- Possible quit with F10 key

It has been successfully tested on a A1200/030/40MHz/10MB.
