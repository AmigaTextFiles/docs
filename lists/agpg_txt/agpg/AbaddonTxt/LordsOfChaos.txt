Author:		Keith Krellwitz (abaddon@wwa.com)
Uploader:	Keith Krellwitz (abaddon@wwa.com)
Type:	 	game/patch

This installer requires the original Lords Of Chaos by Blade.

Thanks to Mark Knibbs for sending the original.

Features:

 - JST and WHDLoad versions
 - Press 'F10' to quit.
 - Save game and wizards to hard drive
 - Fixed 24 bit addressing problems (Hopefully I got them all)
 - Fixed access faults

NOTE:

To save game or wizard you must 1st select Format 'Chaos' Disk.  Once
this is down you can then save wizards or games.
