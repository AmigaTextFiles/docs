Short : HD-Installer for Stack Up V1.0
Type : game/patch
Author : Bored Seal (cossick@email.cz), ICQ:21086453

This patch applies to "Stack Up" by Zeppelin Games (c) 1991 - 1 disk.
Slave requires WHDLOAD V10+.
Game requires 0.5 MB Chip Mem (+ 0.4 other memory for preload option)

FEATURES:

- Full load from HD (disk image shortened)
- RNC disk protection removed
- Blitter waits inserted [7x]
- Highscores are saved
- ButtonWait tooltype added for title picture
- NI/ROM icon (created by JHZ)
- Quit option (default key is '*' on keypad)

Thanks to Helmut Motzkau for disk image and great support!
