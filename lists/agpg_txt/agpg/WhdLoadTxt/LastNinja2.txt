

	That install applies to Last Ninja 2 (c) 1990 System 3 (1 disk version).


	That install requires WHDLoad to be in the path. It can be found in
	the whdload package.


	You need 1.0 MB ChipMem and 0.5 MB OtherMem.
	You can quit the game by pressing F9.
