Short : HD-Installer for Beast Busters V1.1
Type : game/patch
Author : Bored Seal (cossick@email.cz), ICQ:21086453

This patch applies to "Beast Busters" by Activision (c) 1990 - 2 disks.
Slave requires WHDLOAD V10+ for run.
Game requires 0.5 MB Chip Mem (+ 1.3 other memory for preload option)

FEATURES:

- Full load from HD
- RNC disk protection removed
- Sound-off routine rewritten
- Saves highscore (press F10 on title screen)
- Trainer added (F5 press will give you 9 credits)
- NI/MWB icon (created by JHZ, ripped by Midwik)
- Quit option (default key is '*' on keypad)

HISTORY:

V1.0 (10-Oct-1998) - first public release
V1.1 (16-May-1999) - slave optimized
                   - slave now uses original files (no need to XFD them)
                     (please reinstall)
                   - trainer added
