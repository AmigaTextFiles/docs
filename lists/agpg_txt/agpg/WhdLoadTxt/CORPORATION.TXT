	That install requires WHDLoad and DIC to be in the path. Both
	can be found in the WHDLoad package.
	To install from a different drive than DF0: select Expert as
	installation mode.


	You need 1 MB ChipMem.
	You can quit the game by pressing F10 (key can be changed with
	Tooltype "QuitKey=").
	

	Features:
	 - supports 4 different releases/versions
	 - bad keyboard routine fixed
	 - copylock removed
	 - savegames to harddisk
	 - little trainer added: press HELP on the equipment screen to get
	   some more money, use it not to much or the game crashes
	 - normal and mission disk can be used simultaneous
	 - to skip the intro press any key
