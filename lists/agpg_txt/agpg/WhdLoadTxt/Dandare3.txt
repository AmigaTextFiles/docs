Short : HD-Installer for Dan Dare 3 V1.0
Type : game/patch
Author : Bored Seal (cossick@email.cz), ICQ:21086453

This patch applies to "Dan Dare 3" by Virgin/Probe (c) 19?? - 1 disk.
Slave requires WHDLOAD (at least V7) for run.
Game requires 0.5 MB Chip Mem + (+ 0.8 other memory for preload option)

FEATURES:

- Full load from HD
- Disk protection removed
- OS emulation module used
- Trainer option (set CUSTOM1=1 for unlimited lives)
- NI/ROM icon (created by JHZ)
- Quit option (default key is '*' on keypad)
