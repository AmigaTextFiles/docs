Short : HD-Installer for Mig-29 Soviet Fighter V1.0 (26-Dec-1998)
Type : game/patch
Author : Bored Seal (cossick@email.cz), ICQ:21086453

This patch applies to "Mig-29 Soviet Fighter" by Code Masters/Rapier Design (c) 1990 - 1 disk.
Slave requires WHDLOAD (at least V8+) for run.
Game requires 0.5 MB Chip Mem (+ 0.3 other memory for preload option)

FEATURES:

- Full load from HD
- RNC disk protection removed
- Blitter operation fixed (20x)
- Sound fixed (2x)
- NI/MWB icon (ripped by Midwik, created by JHZ)
- Quit option (your default key)

Thanks to Lempkee for disk image.
