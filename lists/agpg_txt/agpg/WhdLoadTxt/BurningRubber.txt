Short : HD-Installer for Burning Rubber V1.1
Type : game/patch
Author : Bored Seal (cossick@email.cz), ICQ:21086453

This patch applies to "Burning Rubber" by Ocean (c) 1993 - 4 disks.
Slave requires WHDLOAD (at least V7) for run.
AGA version requires 2.0 MB Chip Mem (+ 2.8 other memory for preload option).
ECS version requires 1.0 MB Chip Mem (+ 2.7 other memory for preload option).

FEATURES:
- Full load from HD
- 2 version are supported (AGA, ECS)
- RNC disk protection removed (2x)
- RNC decruncher relocated to Fast RAM
- Intro skip option
- Trainer option enabled (in ECS version only)
- NI/MWB icon (created by JHZ)
- Quit option (default key is '*' on keypad)

TOOLTYPES:
  Set CUSTOM1=1 if you want to skip intro. CUSTOM2=1 toolype activates
hidden trainer for ECS version (authors haven't included it in AGA)

HISTORY:
V1.0 - first public release (04-Jan-99 ECS, 18-Jan-99 AGA)

V1.1 - (22-Apr-99)
     * ECS version: - installation changed to files (reinstall please)
                    - trainer works after intro now
     * AGA version: - fixed problem with track pictures (there were same names
                      for different files), please reinstall
