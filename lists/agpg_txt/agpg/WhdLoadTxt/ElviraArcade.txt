This patch applies to "Elvira - The Arcade Game" by Flair Software (c) 1991 - 2 disks.
Slave requires WHDLOAD (at least V7) for run.
Game requires 1.0 MB Chip Mem (+ 1.6 other memory for preload option)

FEATURES:

- Full load from HD
- Disk protection removed
- Manual protection removed
- Decruncher relocated to Fast RAM
- Trainer key added (CUSTOM1)
- LMB problem in intro fixed
- NI/ROM icon (created by JHZ)
- Quit option (default key is '*' on keypad)

ABOUT TRAINER:
 If CUSTOM1 tooltype is set (CUSTOM1=1), pressing F5 key ingame will always
give you 9 lives.
