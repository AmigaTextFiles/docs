Short : HD-Installer for Beyond The Ice Palace V1.0
Type : game/patch
Author : Bored Seal (cossick@email.cz), ICQ:21086453

This patch applies to "Beyond The Ice Palace" by Elite (c) 1988 - 1 disk.
Slave requires WHDLOAD V10+.
Game requires 0.5 MB Chip Mem (+ 0.2 other memory for preload option)
Install requires RawDIC in path.

FEATURES:
- Full load from HD
- 24bit access fault fixed
- Trainer option added (set CUSTOM1=1 for unlimited lives)
- NI/ROM icon (created by JHZ)
- Quit option (default key is '*' on keypad)

Thanks to Stephen Misfud for disk image.
