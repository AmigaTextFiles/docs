Short : HD-Installer for Fist Fighter V1.0
Type : game/patch
Author : Bored Seal (cossick@email.cz), ICQ:21086453

This patch applies to "Fist Fighter" by Zeppelin Games (c) 1993 - 1 disk.
Slave requires WHDLOAD V10+.
Game requires 0.5 MB Chip Mem (+ 0.9 other memory for preload option)

FEATURES:

- Full load from HD
- RNC disk protection removed
- NI/ROM icon (created by JHZ)
- Quit option (your default key)

Thanks to Mark Knibbs for disk image.
