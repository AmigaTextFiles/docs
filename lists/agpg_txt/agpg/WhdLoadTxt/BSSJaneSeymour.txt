Short : HD-Installer for B.S.S. Jane Seymour V1.1
Type : game/patch
Author : Bored Seal (cossick@email.cz), ICQ:21086453,

This patch applies to "B.S.S. Jane Seymour" by Gremlin (c) 1990 - 2 disks.
Slave requires WHDLOAD V10+.
Game requires 0.5 MB Chip Mem (+ 1.8 other memory for preload option)

FEATURES:

- Full load from HD
- Load/Save redirected to HD
- Disk protection removed
- Manual protection removed
- ROM/NI icon (created by JHZ)
- Quit option (default key is '*' on keypad)

HISTORY:
V1.0 (28-Jun-1999) - first public release
V1.1 (01-Jul-1999) - savegame bug fixed (thanks Helmut)
                   - savegame format changed
                   - added CRC check to recognize unsupported versions

Thanks to Helmut Motzkau for disk images and beta testing.
