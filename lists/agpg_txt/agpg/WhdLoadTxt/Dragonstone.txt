Short : HD-Installer for Dragonstone V1.0 (23-Nov-1998)
Type : game/patch
Author : Bored Seal (cossick@email.cz)

This patch applies to "Dragonstone" by Core Design (c) 1994 - 4 disks.
Slave requires WHDLOAD (at least V7.0) for run.
Game requires 1.0 MB Chip Mem (+ 3.6 other memory for preload option)

FEATURES:

- Full loading from HD
- RNC disk protection removed
- RNC decruncher moved to Fast RAM
- NI/MWB icon (created by JHZ, ripped by Midwik)
- Quit option (your default key)

Thanks to Madn3ss for disk images.

Enjoy this great game on your hd.

Bored Seal
