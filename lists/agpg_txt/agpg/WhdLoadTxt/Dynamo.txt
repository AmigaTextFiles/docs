Short : HD-Installer for Dynamo V1.1
Type : game/patch
Author : Bored Seal (cossick@email.cz), ICQ:21086453

This patch applies to "Dynamo" by Code Masters (c) 1992 - 1 disk.
Slave requires WHDLOAD V10+.
Game requires 1.0 MB Chip Mem (+ 0.3 other memory for preload option)

SUPPORTED VERSIONS (date is taken from disk name):
- release 17.07.1992 
- release 02.11.1992

FEATURES:

- Full load from HD
- 2 versions supported
- RNC disk protection removed
- Fake IMP! decruncher moved to Fast RAM
- NI/ROM icon (created by JHZ, ripped by Midwik)
- Quit option (default key is '*' on keypad)

HISTORY:

V1.0 (04-Jun-1999) - first public release
V1.1 (28-Aug-1999) - support for second version added (thanks Helmut)
                   - install script will not check disk name
                   - added check to recognize unsupported versions

Cheat mode is available in Sweet Cheater.

Thanks to Mark Knibbs and Helmut Motzkau for disk images!
