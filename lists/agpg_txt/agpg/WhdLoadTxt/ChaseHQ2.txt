Short : HD-Installer for Chase HQ 2 V1.0
Type : game/patch
Author : Bored Seal (cossick@email.cz), ICQ:21086453

This patch applies to "Chase HQ 2" by Ocean/ICE (c) 1990 - 2 disks.
Slave requires WHDLOAD (at least V7) for run.
Game requires 0.5 MB Chip Mem (+ 1.8 other memory for preload option)

FEATURES:

- Full load from HD
- RNC disk protection removed
- Blitter waits inserted (1x)
- Keyboard handler reprogrammed
- Highscores are saved now
- Trainer and special keys options
- NI/ROM icon (created by JHZ)
- Quit option (default key is '*' on keypad)

NOTE ABOUT TRAINER:
 Set CUSTOM1=1 for unlimited credits. If CUSTOM2=1 tooltype is set, pressing
F5 will set time on 99 seconds and F6 will give you 9 nitros.

Thanks to Riccardo Cretti for disk images!
