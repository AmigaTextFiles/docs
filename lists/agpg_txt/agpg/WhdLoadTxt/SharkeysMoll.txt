Short : HD-Installer for Sharkeys Moll V1.0
Type : game/patch
Author : Bored Seal (cossick@email.cz), ICQ:21086453

This patch applies to "Sharkeys Moll" by Zeppelin Games (c) 1991 - 1 disk.
Slave requires WHDLOAD V10+.
Game requires 0.5 MB Chip Mem (+ 0.5 other memory for preload option)

FEATURES:

- Full load from HD (disk image shortened)
- RNC disk protection removed
- Trainer option (set CUSTOM1=1 for unlimited lives)
- Keyboard handler added
- Cheat key added (press F1 for 3 Molotov bottles)
- ButtonWait tooltype added for title picture
- NI/ROM icon (created by JHZ)
- Quit option (your default key)

Thanks to Richard Wagenfuhrer for disk image.
