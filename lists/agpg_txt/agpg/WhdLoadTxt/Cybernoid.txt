Short : HD-Installer for Cybernoid V1.0
Type : game/patch
Author : Bored Seal (cossick@email.cz), ICQ:21086453

This patch applies to "Cybernoid" by Hewson (c) 1988 - 1 disk.
Slave requires WHDLOAD V10+, install requires RawDIC in path.
Game requires 0.5 MB Chip Mem (+ 0.6 other memory for preload option)

This version is from Action Amiga compilation released by Gremlin in 1989.

FEATURES:

- Full load from HD
- 24bit access faults fixed (3x)
- NI/ROM icon (created by JHZ)
- Quit option (default key is '*' on keypad)

You can find the cheat mode in Sweet Cheater.

Thanks to Carlo Pirri for disk image.
