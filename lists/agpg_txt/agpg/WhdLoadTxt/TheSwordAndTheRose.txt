Short : HD-Installer for The Sword and The Rose V1.0
Type : game/patch
Author : Bored Seal (cossick@email.cz), ICQ:21086453

This patch applies to "The Sword and The Rose" by Code Masters (c) 1991 - 1 disk.
Slave requires WHDLOAD V10+ for run.
Game requires 0.5 MB Chip Mem (+ 0.5 other memory for preload option)

FEATURES:

- Full load from HD
- RNC disk protection removed
- Trainer option added (set CUSTOM1=1 tooltype for unlimited lives)
- NI/ROM icon (created by JHZ, ripped by Midwik)
- Quit option (default key is '*' on keypad)
