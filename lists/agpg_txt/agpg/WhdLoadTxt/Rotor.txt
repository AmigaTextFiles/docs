Short : HD-Installer for Rotor V1.0
Type : game/patch
Author : Bored Seal (cossick@email.cz), ICQ:21086453

This patch applies to "Rotor" by Arcane (c) 1989 - 1 disk.
Slave requires WHDLOAD V10+, install requires RawDIC in path.
Game requires 0.5 MB Chip Mem (+ 0.8 other memory for preload option)

Supported version is from Action Pack compilation (Action Sixteen edition),
released by Digital Integration in 1991.

FEATURES:

- Full load from HD
- Access fault fixed
- NI/ROM icon (created by JHZ, ripped by Midwik)
- Quit option (default key is '*' on keypad)

Big thanks to Helmut Motzkau for Action Sixteen compilation and great support!
