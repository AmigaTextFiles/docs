Short : HD-Installer for Little Puff in Dragonland V1.0 (22-Jan-1999)
Type : game/patch
Author : Bored Seal (cossick@email.cz), ICQ:21086453

This patch applies to "Little Puff in Dragonland" by Code Masters/Consult Software (c) 1989 - 1 disk.
Slave requires WHDLOAD (at least V8+) for run.
Game requires 0.5 MB Chip Mem (+ 0.6 other memory for preload option)

FEATURES:

- Full load from HD
- RNC disk protection removed
- NI/MWB icon (created by JHZ)
- Quit option (default key is '*' on keypad)

Thanks to Mark Knibbs for disk image.
