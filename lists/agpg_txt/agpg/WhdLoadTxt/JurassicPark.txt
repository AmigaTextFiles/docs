	That install applies to "Jurassic Park" © 1993 Ocean
	done by Mr.Larmer/Wanted Team.

	That install requires WHDLoad and The Patcher to be in the path.
	Both can be found in the whdload package.
	To install from a different drive than DF0: select Expert as
	installation mode.

	You need 1.0 MB ChipMem and 0.5 MB OtherMem for ECS version.

	You need 2.0 MB ChipMem and 0.5 MB OtherMem for AGA version.
	You need AGA chipset.
	You need at least a 68020 processor.

	You can quit the game by pressing F10 (key can be changed with
	Tooltype "QuitKey=")
