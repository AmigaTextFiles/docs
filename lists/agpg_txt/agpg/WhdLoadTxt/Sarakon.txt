Short : HD-Installer for Sarakon V1.1
Type : game/patch
Author : Bored Seal (cossick@email.cz), ICQ:21086453

This patch applies to "Sarakon" by Starbyte (c) 1990 - 1 disk.
Slave requires WHDLOAD V10.0+ for run.
Game requires 0.5 MB Chip Mem (+ 0.4 other memory for preload option)

SUPPORTED VERSIONS:
- Starbyte release
- Virgin release

FEATURES:
- Full load from HD
- 2 versions supported
- ROM access fixed (2x)
- OS emulation module used
- MWB icon (created by JHZ)
- Quit option (default key is '*' on keypad)

HISTORY:
V1.0 (15-Jun-1999) - first public release
V1.1 (29-Jun-1999) - support for Virgin release added (thanks Chris)

Thanks to Wolfgang Unger and Chris Vella for disk images.
