Short:  HD-Installer for Revenge of the Mutant Camels V1.0
Type:   game/patch
Author: Codetapper/Action (codetapper@hotmail.com), ICQ: 26051687

This patch applies to "Revenge of the Mutant Camels" (c) 1991 Jeff Minter.
You can download the game from Aminet:
- game/shoot/RevengeCamels.lha (NTSC version)
- game/shoot/RevengeCamelsP.lha (PAL version - much better!)

Slave requires WHDLoad V10+.
Game requires 0.5 MB Chip Mem (+ 0.2 other memory for preload option)

FEATURES:
- Full load from HD
- 2 versions supported (PAL 512k and NTSC 512k)
- MagicWB icon (created by me!)
- Empty keyboard dbf loop fixed
- Rev file can be compressed to save space (FImp, Propack etc)
- Quit option (default key is '*' on keypad)
