Short : HD-Installer for Sly Spy Secret Agent V0.9
Type : game/patch
Author : Bored Seal (cossick@email.cz), ICQ:21086453

This patch applies to "Sly Spy Secret Agent" by Ocean (c) 1989 - 2 disks.
Slave requires WHDLOAD (at least V7) for run and it's not MMU compatible.
Game requires 0.5 MB Chip Mem (+ 1.3 other memory for preload option)

FEATURES:

- Full load from HD
- RNC disk protection removed
- BUTTONWAIT tooltype added for title picture
- NI/ROM icon (created by JHZ)
- Quit option (default key is '*' on keypad)
