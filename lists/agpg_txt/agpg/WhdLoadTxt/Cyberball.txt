Short : HD-Installer for Cyberball V1.0
Type : game/patch
Author : Bored Seal (cossick@email.cz), ICQ:21086453

This patch applies to "Cyberball" by Domark/Tengen (c) 1990 - 1 disk.
Slave requires WHDLOAD V10+, install requires RawDIC in path.
Game requires 0.5 MB Chip Mem (+ 0.8 other memory for preload option)

Supported is Winning Team compilation version.

FEATURES:
- Full load from HD
- RNC disk protection removed (2x)
- Blitter waits inserted (4x)
- NI/ROM icon (created by JHZ)
- Quit option (default key is '*' on keypad)

Thanks to Helmut Motzkau for disk image and testing.
