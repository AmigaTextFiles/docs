Short : HD-Installer for G-LOC V1.0
Type : game/patch
Author : Bored Seal (cossick@email.cz), ICQ:21086453

This patch applies to "G-LOC" by US Gold (c) 1992 - 1 disk.
Slave requires WHDLOAD V10+ for run.
Game requires 0.5 MB Chip Mem (+ 0.6 other memory for preload option)

FEATURES:

- Full load from HD
- RNC disk protection removed
- RNC decruncher moved to Fast RAM
- 24bit access faults fixed
- Blitter waits inserted (12x)
- Trainer option added (set CUSTOM1=1 for unlimited time)
- BUTTONWAIT tooltype added for title picture
- NI/ROM icon (created by JHZ, ripped by Midwik)
- Quit option (default key is '*' on keypad)
