Short : HD-Installer for Escape from the Planet of the Robot Monsters V1.2
Type : game/patch
Author : Bored Seal (cossick@email.cz), ICQ:21086453

This patch applies to "Escape from the Planet of the Robot Monsters" by Tengen/Teque (c) 1990 - 1 disk.
Slave requires WHDLOAD V10+ for run.
Game requires 0.5 MB Chip Mem (+ 0.9 other memory for preload option)

SUPPORTED VERSIONS:

- original release
- Tengen's Arcade Hits Compilation
- Winning Team Compilation

FEATURES:

- Full load from HD
- 3 versions supported
- RNC disk protection removed
- Stackframe errors fixed (5x)
- Gfx fixed (bug in copperlist)
- Blitter waits inserted (53x)
- Trainer option (set CUSTOM1=1 for unlimited credits)
- NI/ROM icon (created by JHZ)
- Quit option (default key is '*' on keypad)

HISTORY:

V1.0 (27-May-1998) - first public release
V1.1 (05-Jun-1999) - support for Tengen compilation added (thanks Bert)
V1.2 (05-Jun-1999) - support for Winning Team compilation added (thanks Helmut)
