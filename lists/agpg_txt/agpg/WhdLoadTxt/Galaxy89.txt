Short : HD-Installer for Galaxy '89 V1.0
Type : game/patch
Author : Bored Seal (cossick@email.cz), ICQ:21086453

This patch applies to "Galaxy '89" by Kingsoft (c) 1989
Slave requires WHDLOAD V10+
Game requires 0.5 MB Chip Mem (+ 0.4 other memory for preload option)

FEATURES:

- Full load from HD
- Disk protection removed
- OS emulation module used
- Highscores are saved
- NI/ROM icon (created by JHZ)
- Quit option (default key is '*' on keypad)

Thanks to Helmut Motzkau for buying me original and support ;-).
