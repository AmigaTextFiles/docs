Short : HD-Installer for World Class Soccer V1.0
Type : game/patch
Author : Bored Seal (cossick@email.cz), ICQ:21086453

This patch applies to "World Class Soccer" by US Gold (c) 1991 - 1 disk.
Slave requires WHDLOAD (at least V10+) for run.
Game requires 0.5 MB Chip Mem (+ 0.7 other memory for preload option)

FEATURES:

- Full load from HD
- Disk protection removed (78 Track Master)
- Load/Save tournament from HD
- NI/ROM icon (created by JHZ and me)
- Quit option (default key is '*' on keypad)

Thanks to Chris Vella for disk image.
