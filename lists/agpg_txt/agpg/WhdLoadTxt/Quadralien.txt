Short : HD-Installer for Quadralien V1.0
Type : game/patch
Author : Bored Seal (cossick@email.cz), ICQ:21086453

This patch applies to "Quadralien" by Logotron/Astral (c) 1988 - 1 disk.
Slave requires WHDLOAD V10.0+ for run.
Game requires 0.5 MB Chip Mem (+ 0.4 other memory for preload option)

FEATURES:

- Full load from HD
- Load/Save redirected to HD
- Some useless CPU dependant loops replaced/removed
- Blitter waits inserted (3x)
- OS emulation module used
- NI/ROM icon (created by JHZ)
- Quit option (default key is '*' on keypad)

Thanks to Stephen Misfud for disk image, betatesting and media support.
