Short : HD-Installer for John Madden Football V1.0
Type : game/patch
Author : Bored Seal (cossick@email.cz), ICQ:21086453

This patch applies to "John Madden Football" by Electronic Arts (c) 1992 - 2 disks.
Slave requires WHDLOAD (at least V7) for run.
Game requires 0.5 MB Chip Mem + (+ 1.6 other memory for preload option)

FEATURES:

- Full load from HD
- Manual protection removed
- Divide by zero operations removed (4x)
- Decruncher optimised and relocated to Fast RAM
- Intro skip option added
- Access fault removed
- NI/ROM icon (created by JHZ)
- Quit option (default key is '*' on keypad)

ABOUT TOOLTYPES:
 If you're tired by watching intro and EA animations after boot, set user
tootype (CUSTOM1=1).
