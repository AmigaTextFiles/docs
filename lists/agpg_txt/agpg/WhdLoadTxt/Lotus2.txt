	That install applies to "Lotus Turbo Challenge II" © 1991 Magnetic 
	Fields Ltd. and Gremlin Ltd.
	Supported is the original release, the version from "Lotus Trilogy"
	and the Amiga Fun rerelease.

	That install requires WHDLoad and RawDIC to be in the path. Both
	can be found in the WHDLoad package.

	You need 0.5 MB ChipMem and 1.0 MB OtherMem.
	You can quit the game by pressing Help (key can be changed with
	Tooltype "QuitKey=")
	
	Features:
	- setting Tooltype "ButtonWait" the game will wait for pressing the
	  joystick fire button before a race starts
	- setting Tooltype "Custom1=1" the game configuration (players name,
	  gear type, 2 player mode...) will be saved automatically after it
	  has been changed and loaded on startup
	- setting Tooltype "Custom2=1" the highscores will be saved
	  automatically after a player has entered it
	- manual protection from the original release removed
	- some bugs removed and delays inserted
