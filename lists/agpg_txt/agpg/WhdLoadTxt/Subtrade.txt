;--- Subtrade HD install V0.9b

This installer requires the original Subtrade disk.

If you have an unsupported version please contact me.

You have to install WHDLoad first.

You need 1MB chip.

Note that this is not the final version of this patch, just a pre-release. Due to
the fact that I don't have the time to play the games I patch it's possible, that
there are still some disk accesses later in the game. I would be glad if someone
could send me a save game near the end.


;--- Features

- Protection removed
- Access faults removed
- Blitter waits added
- Saves to HD
- Return to OS (Numl)
