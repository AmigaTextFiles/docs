Short : HD-Installer for Bionic Commando V1.0
Type : game/patch
Author : Bored Seal (cossick@email.cz), ICQ:21086453

This patch applies to "Bionic Commando" by Go!/Software Creations (c) 1987 - 1 disk.
Slave requires WHDLOAD V10.0+ for run.
Game requires 0.5 MB Chip Mem (+ 0.7 other memory for preload option)

FEATURES:

- Full load from HD
- OS emulation module used
- ROM/NI icon (created by JHZ)
- Quit option (default key is '*' on keypad)

Thanks to Condor and Cyber2th for disk image and support.
