Short : HD-Installer for Seymour Goes To Hollywood V1.1
Type : game/patch
Author : Bored Seal (cossick@email.cz), ICQ:21086453

This patch applies to "Seymour Goes To Hollywood" by Code Masters/Optimus Software (c) 1992 - 1 disk.
Slave requires WHDLOAD V10+.
Game requires 1.0 MB Chip Mem (+ 0.3 other memory for preload option)

FEATURES:
- Full load from HD
- RNC disk protection removed
- IMP! decruncher moved to Fast RAM
- Trainer option (set CUSTOM1=1 for unlimited lives)
- NI/MWB icon (ripped by Midwik, created by JHZ)
- Quit option (default key is '*' on keypad)

HISTORY:
V1.0 (18-Jan-1999) - first public release
V1.1 (16-Sep-1999) - slave optimized and fixed for WHDLoad V10+
                   - decruncher speeded up
                   - added check for unsupported versions
