Short : HD-Installer for International Truck Racing V1.0
Type : game/patch
Author : Bored Seal (cossick@email.cz), ICQ:21086453

This patch applies to "International Truck Racing" by Zeppelin Games (c) 1992 - 1 disk.
Slave requires WHDLOAD V10+.
Game requires 0.5 MB Chip Mem (+ 0.9 other memory for preload option)

FEATURES:

- Full load from HD
- Keyboard handler added
- ButtonWait tooltype added for title picture
- NI/ROM icon (created by JHZ)
- Quit option (press '*' on keypad)

Thanks to Skreen for disk image.
