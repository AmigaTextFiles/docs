Short : HD-Installer for Navy Seals V1.1 (11-Jan-1999)
Type : game/patch
Author : Bored Seal (cossick@email.cz), ICQ:21086453

This patch applies to "Navy Seals" by Ocean (c) 1990 - 1 disk.
Slave requires WHDLOAD (at least V7.0) for run.
Game requires 0.5 MB Chip Mem (+ 0.9 other memory for preload option)

FEATURES:

- Full load from HD
- RNC disk protection removed
- 24bit access fault removed (1x)
- Trainer option (set CUSTOM1=1 for unlimited credits, lives and energy)
- NI/MWB icon (created by JHZ)
- Quit option (your default key)

Cheat (taken from Sweat Cheater):
  Type 'PSBOYS' as highscore enter, then press 'H' for pause and '0'
for level skip.

Thanks to Mark Knibbs for disk image.

Enjoy this decent Ocean-movie-licensed game ;).
