Short : HD-Installer for Xybots V1.0
Type : game/patch
Author : Bored Seal (cossick@email.cz), ICQ:21086453

This patch applies to "Xybots" by Tengen (c) 1989 - 1 disk.
Slave requires WHDLOAD V10+.
Game requires 0.5 MB Chip Mem (+ 0.4 other memory for preload option)
Use corresponding install script! If you have no files on the disk, use
'Install_DiskVer', 'Install_FileVer' is here to install file version.

SUPPORTED VERSIONS:
- TNT compilation version (files, there is also APB on disk)

FEATURES:

- Full load from HD
- NI/ROM icon (created by JHZ)
- Quit option (default key is '*' on keypad)

Find the cheat in Sweet Cheater.

Thanks to Carlo Pirri for disk image.
