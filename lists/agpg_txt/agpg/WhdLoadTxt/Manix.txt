Short : HD-Installer for Manix V1.0 (29-Dec-1998)
Type : game/patch
Author : Bored Seal (cossick@email.cz), ICQ:21086453

This patch applies to "Manix" by Millenium (c) 1990 - 1 disk.
Slave requires WHDLOAD (at least V7) for run.
Game requires 0.5 MB Chip Mem (+ 0.9 other memory for preload option)

FEATURES:

- Full load from HD
- RNC disk protection removed
- ROM access fixed (6x)
- Access fault fixed
- NI/MWB icon (created by JHZ)
- Quit option (your default key)

Thanks to Lempkee for disk image.
