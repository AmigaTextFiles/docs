Short : HD-Installer for Shaq-Fu V1.1
Type : game/patch
Author : Bored Seal (cossick@email.cz), ICQ:21086453

This patch applies to "Shaq-Fu" by Ocean/Dome Development (c) 1994 - 6 disks.
AGA and ECS versions are supported. Use corresponding install script.
Slave requires WHDLOAD (at least V7) for run.
AGA version requires 2.0 MB Chip Mem and AGA chipset (+ 3.7 other memory for preload option)
ECS version requires 1.0 MB Chip Mem (+ 2.8 other memory for preload option)

FEATURES:

- Full load from HD
- 2 versions supported (AGA and ECS)
- RNC disk protection removed
- RNC decruncher moved to Fast RAM
- NI/ROM icon (created by JHZ)
- Quit option (default key is '*' on keypad)

Thanks to Chris Vella  for disk images of ECS version
   and to Sven Boerger for disk images of AGA version.
