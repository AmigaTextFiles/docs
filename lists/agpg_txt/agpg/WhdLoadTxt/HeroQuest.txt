;--- Hero Quest & Witch Lord HD install V1.2

This installer requires the original Hero Quest disk. You will be asked whether
you want to install the data disk, too.

You have to install WHDLoad first.

V1.2	- Imager improved
	- Selfmodifying code removed
	- Sound player fixed
	- New icons

V1.1	- Support for data disk "Return Of The Witch Lord" added
	- More blitter waits added

Don't forget to 'format' your data disk first.
