;--- Grand Monster Slam HD install V1.3 [registered users only]

This installer requires the original Grand Monster Slam disk. It was written
for the version from the Rainbow Arts Mega Box compilation.

You have to install WHDLoad first.

V1.3	- Imager improved
	- Trainer added
	- Saves Highscores to HD
	- Icons changed
	- For registered users only

V1.2	- Slave now fully pc-relative
	- Exit key changed to Numl

V1.1	- Uses 'The Patcher' for image creation
	- Access fault removed

V1.0	- First release

If you have not yet registered WHDLoad, this patch will NOT work. You will return
to the workbench right after the title picture.

Trainer tooltype:	* Custom1 for never miss at Faulton Feeding

If you enable the trainer tooltype your scores will not be saved!

Grand Monster Slam is copyright by Rainbow Arts.
