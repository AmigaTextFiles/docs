;--- F-29 Retaliator HD install V1.1 [registered users only]

This installer requires the two original F-29 Retaliator disks. It supports two
versions.

You have to install WHDLoad first.

V1.1	- Support for another version added. Thanks to Vulture for sending
	  me his game.

If you have not yet registered WHDLoad, this patch will NOT work. You will return
to the workbench right after the title picture.

Thanks to Wolfgang Unger for sending me his original and beta testing.

F-29 Retaliator is copyright by Ocean.


;--- Features

- Supports two versions
- Buttonwait tooltype added
- Selfmodifications fixed
- Protection removed
- NTSC fix
- Saves to HD
- Return to OS (Numl)
