;--- Shinobi HD install V1.1

This installer requires the original Shinobi disk. It supports the versions from
SEGA's 'Arcade Smash Hits' and 'Super SEGA' compilation.

You have to install WHDLoad first.

V1.1	- Support for 'Super SEGA' version added. Thanks to Marc Le Douarain for
	  sending me his game.
	- New icon

To activate cheat mode type LARSXVIII while pausing (F10).

To activate music menue press Q while pausing.

Trainer tooltypes:	* Custom1 for level jump (F1)
			* Custom2 for unlimited ninja magic (Space)

If you enable a trainer tooltype your scores will not be saved!

Shinobi is copyright by SEGA.
