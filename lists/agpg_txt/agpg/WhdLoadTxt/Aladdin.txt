	That install applies to Aladdin (c) 1994 Virgin.


	That install requires WHDLoad and DIC to be in the path. Both
	can be found in the whdload package.


	You need 2 MB ChipMem and 0.5 MB OtherMem.
	You need AGA chipset.
	You can quit the game by pressing F9.
	
	
	the game has used bit #31 sometimes on some pointers which has caused
	problems on 32bit cpu's, this has been fixed.
	
	
	for cheat mode pause the game (P) and use the joystick like this:
		up,down,left,right,fire
		down,up,right,left,fire
		down,right,fire
		right,down,fire
		up,up,fire
		down,up,down,fire
	then use
		F10	complete level
		1-9	skip to level
		Z	move anywhere
		+	move faster (with Z)
		-	move slower (with Z)
