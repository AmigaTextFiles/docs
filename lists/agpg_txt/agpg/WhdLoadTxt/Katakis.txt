;--- Katakis HD install [registered users only]

This installer requires the original Katakis disk.
You have to install WHDLoad first.

You can freely download the game at http://www.factor5.com. Use ITD from
the WHDLoad package to write the adf image back to a disk.

If you have not yet registered WHDLoad, this patch will NOT work. You will
return to the workbench immediately. As long as there are only a handful of
registrations while the download counters grow week by week I will stop making
patches only limited in saving to HD.

Trainer tooltypes:	* Custom1 for unlimited lives
			* Custom2 for invulnerability

If you enable a trainer tooltype your scores will not be saved!

Katakis is copyright by Factor 5.


;--- Features

- Access fault removed
- Trainer tooltypes
- Saves Highscores to HD
- Return to OS (Numl)
