Short : HD-Installer for Colgate V1.0
Type : game/patch
Author : Bored Seal (cossick@email.cz), ICQ:21086453

This patch applies to "Colgate" by Silver Rock (c) 1992 - 1 disk.
Slave requires WHDLOAD (at least V7.0) for run.
Game requires 1.0 MB Chip Mem (+ 0.9 other memory for preload option)

FEATURES:

- Full load from HD
- Manual protection removed
- Trainer option added (set CUSTOM1=1 for unlimited lives)
- NI/MWB icon (created by JHZ)
- Quit option (your default key)

Enjoy this cool game!
