	That install applies to "Starglider 2" © 1988 Argonaut Software.


	That install requires WHDLoad to be in the path. It can be found in
	the WHDLoad package.


	History:

	1.1 - Replaced WHDLoad imager with RawDIC imager.
	    - Removed code protection for 100%.

	1.0 - Initial release.


	You need 0.5 MB ChipMem and 0.5 MB OtherMem.
	You can quit the game by pressing F9 (key can be changed with
	Tooltype "QuitKey=").


	Use the CUSTOM1 tooltype to use different savegame disks.
	(i.e. CUSTOM1=7 will use diskimage seven to save games/highscores)
