Short : HD-Installer for Eagle's Rider V1.0
Type : game/patch
Author : Bored Seal (cossick@email.cz), ICQ:21086453

This patch applies to "Eagle's Rider" by Microids (c) 1991
Slave requires WHDLOAD V10+
Game requires 0.5 MB Chip Mem (+ 1.3 other memory for preload option)

FEATURES:

- Full load from HD
- OS emulation module used
- Savegames redirected to HD
- NI/ROM icon (created by JHZ)
- Quit option (default key is '*' on keypad)

Thanks to Helmut Motzkau for disk images.
