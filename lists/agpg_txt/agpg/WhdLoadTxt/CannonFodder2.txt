	That install applies to "Cannon Fodder 2" © 1994 Sensible Software.
	The english and the german version is supported.


	That install requires WHDLoad to be in the path. It can be found in
	the WHDLoad package.


	You need 1 MB ChipMem and 0.5 MB OtherMem.
	You can quit the game by pressing F10 (key can be changed with
	Tooltype "QuitKey=").
	
	
	Savegames to harddisk are fully supported. To import savegames from
	floppy simply copy them to the "save" directory.
	Enter "JOOLS" on saving to get a 4 star general.
	Find secret key for some ammo.
