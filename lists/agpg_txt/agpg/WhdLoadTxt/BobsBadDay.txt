Short : HD-Installer for Bob's Bad Day V1.1
Type : game/patch
Author : Bored Seal (cossick@email.cz), ICQ:21086453

This patch applies to "Bob's Bad Day" by Psygnosis/Dome Development (c) 1994 - 2 disks.
Slave requires WHDLOAD (at least V7) for run.
Game requires 1.0 MB Chip Mem (+ 1.0 other memory for preload option)

FEATURES:

- Full load from HD
- 2 versions are supported
- RNC disk protection removed (Steeler version)
- RNC decruncher moved to Fast RAM
- 24bit adress error in intro fixed
- NI/ROM icon (created by JHZ)
- Quit option (default key is '*' on keypad)

Thanks to Steeler for disk images and to Carlo Pirri for second version.

Enjoy this cool game!
