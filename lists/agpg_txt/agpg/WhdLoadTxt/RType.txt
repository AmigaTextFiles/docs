;--- R-Type HD install V1.3

This installer requires the original R-Type disk. It supports three versions:
PAL, NTSC and LIGHT FORCE.

You can freely download the PAL game at http://www.factor5.com.

You have to install WHDLoad first.

V1.3	- Support for LIGHT FORCE version added (thanks to Bert for his disk)
	- Bugfix, V1.2 crashed after 3 minutes listening to the title music

V1.2	- Support for NTSC version added (thanks to Chris Vella for his image)
	- NTSC fix for PAL version added
	- Trainer tooltypes added

V1.1	- Slave now fully pc-relative

Trainer tooltypes:	* Custom1 for unlimited lives
			* Custom2 for invulnerability

R-Type is copyright by Rainbow Arts.


;--- Features

- Supports three versions
- Selfmodifying code fixed
- Access faults removed
- NTSC fix
- Trainer
- Saves Highscores to HD [registered users only]
- Return to OS (Numl)
