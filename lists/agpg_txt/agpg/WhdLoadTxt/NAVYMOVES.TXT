Short : HD-Installer for Navy Moves V1.1
Type : game/patch
Author : Bored Seal (cossick@email.cz), ICQ:21086453

This patch applies to "Navy Moves" by Dinamic Software (c) 1989 - 1 disk.
Slave requires WHDLOAD (at least V7.0) for run.
Game requires 0.5 MB Chip Mem (+ 0.9 other memory for preload option)

FEATURES:

- Full load from HD
- Code check removed at part 2
- Access faults removed (2x)
- NI/MWB icon (created by JHZ)
- Quit option (your default key)

Big thanks to Chris Vella for disk image and great support.
