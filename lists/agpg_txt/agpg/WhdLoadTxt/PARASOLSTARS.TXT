;--- Parasol Stars HD install V1.1 [registered users only]

This installer requires the original Parasol Stars disk. It was written for the
PAL version.

You have to install WHDLoad first.

If you have not yet registered WHDLoad, this patch will NOT work. You will return
to the workbench immediately.

V1.1	- Caches are now flushed to work on 060

Trainer tooltype:	* Custom1 for Leveljump (F1)

If you enable the trainer tooltype your scores will not be saved!
