Short : HD-Installer for Mean Machine V1.0
Type : game/patch
Author : Bored Seal (cossick@email.cz), ICQ:21086453

This patch applies to "Mean Machine" by Code Masters/Optimus (c) 1991 - 1 disk.
Slave requires WHDLOAD (at least V10+) for run.
Game requires 0.5 MB Chip Mem (+ 0.2 other memory for preload option)

FEATURES:

- Full load from HD
- RNC disk protection removed
- ROM/NI icon (ripped by Midwik, created by JHZ)
- Quit option (default key is '*' on keypad)

Thanks to Condor for disk image.
