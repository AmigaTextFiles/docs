Short : HD-Installer for Army Moves V1.1
Type : game/patch
Author : Bored Seal (cossick@email.cz), ICQ:21086453

This patch applies to "Army Moves" by Imagine (c) 1987 - 1 disk.
Slave requires WHDLOAD V10+ for run.
Game requires 0.5 MB Chip Mem (+ 0.3 other memory for preload option)

FEATURES:

- Full load from HD
- ButtonWait tooltype for title picture added
- ROM access faults in star generator routine fixed (2x)
- Keyboard routine added
- Trainer option (set CUSTOM1=1 for unlimited lives)
- Collision on/off key added (press 'Help' for switching your invulnerability)
- MWB/NI icon (created by JHZ, ripped by Midwik)
- Quit option (your default key)

HISTORY:

V1.0 (02-Dec-1998) - first public release
V1.1 (20-May-1999) - slave optimized (especially ROM access fix)
                   - OS emulation module removed (it was needed only for DOS_LoadSeq)
                   - trainer and collision key added
                   - fix for Snoop tooltypes added
