Short : HD-Installer for Dizzy Prince of the Yolkfolk V1.0
Type : game/patch
Author : Bored Seal (cossick@email.cz), ICQ:21086453

This patch applies to "Dizzy Prince of the Yolkfolk" by Code Masters (c) 199? - 1 disk.
Slave requires WHDLOAD V10+.
Game requires 1.0 MB Chip Mem (+ 0.3 other memory for preload option)

FEATURES:

- Full load from HD
- RNC disk protection removed
- IMP! decruncher moved to Fast RAM
- Trainer option (set CUSTOM1=1 for unlimited lives)
- NI/ROM icon (created by JHZ, ripped by Midwik)
- Quit option (default key is '*' on keypad)

Thanks to Moesy for disk image.
