Short : HD-Installer for Invest V1.1
Type : game/patch
Author : Bored Seal (cossick@email.cz), ICQ:21086453

This patch applies to "Invest" by Starbyte (c) 1989.
Slave requires WHDLOAD V10.0+ for run.
Game requires 0.5 MB Chip Mem (+ 2.0 other memory for preload option)

SUPPORTED VERSIONS: (use corresponding install script)
- original Starbyte release (2 disks)
- Amiga Spiele #15 coverdisk (1 disk)

FEATURES:
- 2 versions supported
- Full load from HD
- Load/Save redirected to HD
- OS emulation module used
- ROM/NI icon (created by JHZ)
- Quit option (default key is '*' on keypad)

HISTORY:
V1.0 (24-Jun-1999) - first public release
V1.1 (30-Jun-1999) - support for original release added (thanks Sven)
                   - savedisk is now stored on Disk.3! reinstall or rename
                     Disk.2 from old installation, please.
