Crystal Caliburn

Name: Crystal Caliburn
Genre: Misc
Hardware needed: 4MB RAM, mouse, 256 color-screenmode
Software needed: System 6.07
Description: Displays the pinball-table on one screen - without scrolling. Good graphics and sounds. Only one table available. Very hard to hold the marble on the table...
Tested on: A4000, Cyberstorm040/40MHz, Kick/WB3.0, Cybervision64, A2091-SCSI-Controller, Toshiba XM3801B-SCSI-CD-ROM drive, 22MB RAM (10MB assigned to Shapeshifter), MacOS 7.5.5,
Rating: Also good playable on A4000/030/25MHz/Cybervision64/16MB Pinball Dreams on Amiga is much much better!!!
Demo available: Don`t know
Tested by: Andreas Etzrodt
