Lode Runner

The game works but you must be sure that the CD is in the drive (copy protection). This caused some error-messages which could be clicked away.

Tested on:

  • Amiga 4000 Rev 3.1
  • CyberStorm 060/50Mhz MK1
  • Fastlane Z3
  • CyberVision 64/4MB
  • 48MB FastRAM
  • 2MB ChipRAM
  • Kickstart 3.0
  • Workbench 3.1
  • Toshiba 4x CD-ROM XM-5401TA
