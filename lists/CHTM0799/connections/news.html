*NEW* CoNnEcTiOnS HTML Version

CoNnEcTiOnS is now available in HTML Format.  Shortly after completing the
June Issue of CoNnEcTiOnS, we found an AmigaGuide to HTML Convertor on the
AMINET, although not perfect, it was enough to point us in the right
direction to create a fully-functioning HTML Version of CoNnEcTiOnS.  After
uploading this to the AMINET, we found that PC Users, using Microsoft's
Internet Explorer, couldn't read it properly, due to an incorrect Background
Colour command.  Apparently the PC's Web-Browsers default to a White
Background, whereas the AMIGA Web-Browsers default to a Grey Background,
which made the White Text in CoNnEcTiOnS disappear into the Background on
the PC!!!  This now has been corrected, and the HTML Version of CoNnEcTiOnS
now works fine on the PC.  The PC Text-File Version of CoNnEcTiOnS has now
been dropped, as the HTML Version of CoNnEcTiOnS is the nearest we can get
to a PC Executable of CoNnEcTiOnS, giving the PC User the chance to read
CoNnEcTiOnS as we originally intended it to be read.  The HTML Version of
CoNnEcTiOnS will be available, along with the AmigaGuide Version, every
month from the AMINET in DOCS/LISTS/.
