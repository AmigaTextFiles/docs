HISTORY

At the end of 1996, a young Coder called ZED worked with a little piece of
software called "SHITTERM" and System X was born.  Now 3 years later, he
has left the development of System X, and was asked for the Source Code by
another budding Coder called [R]Fox, which he was given, and development of
System X carried on.

WHAT IS THIS?

System X is a Ami-Express style BBS Software for the AMIGA, the Software is
currently available in 68000 and 68020 versions.

WHAT YOU NEED TO RUN IT

An AMIGA (obviously) - (Not tested to see if it runs on an AMIGA Emulator)
A Modem
A HardDrive (Not essential, but works a lot better)
Workbench 2.0+
Reqtools.library 38+
Fifo.library/Fifo.handler (Shell based Doors only)

FEATURES

The Archive, when you get it, contains all the Files needed to run the
Software.  There are several Utilities in the archive aswell.  The Software
also supports 7 different Door types, with more support being worked upon.
These 7 different Door types are as follows:-

1. Ami-Express BBS Doors
2. Ami-Express AREXX Doors
3. DayDream BBS Doors
4. DayDream AREXX Doors
5. MAX's/Paragon Doors
6. Shell/DOS/STDIO Doors
7. Fame BBS Doors

Other features include:-

256 Conferences
256 File areas per Conference (Total of 65,536 different File areas)
500 different Mail bases per Conference
Full XPR#?.library support 
Hydra/SModem Support
GUI Configuration
GUI User Editor
Lots of Internal Functions
MCI Command Support

INSTALLATION

It is very easy to install, you just follow the install script provided or
you can do it yourself.
