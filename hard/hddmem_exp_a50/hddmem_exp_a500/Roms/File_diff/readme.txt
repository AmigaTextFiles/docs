Этот набор программ предназначен для снижения трафика при передаче
новых версий файлов (одинаковой длины). Програмы вместе с исходниками
на SAS C - freeware.

Hапример вы имеете файл rom1 длинной 512кб и файл rom2 такой же
длинны, но 100 байт изменено.
Цель состоит в том, чтобы переслать файл rom2 вашему другу, у
которого имеется файл rom1. Для этого можно передать только
100 изменённых байт.
При этом вы набираете в shell (файлы лежат, например, в "ram:") :

diff_cut ram:rom1 ram:rom2 ram:diff
После этого получится файл изменений diff.

Чтобы посмотреть изменения можете набрать:
diff_view ram:diff


Далее пересылаете файл diff другу.
Теперь он может получить rom2 из rom1 при помощи вашего diff.

Для этого он набирает:
diff_paste ram:rom1 ram:diff ram:rom2

И теперь он получает файл rom2.

Конечно, если длинна diff превосходит длинну rom2, то все эти
программы смысла не имеют. :)
