Search in old PCB boards or buy new chip: 74F04 or any from 74x04 familly. It is very common chip. I suggest you to find some in SOP/SMD package if you want to place your Fast Amiga Sampler in cable holder. Also you need some discrete parts: 10 resistors ~30Ohm to 100Ohm, 2 pcs. ~3.3K, one trimmer ~10K. (These details are not critical, use any in 30% range :)) Also you need some capacitors: 1 pcs 4.7µF, 2 pcs. ~0.1µF and maybe one 10µF.

OK! We have all the stuff. Let's do it!
Order of the work:
  • Make PCB. For some people it is very hard work. To help them here is Portable document file (.pdf) to download and print. I think it is the best way to export PCB layout to any platform without any loss in quality. If your hands are very steady, you can build this sampler "on the air" (Good luck).
  • Solder resistors, capcitors, socket and 74F04.
  • Check if you didn't make any error.
  • Plug it to your Amiga. (Don't forget to pray God for luck) Start software in sampling/monitor mode.
  • Check for signals with osciloscope.
  • Check if proper signals are on MAX153 traces:
    1x20+5V
    2HI190
    3HI18HI
    4HI17HI
    5HI16HI
    6PULSE15HI
    7HI14HI
    8PULSE130
    9NC120...5V (2V)
    100110
    NC-no connection; +5V-Vdd,Vcc, power from Amiga; HI-logical "1", high, 2.4...5V; 0- 0V, GND, logical/analog ground; PULSE-short pulses of logical high at each sampling request; x-something from 0V to 5V.
  • Disconnect all, solder MAX153 in.
  • Plug sampler to Amiga. Check if MAX153 is not hot. Start your samplig software in sample/monitor mode. You must see some signal on your screen. Conect sampler input to some audio source (LINE OUT). You must get some sound/noice image. Mute signal and use trimmer to adjust zero.
POSSIBLE PROBLEMS:
  • NOT WORKING AT ALL- your MAX153, 74F04 or 8520 is not working. Check if 74F04 is inverting signal from Amiga with osciloscope. If so, check if MAX153 pin 9 have pulse. This means that MAX153 is working. Check if some pulses are on D0...D7 lines. If so, you CIA (8520) is dead :(
  • 50/60Hz OR HIGH FREQUENCY NOICE IN OUTPUT- check if your Amiga and audio source are properly grounded. Especially you need to ground TV.
  • BAD QUALITY OF SOUND- overload of sampler. Check your samples for square signals. If yes, use lower volume. If from LINE OUT, you need to build external atennuator (voltage divider).
  • WHEN I PLUG SOMETHING IT WORKS FOR A WHILE AND THEN SOUND DISSAPEARS- DC present in Audio lines. Use proper C separattion, proper cable load. If you sample from POWER AMPL. output use 4..8Ohm load or connect speakers (not recommended)
