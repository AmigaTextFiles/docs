Pin table

The pin table is perhaps the information you're looking for. Should be simple to read. Contains mostly the following three columns; Pin, Name & Description.

Pin Name Description
1 CLOCK Key Clock
2 GND GND
3 DATA Key Data
4 VCC +5 VDC
5 n/c Not connected
