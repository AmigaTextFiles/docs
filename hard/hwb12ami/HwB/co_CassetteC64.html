C64 Cassette

(At the computer)

6 PIN MALE EDGE at the computer.

Pin Name Dir Description
A-1 GND Ground
B-2 +5V +5 Volts DC
C-3 MOTOR Cassette Motor
D-4 READ Cassette Read
E-5 WRITE Cassette Write
F-6 SENSE Cassette Sense
Note: Direction is Computer relative Cassette.
