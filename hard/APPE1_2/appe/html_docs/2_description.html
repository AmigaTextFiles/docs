APPE v1.2
Amiga Playstation Pad Emulator
(c) Benjamin Morente Arraez - AckMAN 1999


Description

APPE is an interface capable to use PlayStation pads like as Amiga Joysticks. There's no need for any kind of software to use it, it's 100% compatible with the 100% of the soft that uses a Joystick, and works with any digital PlayStation pad on any Amiga model.

Is possible use it also on any computer that uses standard Joysticks with DB9 connector like the Commodore 64, MSX, Spectrum (requires joystick interface) Amstrad CPC, Atari... etc.

The buttons used by now are: Direction, square, X, and circle on the Playstation Pad, (Direction, Fire 1, 2 and 3)
