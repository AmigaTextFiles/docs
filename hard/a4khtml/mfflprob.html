MultiFace and FastLane Problems

When using both the MultiFace III and FastLane boards in the A4000, the FastLane should be in a lower slot, closer to the motherboard. If the MultiFace board is lower than the FastLane, using the MultiFace software's PIT0: device will cause the machine to lock up.

See also: MultiFaceCard III Reference and FastLane Reference.
