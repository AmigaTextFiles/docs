                             -- SAPEP History --
Version 1.0         Released September 10th 1991
    Unit could only do limited types of eproms.

Version 2.0beta     Internal release October 29th 1992

Version 2.0         Released December 14th 1992
    Completely new software and hardware.  Unit can program pretty much any
standard 27xxx type eprom.

o Known Bug:  File path handling. Doesn't like files in directories.
o Known Bug:  Uses all-ANSI file handling.  Not very Amiga-friendly.
