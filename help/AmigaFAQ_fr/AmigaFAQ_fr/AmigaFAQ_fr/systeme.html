Le système d'exploitation (système)

Quelle est la dernière version du système Amiga ?

la dernière est la version 3.1, aussi connu comme "version 40".

Quelle version du système ce trouve dans mon ordinateur ?

Il y a plusieurs manières de retrouver quelle version vous utilisez.

Choisissez le menu "Version" du Workbench, une requête apparaitra, et vous indiquera la version de la ROM et du Workbench. Ou alors ouvrez un shell et tappez la commande "version", vous verrez apparaître les numéros de version. Les numéros de version sont généralement indiqués sur l'écran au démarrage de l'ordinateur. Les versions sont souvent sous la forme de numéros comme 37, 38,... A transcrire de la manière suivante: 

40 = 3.1
39 = 3.0
38 = 2.1
37 = 2.04
36 = 2.0
35 = 1.4
34 = 1.3
33 = 1.2
par exemple, si je tappe "version" dans un shell, et que j'obtiens: 
> Kickstart 40.63, Workbench 40.42
Cela signifie que j'utilise la version 3.1

Est-ce que je dois passer à la version 3.1 ?

réponse courte: faites-le tout de suite.

Si vous utilisez la version 1.3, ou plus ancien, le fait de passer à la 3.1 vous fera comme si vous aviez un nouvel ordinateur. Il a un meilleur aspect, il fonctionne mieux, et il a plus de possibilités. La plupart des nouveaux programmes ne fonctionnent pas avec la version 1.3. , mais tous fonctionnent avec la 3.1, il est difficile d'aller sur Internet avec la version 1.3 par exemple.

Si vous avez la version 2.0 ou 2.04 la réponse est également oui. l'aspect n'est pas très différent, mais des nouvelles fonctions le rendent très intéressant, comme par exemple le support de multiples langues autres que l'anglais grâce à la Locale.

Une autre possibilité est d'utiliser un kit pour la version 2.1. Toutefois, vu le prix relativement élevé de ce kit, vous avez intérêt à économiser quelques francs pour passer directement a la version 3.1, d'autant plus que la version 2.1 ne sera certainement plus supportée une fois que de nouvelles versions seront sorties.

le passage de la version 2.1 à la 3.1 ressemble plutot à une mise à jour, mais je la conseille tout de même dans tous les cas.

Passer de la version 3.0 à la 3.1 est moins important (vous ne pouvez pas passer à la version 3.0, elle est montée en standard dans les A1200 et A4000) S'il est intéressant pour vous de payer le prix ou si vous pensez developper des programmes, je vous recommande la mise à jour. Si vous êtes une de ces personnes qui adorent avoir le dernier cri, faites-le. Sinon, si vous avez le choix entre un kit 3.1 ou un peu plus de mémoire, il est plus intéressant de rajouter de la mémoire.

Cependant vous pouvez utiliser le kit logiciel 3.1 avec les ROM 3.0, vous perdrez quelques rares possibilités, et il revient un peu moins cher d'acheter uniquement les disquettes.

Qu'implique le passage à une nouvelle version ?

Le système d'exploitation (système) de l'Amiga comporte une série de programmes distribués sur disquette, et des puces de mémoire morte (ROM) que vous devez installer à l'intérieur de l'ordinateur. la ROM est connue sous le nom de "Kickstart", elle contient la partie principale du système qui doit être disponible à tout moment.

Pour passer à une nouvelle version vous devez remplacer la ou les ROM qui se trouvent dans l'ordinateur, sauf si vous passez de la version 2.04 à la version 2.1, vu qu'elle utilise les ROM 2.04.

Vous pouvez aussi passer à la version 3.1 en utilisant les ROM 3.0, vous perdrez quelques rares fonctionalités.

Quelles version du système peuvent fonctionner sur mon ordinateur ?

Vous pouvez changer de version sur n'importe quel Amiga, à part le A600 sur lequel les ROM sont soudées en surface et il n'existe pas de ROM de remplacement soudée, et le A1000 qui est dépassé.

Le kit 3.1 existe en une version pour A500/A2000, pour A1200, et pour A3000/A4000, Lorsque vous commandez le kit vous devez préciser quel modèle Amiga vous utilisez.

La version 3.1 est une version corrigée de la 3.0 - qui se trouve dans les A1200 et A4000 - il n'existe pas de ROM 3.0 pour les autres machines.
