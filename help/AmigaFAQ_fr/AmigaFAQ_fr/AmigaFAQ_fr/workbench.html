Utiliser le Workbench

Je clique sur l'icône "install" et il me dit "Impossible d'ouvrir l'outil installer", pourquoi ?

installer est un programme utilisé pour installer les logiciels sur votre disque dur. Il s'agit en fait d'un simple langage de programmation, qui est utilisé pour créer des routines d'installation utilisant toutes la même interface, de cette manière lorsque vous installez des programmes tout vous paraît familier.

Malheureusement, quand vous installez le système d'exploitation sur le disque dur, l'outil installer n'est pas copié pour vous. De ce fait n'importe quel programme qui s'installe avec un script installer, mais n'inclus pas le programme, s'attend a le trouver dans le repertoire C:, ou il ne se trouvera pas.

Comment est-ce que je peut changer l'image d'une icône ?

Vous devez utiliser le programme IconEdit, qui se trouve dans le répertoire Tools, il vous permet de lire l'image d'un icône et la modifier.
