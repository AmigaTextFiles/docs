Utiliser le Shell

Comment ouvrir une fenêtre shell

C'est relativement simple. Ouvrez le répertoire System de votre partition Workbench, et double-cliquez sur l'icône Shell.

Quelles sont les commandes Shell les plus importantes ?

Voila quelques unes des commandes Shell les plus importantes:

dir
affiche la liste des fichiers du répertoire courant

cd < répertoire ou volume >
Vous permet de changer le répertoire courant, n'oubliez pas que les noms de volume doivent être suivis de deux points (:), cette commande n'est pas nécessaire si vous utilisez Workbench 2.1 ou plus, il vous suffit de saisir le nom du répertoire ou du volume.

copy < source > < destination >
cette commande permet de copier fichiers ou des répertoires d'une place vers une autre.

delete < fichier ou répertoire >
cette commande vous permet d'effacer le fichier ou le répertoire spécifié. Pour supprimer un répertoire ainsi que les fichiers qu'il contient utilisez la commande suivante:
delete < fichier ou répertoire > all
dans le cas contraire la commande refusera d'effacer un répertoire qui contient encore des fichiers.

makedir < nom >
vous permet de créer un répertoire avec le nom specifié

Vous pouvez aussi utiliser la plupart des programmes graphiques à partir d'un Shell, par exemple pour utiliser Memacs, dans le répertoire Tools de la partition Workbench, saisissez:
cd sys:tools
memacs

Pour fermer une fenêtre Shell, utilisez la commande suivante:
endshell

Pour ouvrir une deuxième fenêtre Shell, utilisez la commande suivante:
newshell
