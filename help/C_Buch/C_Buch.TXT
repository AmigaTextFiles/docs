C und C++ Literatur für den AMIGA
=================================


Die nachfolgende Liste enthält eine Auswahl an Büchern zur 
Programmierung in C, C++, des Amiga Betriebssystems und weiteren 
Themen wie z.B. Dateiformate, Softwareentwicklung und 
Grafikprogrammierung.



Programmieren in C++

Die C++ Programmiersprache
B. Stroustrup
4 Auflage, 1994, 729 Seiten, 90 DM
Addison-Wesley Verlag
ISBN 3-89319-386-3
Amerikanische Originalausgabe:
ISBN 0-201-53992-6

Das bekannte Standardwerk des Erfinders der Programmiersprache C++ 
wurde in der vorliegenden Fassung um die bislang beschlossenen 
Entwürfe zur ANSI-/ISO-Standardisierung erweitert.


Design und Entwicklung von C++
Bjarne Stroustrup
1994, 594 Seiten,  80 DM
Addison-Wesley Verlag
ISBN 3-89319-755-9
Amerikanische Originalausgabe: 
ISBN 0-201-54330-3

Der Autor zeigt, welche Problematiken und Zielsetzungen zur Spachidee 
C++ führten und welche Erfahrungen aus der praktischen Arbeit auf die 
Entwicklung der Sprache Einfluß nahmen. Die Darstellung der Hintergründe 
liefert logische "Brücken" für das Verständnis von Sprachdetails. 
Für Fortgeschrittene und Profis.


C++ lernen und beherrschen: Einführung und Leitfaden
S. B. Lippmann
2. Auflage 1992, 624 Seiten, 89 DM 
Addison-Wesley Verlag
ISBN 3-89319-375-8

Dieses Buch bietet eine leicht verständliche Einführung in die Sprache 
C++. Die umfassende Darstellung enthält zahlreiche Beispiele. 
Für Einsteiger und Fortgeschrittene.


Objektorientiertes Programmieren in C++
Von der Klasse zur Klassenbibliothek
Nicolai Josuttis
1994, 553 Seiten, 80 DM
Addison-Wesley Verlag
ISBN 3-89319-637-4

Alle Sprachmittel von C++ werden erstmalig in einem integrierten Konzept 
erklärt. Dabei steht die Idee im Vordergrund, die Konzepte von C++ von 
vornherein unter dem Aspekt der objektorientierten Programmierung kennen-
zulernen, um so die neuen Sprachmittel optimal einzusetzen. Auf die 
Probleme bei der praktischen Umsetzung der Konzepte wird eingegangen. 
Sprachkenntnisse in C werden vorausgesetzt. 
Für Fortgeschrittene und Profis.


Effektiv C++ programmieren
50 Möglichkeiten zur Verbesserung Ihrer Programme
Scott Meyers
2. Auflage 1995, 342 Seiten, 60 DM
Addison-Wesley Verlag
ISBN 3-89319-539-4
Amerikanische Originalausgabe:
ISBN 0-201-56364-9

Das Buch präsentiert 50 eindeutige Regeln, effiziente und portable Software 
zu erstellen. Sie können so von den Erfahrungen professioneller C++-
Entwickler lernen. Zu jeder Regel gibt es Beispiel, die den Umgang mit 
dieser Regel erklären. Der Autor setzt Grundkenntnisse in C++ voraus. 
Für Profis.
