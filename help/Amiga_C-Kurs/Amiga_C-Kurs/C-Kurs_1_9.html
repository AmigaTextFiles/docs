Schleifen mit do-while


Die do-while Schleife ist im Gegensatz zur while-Schleife eine Fußgesteuerte Schleife. Das bedeutet, daß die Laufbedingung erst am Ende der Schleife geprüft wird. Die do-while-Schleife läuft also mindestens einmal durch, auch wenn die Laufbedingung von vorne herein verletzt ist.

Der Syntax einer do-while-Schleife ist:

  do Anweisung while (Ausdruck) ;

Die Verwendung der do-while Schleife wird in Listing 1.10 demonstriert:

/*  Listing 1.10
 *  Schleifen mit do-while
 *  Die do-while-Schleife ist "Fußgesteuert" und läuft mindestens
 *  einmal durch.
 */

#include <stdio.h>

int main(void)
{
  int n,zahl;           // Variable für den Zähler
  n=1;
  zahl=1;

  do // Anfang der while-Schleife
   {

    printf("Dies ist Durchlauf Nummer %d.\n",n);
    printf("Bitte geben Sie eine Zahl ein (Ende mit 0): ");

    scanf("%d",&zahl);  // Variable zahl einlesen
    printf("Sie haben die Zahl %d eingegeben.\n\n",zahl);

    n++;  // Zähler um 1 erhöhen
  } while (zahl != 0);  // Ende der while-Schleife
  printf("Auf wiedersehen!");

  return 0;
}

Am Ende unserer do-while-Schleife überprüfen wir mit "(zahl != 0)", ob die Variable zahl einen Wert ungleich 0 (Vergleichsoperator "!=") angenommen hat. Wenn die Aussage "zahl ungleich 0" wahr ist, wiederholen wir die Schleife, ansonsten (d.h. die Variable zahl enthält den Wert 0) nicht.

Initialisieren Sie einmal die Variable zahl mit dem Wert 0. Die Schleife läuft trotzdem (mindestens) einmal durch. Ob sie ein weiteres mal durchlaufen wird, hängt dann von Ihrer Eingabe ab.

Zur Erinnerung: Eine while-Schleife ist "Kopfgesteuert", d.h. es wird erst überprüft, ob die Laufbedingung erfüllt ist, oder nicht. Dementsprechend wird dann entschieden, ob die Anweisungen im Schleifenrumpf ausgeführt werden, oder nicht.

Bei der "Fußgesteuerten" do-while-Schleife hingegen wird der Schleifenrumpf immer mindestens einmal ausgeführt. Erst am Ende der Schleife wird dann überprüft, ob die Schleife erneut ausgeführt werden soll oder nicht.
