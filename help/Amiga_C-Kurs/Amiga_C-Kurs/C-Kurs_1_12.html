Der sizeof Operator


Die kleinste Einheit im Speicher eines Computers ist das Bit. Dieses kann entweder den Wert 0 oder 1 annehmen. 8 Bit werden als 1 Byte bezeichnet.

Um herauszufinden, wieviel Speicherplatz ein Datentyp oder eine Variable im Speicher braucht, stellt uns die Sprache C den sizeof Operator bereit. Dieser ermittelt, wieviel Byte ein Objekt im Speicher belegt.

Listing 1.13 demonstriert die Verwendung des sizeof Operators anhand einiger elementarer Datentypen:

/*  Listing 1.13
 *  Der sizeof Operator
 */

#include <stdio.h>

int main(void)
{
  printf("Typ            Byte\n\n");
  printf("int            %d\n",sizeof(int));
  printf("float          %d\n",sizeof(float));
  printf("double         %d\n",sizeof(double));
  printf("long           %d\n",sizeof(long));
  printf("char           %d\n",sizeof(char));
  printf("string \"Hello\" %d\n",sizeof("Hello"));

  return 0;
}

Das obige Programm erzeugt folgende Ausgabe: 

Typ            Byte

int            4
float          4
double         8
long           4
char           1
string "Hello" 6

Wir sehen z.B. daß eine Variable vom Typ Integer 4 Byte, also 32 Bit (4 mal 8 Bit) belegt. Eine Variable vom Typ char belegt genau ein Byte, also 8 Bit. Beim String "Hello" kommt zu den 5 Byte für die einzelnen Buchstaben noch ein zusätzliches Byte, das sogenannte Nullzeichen \0 hinzu. Dieses markiert das Ende des Strings im Speicher.

Wieviel Speicherplatz ein bestimmter Datentyp belegt, ist in der Sprache C nicht normiert. Es kann also durchaus sein, daß Sie auf unterschiedlichen Zielsystemen unterschiedliche Werte erhalten.
