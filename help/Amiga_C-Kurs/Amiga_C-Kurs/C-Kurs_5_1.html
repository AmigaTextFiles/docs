Lokale Variablen


In der Lektion über Bezeichner, Datentypen und Variablen haben wir gelernt, wie man Variablen deklariert und initialisiert (= ihnen einen Anfangswert zuweist). Im Abschnitt über Funktionen haben wir gelernt, wie man einer Funktion Parameter übergibt. Dabei haben wir uns allerdings keine Gedanken über die Gültigkeit und Lebensdauer von Variablen gemacht.

Man unterscheidet zwischen globalen und lokalen Variablen. Lokale Variablen sind nur innerhalb einer Funktion gültig. Globale Variablen sind im gesamten Programm gültig.

Beachten Sie, daß auch die in der Funktion main deklarierten Variablen nur innerhalb dieser Funktion (= lokal) gültig sind. Wenn man in einer anderen Funktion eine Variable gleichen Namens hat, so ist dies nicht die selbe.

in Listing 5.1 demonstrieren wir die Verwendung von lokalen Variablen:

/*  Listing 5.1
 *  Lokale Variablen
 */
#include <stdio.h>

// Funktionsprototyp
 void print_quadrat(int x);

int main(void)
{
  int x;  // Variable x ist lokal in der funktion main
  x=5;

  printf("Der Wert der Variablen x ist %d.\n",x);
  print_quadrat(x);
  printf("Der Wert der Variablen x ist %d.\n",x);

  return 0;

}

/*  print_quadrat schreibt das Quadrat von x an die
 *  Standardausgabe.
 */
 void print_quadrat(int x)
{
  // Dieses x ist lokal in der Funktion print_quadrat
   x = x*x;
  printf("Das Quadrat von x ist %d.\n",x);
}

Innerhalb unserer Funktion main deklarieren wir eine Variable Namens x und weisen ihr den Wert 5 zu. Mit printf("Der Wert der Variablen x ist %d.\n",x); geben wir den Wert dieser Variable in der Standardausgabe aus.

Als Eingabeparameter unserer Funktion print_quadrat haben wir ebenfalls eine Variable Namens x. Innerhalb dieser Funktion rechnen wir mit x=x*x das Quadrat von x aus. Nach dieser Zeile enthält also die lokale Variable x, die innerhalb der Funktion print_quadrat gültig ist, den Wert des Quadrats von x. Wenn wir z.b. 5 eingeben, enthält die lokale Variable x nach der Zeile x=x*x; den Wert 25. Diesen Wert geben wir dann innerhalb der Funktion mit printf aus.

Da wir aber nur der lokalen Variable x, die nur innerhalb der Funktion print_quadrat gültig ist, einen neuen Wert zugewiesen haben, bleibt der Wert der Variable x, die nur lokal innerhalb der Funktion main gültig ist, der selbe (wir haben diese Variable mit dem Wert 5 initialisiert).

Die Ausgabe des obigen Programms sieht also wie folgt aus: 

  Der Wert der Variablen x ist 5.
  Das Quadrat von x ist 25.
  Der Wert der Variablen x ist 5.

Die erste Zeile wurde von der Funktion main ausgegeben, die zweite Zeile von unserer Funktion print_quadrat. Die letzte Zeile wurde wiederum von der Funktion main ausgegeben. Der Wert der in der Funktion main lokalen Variablen x hat sich durch die Berechnung innerhalb der Funktion print_quadrat also nicht geändert.

Der Unterschied sollte jetzt klar sein: Obwohl wir sowohl in der Funktion main, als auch in der Funktion print_quadrat jeweils eine Variable Namens x deklariert haben, sind diese beiden Variablen nicht identisch, da x jeweils nur lokal innerhalb dieser Funktionen deklariert wurde.
