Aufzählungstypen


Eine Aufzählung (englische Bezeichnung: enumeration) ist ein Typ, der eine Menge durch den Anwender spezifizierter Werte speichern kann.

In Listing 3.1 definieren wir den Aufzählungstyp Sonnensystem:

 // Aufzählungstyp: Sonnensystem. Die Nummerierung beginnt mit 0 (Sonne)
   enum Sonnensystem {Sonne,Merkur,Venus,Erde,Mars,Jupiter,Saturn,Uranus,Neptun,Pluto};

Die Aufzählung wird fast genauso wie ein int benutzt. Standardmäßig werden bei enum-Typen den einzelnen Einträgen Werte von 0 an aufsteigend zugewiesen.

In den geschweiften Klammern zählen wir die Planeten von innen nach außen der Reihe nach, jeweils durch Komma getrennt, auf. Die Sonne nehmen wir dabei als 0. Element mit in unsere Aufzählung auf. Die Sonne ist also Nummer 0, der Merkur ist Nummer 1, die Venus Nummer 2 usw. .

Jede Aufzählung ist ein unterschiedlicher Typ. In unserem Beispiel ist Erde vom Typ Sonnensystem.

In Listing 3.1 wird gezeigt, daß man Aufzählungstypen wie int benutzen kann:

/*  Listing 3.1
 *  Aufzählungstypen
 */
#include <stdio.h>

int main(void)
{
  // Aufzählungstyp: Sonnensystem. Die Nummerierung beginnt mit 0 (Sonne)
   enum Sonnensystem {Sonne,Merkur,Venus,Erde,Mars,Jupiter,Saturn,Uranus,Neptun,Pluto};

  int Eingabe;

  printf("Wählen Sie einen Planeten aus dem Sonnensystem.\n");
  printf("Geben Sie eine Zahl von 0 bis 9 ein (0 für die Sonne): ");
  scanf("%d",&Eingabe);

  /*   Bei dieser switch Verzweigung verhält sich unser Aufzählungstyp
   *   Sonnensystem wie eine Variable vom Typ int.
   */
   switch (Eingabe)
  {
    case Sonne:
      printf("Die Sonne ist ein G2-Stern, Typ orange.\n");
    break;

    case Merkur:
      printf("Der Merkur ist der Sonne an nächsten.\n");
    break;

    case Venus:
      printf("Die Venus ist der %d. Planet des Sonnensystems.\n",Venus);
    break;

    case Erde:
      printf("Die Erde ist der Heimatplanet der Menschen.\n");
    break;

    case Mars:
      printf("Der Mars wird auf Grund seiner Farbe ");
      printf("auch \"der rote Planet\" genannt.\n");
    break;

    case Jupiter:
      printf("Der Jupiter ist ein Gasplanet.\n");
    break;

    case Saturn:
      printf("Der Saturn hat deutlich erkennbare Ringe.\n");
    break;

    case Uranus:
      printf("Der Uranus umkreist die Sonne einmal in 84 Erdenjahren.\n");
    break;

    case Neptun:
      printf("Neptuns blaue Farbe kommt daher, daß Methan in seiner ");
      printf("Atmosphäre rotes Licht absorbiert.\n");
    break;

    case Pluto:
      printf("Bei Pluto streiten sich die Experten, ob man ihn auf Grund ");
      printf("seiner geringen Größe überhaupt noch als Planet zählen kann.\n");
    break;

    default:
      printf("Fehlerhafte Eingabe.\n");

  }

  return 0;
}

Unsere Eingabe-Variable ist vom Typ int. Jetzt erwarten wir eigentlich, daß wir auch bei den case Marken unser switch Verzweigung mit int vergleichen, also z.B. case 0:, case 1: usw. . Statt dessen können wir aber die Namen verwenden, die wir zuvor in unserem Aufzählungstyp Sonnensystem festgelegt haben. Deshalb können wir case Sonne: statt case 0: schreiben, case Merkur: statt case 1: usw. .

Daß die einzelnen Elemente unseres Aufzählungstyps Sonnensystem tatsächlich vom Typ int sind, sehen wir hier:

    case Venus:
      printf("Die Venus ist der %d. Planet des Sonnensystems.\n",Venus);
    break;

Bei dieser printf Anweisung wird tatsächlich der Text 

Die Venus ist der 2. Planet des Sonnensystems.
ausgegeben. Wir erinnern uns: In unserem Aufzählungstyp Sonnensystem haben wir Venus als 2. Element aufgezählt. Also wurde der Konstanten Venus der Wert 2 (vom Typ int) zugewiesen.
