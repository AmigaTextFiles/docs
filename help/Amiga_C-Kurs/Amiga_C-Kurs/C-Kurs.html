Einleitung


Willkommen beim Amiga C Kurs für Einsteiger. Dieser Kurs soll dazu dienen, Anfänger mit garkeiner oder nur geringer Programmiererfahrung an die Programmiersprache C heranzuführen. Dabei soll auch das weit verbreitete Vorurteil C sei zu schwierig, weil zu kryptisch aus dem Weg geräumt werden. Vorkenntnisse sind nicht erforderlich.

Wir beginnen mit einer kurzen Einführung in die Entwicklerumgebung von StormC 3.0. Dieser Compiler wurde gewählt, weil er einfach zu bedienen und weit verbreitet ist. Die Vollversion von StormC 3.0 befindet sich als Beigabe auf der AmigaOS 3.5 Developer CD, die bei jedem Amiga-Händler erhältlich ist. Auf der Developer CD befindet sich neben vielen Beispielsources auch das "Rom Kernel Reference Manual" als Amiga-Guide. Das "Rom Kernel Reference Manual" ist die Referenz-Anleitung zur AmigaOS API. Die gedruckte Version ist leider nur schwer auf dem Gebrauchtmarkt aufzutreiben. Schon deshalb empfiehlt sich der Kauf dieser CD Rom.

Sie können zu diesem Kurs auch einen beliebigen anderen C-Compiler wie z.B. den frei erhältlichen GCC benutzen. Es sei aber gesagt, daß die Installation von GCC gerade für Anfänger schwierig ist.

Dieser Kurs wird im Laufe der Zeit noch erweitert. Schauen Sie also ruhig bei Gelegenheit wieder vorbei.
