Verzweigungen mit if else


Oft benötigt man beim Programmieren Fallunterscheidungen der Form "Falls die Bedingung zutrifft, mache dies, ansonsten mache jedes".

In unserem nächsten Beispiel suchen wir das Maximum zweier Zahlen.

/*  Listing 1.5
 *  Verzweigungen mit if else
 *  Verwendung von Anweisungsblöcken
 */

#include <stdio.h>

int main(void)
{
  int a,b,min,max;

  printf("Geben Sie bitte zwei Zahlen ein:\n");
  scanf("%d %d",&a,&b);  // Variablen a und b einlesen
  if (a>b)
  {
     max = a;  // Falls a größer b ist, ist a das Maximum
      min = b;  // und b das Minimum
   }
  else
  {
     max = b;  // Ansonsten ist b das Maximum
      min = a;  // und a das Minimum
   }

  printf("%d ist größer als %d.",max,min);

  return 0;
}

Zuerst lesen wir mit Hilfe der Funktion scanf zwei Zahlen a und b ein. scanf ist das Gegenstück von printf zum Einlesen von Variablen aus der Standardeingabe.

Dann vergleichen wir die Variablen a und b. Dazu benutzen wir den Vergleichsoperator ">". Dabei gibt es zwei mögliche Ergebnisse: Entweder die Aussage "a ist größer als b" ist wahr, oder falsch.

Falls die Aussage, die hinter dem Schlüsselwort if steht wahr ist, wird die nachfolgende Anweisung ausgeführt. In unserem Beispiel wird der Variablen "max" der Wert der Variablen a zugewiesen.

Die Anweisung hinter else legt fest, was passieren soll, falls die Aussage hinter if falsch ist. Im obigen Beispiel kommt nurnoch die Variable b als Maximum in Frage, falls die Aussage "(a>b)" nicht zutrifft. Also weisen wir in diesem Fall der Variablen "max" den Wert der Variablen b zu.

Der else-Zweig ist optional. Wenn Sie eine nur Fallunterscheidung der Form "Wenn die Bedingung zutrifft, dann mache Folgendes" brauchen, lassen Sie das else weg und das Programm geht wie gewohnt weiter.

Falls Sie die Ausführung mehrerer Anweisungen von einer Bedingung abhängig machen wollen, müßen Sie einen sogenannten Anweisungsblock benutzen. Ein Anweisungsblock wird mit einer geöffneten geschweiften Klammer ({) eingeleitet und mit einer geschlossenen geschweiften Klammer (}) abgeschlossen. Alle Anweisungen, die zwischen den geschweiften Klammern stehen, gehören zum Anweisungsblock.

Im Listing 1.5 wird der auf if folgende Anweisungsblock ausgeführt, falls die Bedingung "(a>b)" zutrifft, ansonsten der Anweisungsblock nach else.
