Arithmetische Operatoren


Die arithmetischen Operatoren von C für die Grundrechenarten Addition (+), Subtraktion(-), Multiplikation (*) und Division (/) sind eigentlich selbsterkärend. Schauen wir uns das nächste Beispielprogramm an:

/*  Listing 1.3
 *  Demonstriert einige Arithmetischen Operatoren
 */

#include <stdio.h>

int main(void)
{
  int a,b,c;

  a=12;
  b=2;
  c=3;

  printf("a=%d b=%d c=%d\n",a,b,c);
  printf("a + b ergibt %d\n", a+b);
  printf("a - b ergibt %d\n", a-b);
  printf("a * b ergibt %d\n", a*b);
  printf("a / c ergibt %d\n", a/c);
  printf("a * b / c ergibt %d\n", a*b/c);
  printf("a + b * c ergibt %d\n", a+b*c);
  printf("(a + b) * c ergibt %d\n", (a+b)*c);

  return 0;
}

Wenn Sie dieses Beispielprogramm compilieren und starten, sieht die Ausgabe so aus: 

  a=12 b=2 c=3
  a + b ergibt 14
  a - b ergibt 10
  a * b ergibt 24
  a / c ergibt 4
  a * b / c ergibt 8
  a + b * c ergibt 18
  (a + b) * c ergibt 42

Wie wir sehen, gelten auch in C die Prioritätsregeln der Mathematik, d.h. "Punkt vor Strich". Auch Klammersetzung können wir wie in der Mathematik gewohnt verwenden.

Gemäß den Gesetzen der Mathematik ist eine Division durch 0 nicht erlaubt. Dies ist ein häufiger Laufzeitfehler, da während das Programm läuft durchaus eine Variable, durch die Sie teilen, Null werden kann. Um solche Fehler zu vermeiden, kann man beispielsweise abfragen, ob die Variable, durch die man teilen möchte, den Wert 0 angenommen hat oder nicht. Wie man solche Fallunterscheidungen macht, werden wir später lernen.

Für die Schreibfaulen unter uns gibt es noch verkürzte Notationen:

// Beispiele für verkürzte Notationen:
a += b;  // Entspricht a = a + b
 a -= b;  // Entspricht a = a - b

Falls Sie Vorkenntnisse in BASIC haben, kennen Sie vielleicht INC und DEC. Die Inkrement- und Dekrement-Operatoren sehen in C wie folgt aus:

// Inkrement und Dekrement
a++;  // Entspricht a = a + 1
 a--;  // Entspricht a = a - 1

Ein weiterer arithmetischer Operator ist der Modulo-Operator (%). Er gibt den Rest der Division zurück.

// Der Modulo-Operator
a = 6;
b = 4;

printf("%d",a%b);  // Ausgabe: 2

Im obigen Beispiel rechnen wir 6 durch 4. Die 4 passt in 6 nur 1 mal hinein, der Rest ist 2.

Um das Vorzeichen zu wechseln, schreiben wir einfach ein Minus vor die Variable:

// Beispiel für Vorzeichenwechsel:
a = -5;
b = -a;

printf("a=%d b=%d",a,b);  // Ausgabe: a=-5 b=5

Bitte beachten Sie, daß das Ergebnis einer Division zweier Variablen vom Typ Integer wieder Integer ist:

// Division von Ganzzahlen liefert ganzzahliges Ergebnis
int a,b,Ergebnis;

a=6;
b=4;

Ergebnis = a / b;

printf("%d",Ergebnis);  // Ausgabe: 1

Wenn Sie für Ihre Berechnungen Nachkommastellen brauchen, benutzen Sie den Typ float bzw. double:

// Division von Fließkommazahlen:
float a,b,Ergebnis;

a=6.0;
b=4.0;

Ergebnis = a / b;

printf("%f",Ergebnis);  // Ausgabe: 1.5

Mathematische Funktionen, wie Potenz, Wurzel, Logarithmus und die trigonometrischen Funktionen (Sinus, Cosinus, etc.) sind in der Header-Datei math.h definert.
