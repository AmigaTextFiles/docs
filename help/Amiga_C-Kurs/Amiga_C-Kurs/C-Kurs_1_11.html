Aussagelogische Operatoren


Falls Sie Vorkenntnisse in BASIC haben, kennen Sie vielleicht die BASIC-Schlüsselwörter AND, OR und NOT. Diese Aussagelogischen Operatoren gibt es in C auch. Hier eine tabellarische Übersicht:

Logische Operatoren
Operator Bedeutung Beispiel Ergebnis: 1 (wahr) oder 0 (falsch)
&& logisches UND x&&y 1, falls x und y ungleich 0
|| logisches ODER x||y 1, falls x oder y oder beide ungleich 0
! logisches NICHT !x 1, falls x gleich 0,
In anderen Fällen ist das Ergebnis 0.

Die Verwendung von UND- (&&) und ODER- (||) Verknüpfungen demonstriert Listing 1.11.1:

/*  Listing 1.11.1
 *  Aussagelogische Operatoren
 */

#include <stdio.h>

int main(void)
{
  char Zeichen;
  int a,b,Fehler;

  Fehler = 0;

  printf("Drücken Sie bitte die Taste A auf Ihrer Tastatur!\n");
  Zeichen = getchar();

  // Beispiel für ODER-Verknüpfung
   if (Zeichen == 'a' || Zeichen == 'A')
  {
    printf("Gut gemacht!\n\n");
  }
  else
  {
    printf("Das war leider die falsche Taste.\n\n");
    Fehler++;
  }

  printf("Geben Sie jetzt zwei ganze Zahlen ein.\n");
  printf("Dabei soll die erste kleiner wie die zweite sein ");
  printf("und beide zusammengezählt weniger wie 10 ergeben.\n");

  scanf("%d %d",&a,&b);

  // Beispiel für UND-Verknüpfung
   if (a<b && a+b<10)
  {
    printf("Sehr gut!\n\n");
  }
  else
  {
    printf("Das war leider falsch.\n\n");
    Fehler++;
  }

  printf("Fertig. Sie haben %d Fehler gemacht.\n",Fehler);


  return 0;

}

Schauen wir uns die ODER-Verknüpfung aus diesem Beispiel genauer an:

  // Beispiel für ODER-Verknüpfung
   if (Zeichen == 'a' || Zeichen == 'A')
  {
    printf("Gut gemacht!\n\n");
  }
  else
  {
    printf("Das war leider die falsche Taste.\n\n");
    Fehler++;
  }

Hier fragen wir ab, ob die Variable "Zeichen" das Zeichen "a" oder das Zeichen "A" enthält. Wenn die Variable "Zeichen" eines dieser beiden Zeichen enthält, ist die Bedingung erfüllt (wahr).

Die zweite Aussagelogische Verknüpfung in diesem Beispiel ist eine UND-Verknüpfung. Das bedeutet, daß beide Aussagen, also die Aussage a<b (a ist kleiner als b) und gleichzeitig a+b<10 (a plus b ist kleiner als 10), erfüllt sein müssen, damit die Bedingung erfüllt (wahr) ist.

  // Beispiel für UND-Verknüpfung
   if (a<b && a+b<10)
  {
    printf("Sehr gut!\n\n");
  }
  else
  {
    printf("Das war leider falsch.\n\n");
    Fehler++;
  }

Als weiteres Beispiel wollen wir eine Funktion besprechen, die prüft, ob ein gegebenes Jahr ein Schaltjahr ist.

Ein Jahr ist ein Schaltjahr, wenn

  • Die Jahreszahl ohne Rest durch 4 und gleichzeitig nicht ohne Rest durch 100 teilbar ist.
  • die Jahreszahl ohne Rest durch 400 teilbar ist.

Damit der Code übersichtlich bleibt, führen wir 3 Hilfsvariablen vom Typ BOOL ein. Der Datentyp BOOL ist in der Amiga Include Datei exec/types.h definiert. Deshalb müssen wir diese mit #include <exec/types.h> einbinden. Eine boolsche Variable kann entweder den Wert TRUE (wahr) oder den Wert FALSE (falsch) annehmen. Wir teilen unsere Bedingungen auf und weisen das Ergebnis den Hilfsvariablen zu:

Bed1:
Die Jahreszahl ist durch 4 ohne Rest teilbar
Bed2:
Die Jahreszahl ist nicht durch 100 ohne Rest teilbar
Bed3:
Die Jahreszahl ist ohne Rest durch 400 teilbar

Den Rest einer Division erhalten wir - wie in der Lektion "arithmetische Operatoren" besprochen - mit dem Modulo Operator %.

BOOL Schaltjahr (int Jahr)
{
  BOOL Bed1, Bed2, Bed3;

  Bed1 = Jahr % 4 == 0;
  Bed2 = Jahr % 100 != 0;
  Bed3 = Jahr % 400 == 0;

  return (Bed1 && Bed2) || Bed3;
}

Der Rückgabewert der Funktion Schaltjahr ist vom Typ BOOL, kann also den Wert TRUE oder FALSE annehmen. Ist ein Jahr ein Schaltjahr, dann gibt die Funktion TRUE zurück, ansonsten FALSE. Damit ein Jahr ein Schaltjahr ist, müssen Bedingung 1 und Bedingung 2 gleichzeitig erfüllt sein oder es muß Bedingung 3 erfüllt sein. Dies drücken wir mit (Bed1 && Bed2) || Bed3 aus.

Um einen Überblick zu erhalten, welche Fälle auftreten können, schauen wir uns die zu unserer Funktion Schaltjahr zugehörige Wahrheitstabelle an. Da wir 3 Bedingungen verknüpfen, können insgesamt 2 hoch 3 - also 8 Fälle auftreten:

Wahrheitstabelle der Funktion "Schaltjahr"
Bed1 Bed2 Bed3 Schaltjahr
FALSE FALSE FALSE FALSE
FALSE FALSE TRUE TRUE
FALSE TRUE FALSE FALSE
FALSE TRUE TRUE TRUE
TRUE FALSE FALSE FALSE
TRUE FALSE TRUE TRUE
TRUE TRUE FALSE TRUE
TRUE TRUE TRUE TRUE

Beim Jahr 2000 z.B. trifft Bedingung 1 zu, Bedingung 2 nicht und Bedingung 3 trifft zu. Das Jahr 2000 ist also ein Schaltjahr, weil die Jahreszahl ohne Rest durch 4 teilbar ist. Dies entspricht der 6. Zeile unserer Wahrheitstabelle.

In Listing 1.11.2 testen wir unsere Funktion Schaltjahr:

/*   Listing 1.11.2
 *   Schaltjahr.c
 */

#include <stdio.h>
#include <exec/types.h>

BOOL Schaltjahr (int Jahr)
{
  BOOL Bed1, Bed2, Bed3;

  Bed1 = Jahr % 4 == 0;
  Bed2 = Jahr % 100 != 0;
  Bed3 = Jahr % 400 == 0;

  return (Bed1 && Bed2) || Bed3;
}


int main(void)
{
  int J;

  printf("Jahr? ");
  scanf("%d",&J);

  printf("Das Jahr %d ist ",J);

  if (Schaltjahr(J))
  {
    printf("ein ");
  }
  else
  {
    printf("kein ");
  }

  printf("Schaltjahr.\n");

  return 0;
}

Neben den Aussagelogischen Operatoren gibt es in C auch noch Bit-Operatoren, die Variablen (oder Speicherstellen) Bitweise vergleichen. Diese Bitoperatoren betrachten wir hier nicht.
