Schleifen mit for


Oft möchte man, daß eine oder mehrere Anweisungen mehrfach hintereinander ausgeführt werden. Dazu benutzt man Schleifen. In C gibt es while-, do-while und for-Schleifen.

In diesem Abschnitt schauen wir uns die for-Schleife an.

Der Syntax einer for-Schleife ist:

  for([Ausdruck1]; [Ausdruck2]; [Ausdruck3]) Anweisung

Ausdruck1 und Ausdruck3 sind beliebige Ausdrücke. Ausdruck2 ist die Laufbedingung. Die for-Schleife wird solange fortgesetzt, wie die Laufbedingung erfüllt ist.

/*  Listing 1.7
 *  Schleifen mit for
 */
#include <stdio.h>

int main(void)
{
  int i,max;

  printf("Wie oft? ");
  scanf("%d",&max);

  /*   Schleife wird forgestzt, solange der Zähler i
   *   kleiner oder gleich max ist. Dabei wird i in jedem
   *   Durchlauf mit i++ um 1 erhöht.
   */
  for (i=1;i<=max;i++)
  {
    printf("%d. Ich darf den Lehrer nicht ärgern.\n",i);
  }

  return 0;
}

In Listing 1.7 ist die Laufbedingung (i<=max), d.h. i kleiner oder gleich max. Geben wir zum Beispiel für max den Wert 10 ein, so wird die for-Schleife genau 10 mal durchlaufen. Da wir in Ausdruck3 i++ angegeben haben, wird bei jedem Durchlauf die Variable i um 1 erhöht. Beim 11. Durchlauf wird i=11, wodurch die Laufbedingung (i<=max) nicht mehr erfüllt ist. Die Schleife bricht dann ab.

Für Anweisung haben wir hier einen Anweisungsblock verwendet. Alle Anweisungen in einem solchen Anweisungsblock gehören zur jeweiligen for-Schleife. Da wir in Listing 1.7 aber nur eine Anweisung in der for-Schleife wiederholen möchten, hätten wir auch auf einen Anweisungsblock verzichten und die Anweisung direkt hinschreiben können:

  for (i=1;i<=max;i++)
    printf("%d. Ich darf den Lehrer nicht ärgern.\n",i);

Ein häufiger Anfängerfehler ist folgender:

  // Hier hat die for-Schleife keine Anweisung!
  for (i=1;i<=max;i++);
    printf("%d. Ich darf den Lehrer nicht ärgern.\n",i);

Wenn man direkt nach der Laufbedingung ein Semikolon setzt, hat die for-Schleife keine Anweisung. In diesem Fall würde die Schleife zwar max mal durchlaufen, aber der Text würde nur einmal ausgegeben.

Auch verschachtelte Schleifen sind möglich:

/*  Listing 1.8
 *  Verschachtelte Schleifen mit for
 *  Das kleine Einmaleins
 */

#include <stdio.h>

int main(void)
{
  int a,b;

  printf("Das kleine Einmaleins\n\n");

  for (a=1;a<=10;a++)  // Schleife a
   {
    for (b=1;b<=10;b++)  // Schleife b
     {
      printf("%d * %d = %d\n",b,a,a*b);
    } // Ende der Schleife b
    printf("\n"); // Zeilenumbruch ausgeben
  } // Ende der Schleife a
  return 0;
}

Beim ersten Durchlauf der Schleife a wird Schleife b 10 mal wiederholt, beim zweiten Durchlauf von Schleife a weitere 10 mal und so weiter.
