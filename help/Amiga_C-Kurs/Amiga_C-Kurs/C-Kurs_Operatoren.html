Operatoren


Die arithmetischen Operatoren
Operator Bedeutung Beispiel Ergebnis
* Multiplikation x*y Produkt von x und y
/ Division x/y Quotient aus x und y
% Modulodivision x%y Rest der Division aus x und y
+ Addition x+y Summe von x und y
- Subtraktion x-y Differenz von x und y
+ (unär) Plus-Vorzeichen +x Wert von x
- (unär) Minus-Vorzeichen -x Arithmetische Negation von x
++ Inkrement ++x
x++		 x wird inkrementiert (x=x+1), und zwar bei ++x vor, bei x++ nach der Verwendung von x
-- Dekrement --x
x--		 x wird dekrementiert (x=x-1), und zwar bei --x vor, bei x-- nach der Verwendung von x

Zuweisungsoperatoren
Operator Bedeutung Beispiel Ergebnis
= Einfache Zuweisung x=y an x wird der Wert von y zugewiesen
op= Zusammengesetzte Zuweisung x+=y x op= y ist äquivalent mit x = x op (y) (op ist ein binärer arithmetischer oder binärer Operator)

Vergleichsoperatoren
Operator Bedeutung Beispiel Ergebnis: 1 (wahr) oder 0 (falsch)
< kleiner x<y 1, falls x kleiner y
<= kleiner gleich x<=y 1, falls x kleiner oder gleich y
> größer x>y 1, falls x größer y
>= größer gleich x>=y 1, falls x größer oder gleich y
== gleich x==y 1, falls x gleich y
!= ungleich x!=y 1, falls x ungleich y

Logische Operatoren
Operator Bedeutung Beispiel Ergebnis: 1 (wahr) oder 0 (falsch)
&& logisches UND x&&y 1, falls x und y ungleich 0
|| logisches ODER x||y 1, falls x oder y oder beide ungleich 0
! logisches NICHT !x 1, falls x gleich 0,
In anderen Fällen ist das Ergebnis 0.

Bit Operatoren
Operator Bedeutung Beispiel Ergebnis (für jede Bit-Position)
& bitweise UND x & y 1, falls in x und in y eine 1
| bitweise ODER x | y 1, falls in x oder in y oder beiden eine 1
^ bitweise Exklusiv-ODER x ^ y 1, falls entweder in x oder in y eine 1, sonst 0
~ bitweise NICHT ~x 1, falls in x eine 0
<< Links-Shift x << y x wird um y Bit-Positionen nach links verschoben.
>> Rechts-Shift x >> y x wird um y Bit-Positionen nach rechts verschoben.

Operatoren für Speicherzugriff
Operator Bedeutung Beispiel Ergebnis
& Adresse von &x Konstanter Zeiger auf x
* Verweisoperator *p Objekt (oder Funktion), aus das (die) p zeigt
[] Vektorelement x[i] *(x+i): Im Vektor x das Element mit dem Index i
. Auswahl einer Struktur- oder Union-Komponente s.x Komponente x in der Struktur oder Union s
-> Auswahl einer Struktur- oder Union-Komponente p->x Komponente x der Struktur oder Union, auf die p zeigt

Sonstige Operatoren
Operator Bedeutung Beispiel Ergebnis
() Funktionsaufruf pow(x,y) Aufruf einer Funktion
(Typ) explizite Typumwandlung (Cast) (long)x Wert von x mit dem in Klammern angegebenen Typ
sizeof Größe in Byte sizeof(x) Anzahl der Byte, die x belegt
?: Bedingte Bewertung x?y:z y, falls x ungleich 0, sonst z
, Sequenzoperator x,y Zuerst x, dann y auswerten
