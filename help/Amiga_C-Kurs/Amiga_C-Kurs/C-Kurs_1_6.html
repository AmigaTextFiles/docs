Verzweigungen switch


Für Verzweigungen mit vielen Auswahlmöglichkeiten, wie z.B. Menüs, ist die switch-Verzweigung erste Wahl, wie das folgende Programm demonstriert:

/*  Listing 1.6
 *  Verzweigungen switch
 */

#include <stdio.h>

int main(void)
{
  int auswahl;
  char OS_name[80];

  printf("Welches ist Ihr Lieblings Betriebssystem?\n\n");

  printf("1. AmigaOS\n");
  printf("2. Linux\n");
  printf("3. Windows\n");
  printf("4. MacOS\n");
  printf("5. Irix\n");
  printf("6. Solaris\n");
  printf("7. keines der genannten\n\n");

  scanf("%d",&auswahl);

  printf("Ihr Lieblings Betriebssystem ist also ");

  switch (auswahl)
  {
    case 1: printf("AmigaOS.\n");
            printf("Dann sind Sie ja bei diesem Kurs genau richtig.\n");
            break;

    case 2: printf("Linux.\n");
            break;

    case 3: printf("Windows.\n");
            break;

    case 4: printf("MacOS.\n");
            break;

    case 5: printf("Irix.\n");
            break;

    case 6: printf("Solaris.\n");
            break;

    default: printf("keines der genannten.\n");
             printf("Bitte geben Sie den Namen Ihres Betriebssystems ein: ");
             scanf("%s",&OS_name);
             printf("Sie mögen %s am liebsten.\n",OS_name);
  }

  return 0;
}

Dabei wird der switch-Ausdruck (hier: die Variable "auswahl") mit den Konstanten der case-Marken (hier 1-6) verglichen und dann zur passenden Marke verzweigt.

Alle case-Marken, sowie die default-Marke stehen in einem Anweisungsblock, der mit einer geöffneten, geschweiften Klammer ("{") eingeleitet und mit einer geschlossenen, geschweiften Klammer ("}") abgeschlossen wird.

Nach der Verzweigung wird das Programm bis zur nächsten break-Anweisung fortgesetzt. Setzt man die break-Anweisung vor der nächsten case-Marke nicht, so werden auch die folgenden Anweisungen bis zur nächsen break-Anweisung ausgeführt.

Zur default-Marke wird dann gesprungen, wenn der Ausdruck hinter switch mit keiner der case-Marken übereinstimmt. In unserem Beispiel wird z.B. bei der Eingabe einer 9 zur default-Marke gesprungen und der Benutzer wird aufgefordert, den Namen seines Lieblings Betriebssystems einzugeben.
