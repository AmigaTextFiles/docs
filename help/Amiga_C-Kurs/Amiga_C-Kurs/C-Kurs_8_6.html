Images


Statt mit Grafikprimitiven in einen RastPort zu zeichnen, können wir auch kleine Bildchen vordefinieren und diese anschließend in ein Bild "stempeln". Das funktioniert ähnlich wie die Verwendung des Werkzeugs "Pinsel" in einem Malprogramm.

Wie das funktioniert, lernen wir am besten, indem wir uns eine einfache schwarz/weiß (1 Bit Farbtiefe) Grafik anschauen. Das folgende Bild könnte z.B. ein Raumschiff aus einem klassischen Ballerspiel darstellen:

Die grauen Gitterlinien sollen dabei die Matrix-Struktur verdeutlichen. Diese Matrix hat 14 Zeilen und 16 Spalten. Jedes Element der Matrix ist entweder ein schwarzer oder ein weißer Bildpunkt. Diesen Sachverhalt können wir mit Hilfe von Nullen und Einsen darstellen (Binärdarstellung). 0 steht für einen schwarzen, 1 für einen weißen Bildpunkt. Die Binärdarstellung der ersten Zeile des obigen Bildes wäre demnach:

0000001111000000

Mit etwas Fantasie kann man aus der Binärdarstellung immernoch das ursprüngliche Bild erkennen. In der Regel benutzt man aber die Hexadezimaldarstellung. Dabei interpretiert man jede Zeile der Matrix als Zahlenwert.

Die erste Zeile wäre als Dezimalzahl geschrieben also 960 - das entspricht der Hexadezimaldarstellung 0x03C0.


Die einzelnen Zeilen der Matrix tragen wir nun in ein Array vom Typ UWORD ein. UWORD ist in der AmigaOS Header-Datei exec/types.h als Synonym für unsigned short definiert - also ein vorzeichenloser 16 Bit Typ. Die Breite unseres Bildchens wurde mit Bedacht gewählt: Da das Bild genau 16 Pixel breit ist, können wir jede Zeile der Bildmatrix in ein Element unseres Arrays eintragen. Da unsere Matrix 14 Zeilen hat, muß auch unser Array 14 Elemente haben.

UWORD UfoData[14] =
{
    0x03C0,0x0FF0,0x1FF8,0x3FFC,0x524A,0x9249,0xFFFF,0x7FFE,
    0x318C,0x1FF8,0x0FF0,0x13C8,0x2004,0xC003
};

Zusätzlich müssen wir eine Variable vom Strukturtyp Image deklarieren und ausfüllen. Der Strukturtyp Image ist in intuition/intuition.h definiert.

/* Image-Struktur Ufo */
 struct Image Ufo =
{
  0, 0,           // linker Rand, rechter Rand
   16, 14, 2,      // Höhe, Breite, Farbtiefe
   UfoData,        // Bilddaten
   0x0001, 0x0000, // PlanePick, PlaneOnOff
   NULL            // Zeiger auf nächstes Bild
 };

Das Array UfoData mit den Bilddaten, sowie die Image Struktur Ufo schreiben wir in eine Header-Datei namens Ufo.h:

/*  Ufo.h
 *  Image Daten für das Ufo-Bild
 */
/* Bilddaten */
 UWORD UfoData[14] =
{
    0x03C0,0x0FF0,0x1FF8,0x3FFC,0x524A,0x9249,0xFFFF,0x7FFE,
    0x318C,0x1FF8,0x0FF0,0x13C8,0x2004,0xC003
};

/* Image-Struktur Ufo */
 struct Image Ufo =
{
  0, 0,           // linker Rand, rechter Rand
   16, 14, 2,      // Höhe, Breite, Farbtiefe
   UfoData,        // Bilddaten
   0x0001, 0x0000, // PlanePick, PlaneOnOff
   NULL            // Zeiger auf nächstes Bild
 };

Wer es sich einfach machen will, kann auch das Malprogramm "Personal Paint" benutzen, um damit Grafiken als C-Source zu speichern.

Mit Hilfe der Funktion DrawImage können wir unser Ufo-Bildchen in einen RastPort rp stempeln:

  DrawImage(rp, &Ufo,x,y);

x und y sind dabei die Offsets, d.h. der Abstand zwischen dem Nullpunkt des Koordinatensystems und dem linken, oberen Eckpunkt unseres Bildchens.


Das folgende Beispielprogramm zeichnet eine ganze Armada von Raumschiffen in ein Fenster:

/*  Listing 8.6.1
 *  Images
 */
#include <exec/types.h>
#include <exec/exec.h>
#include <intuition/intuition.h>
#include <graphics/gfx.h>
#include <dos/dos.h>

#include <proto/intuition.h>
#include <proto/graphics.h>
#include <proto/exec.h>

/* Ufo Image-Daten */
 #include "Ufo.h"

// Symbolische Konstanten
 #define WIDTH 384   // Breite des Fensters
 #define HEIGHT 204  // Höhe des Fensters
struct Window *Fenster;               // Zeiger auf Window-Struktur
 struct IntuitionBase *IntuitionBase;  // Zeiger auf IntuitionBase-Struktur
 struct GfxBase *GfxBase;              // Zeiger auf GfxBase-Struktur
 struct RastPort *rp;                  // Zeiger auf RastPort-Struktur

int main(void)
{
  int x,y;

  // Intuition Library öffnen
   IntuitionBase = (struct IntuitionBase *) OpenLibrary("intuition.library",36L);

  if (IntuitionBase != NULL)
  {
    // Graphics Library öffnen
     GfxBase = (struct GfxBase *) OpenLibrary("graphics.library",0L);

    if (GfxBase != NULL)
    {

      // Fenster mittels Tags öffnen
       Fenster = OpenWindowTags(NULL,
                               WA_Left, 100,                   // Abstand vom linken Rand
                                WA_Top, 100,                    // Abstand vom oberen Rand
                                WA_Width, WIDTH,                // Breite
                                WA_Height, HEIGHT,              // Höhe
                                WA_Title, "Images",             // Fenstertitel
                                WA_CloseGadget, TRUE,           // Close-Gadget
                                WA_DragBar, TRUE,               // Ziehleiste
                                WA_DepthGadget, TRUE,           // Depth-Gadget
                                WA_GimmeZeroZero, TRUE,         // Ursprung 0/0
                                WA_IDCMP, IDCMP_CLOSEWINDOW,
                               WA_Activate, TRUE,              // Fenster aktivieren
                                TAG_DONE);

      if (Fenster != NULL)
      {

        rp = Fenster->RPort;  // rp zeigt auf RastPort des Fensters
        SetRast(rp, 0L);      // Fensterinhalt löschen
        for(x=0; x<=15; x++)  // Schleife für Spalten
         {
          for(y=0; y<=7; y++) // Schleife für Zeilen
           {
            // Bild in RastPort stempeln
             DrawImage(rp, &Ufo,24*x,24*y);
          }
        }
        // Auf Close-Gadget warten
         Wait(1L << Fenster->UserPort->mp_SigBit);

        CloseWindow(Fenster);   // Fenster schließen
       } // end if
    } // end if
  } // end if
  // Libraries schließen
   if (GfxBase != NULL) CloseLibrary((struct Library *)GfxBase);
  if (IntuitionBase != NULL) CloseLibrary((struct Library *)IntuitionBase);

  return 0;

}
