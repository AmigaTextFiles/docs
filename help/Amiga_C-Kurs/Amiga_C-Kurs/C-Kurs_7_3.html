Elemente eines Fensters


Das folgende Bild zeigt ein Intuition-Fenster mit allen Gadgets:


Welche Gadgets ein Fenster hat, kann man mit Hilfe der Fensterattribute beim Öffnen des Fensters festlegen.

Die in der folgenden Tabelle sind wichtigsten Fensterattribute aufgeführt:

Name Mögliche Werte Bedeutung
WA_Left Zahlenwert vom Typ WORD Abstand des Fensters vom linken Bildschirmrand.
WA_Top Zahlenwert vom Typ WORD Abstand des Fensters vom oberen Bildschirmrand.
WA_Width Zahlenwert vom Typ WORD Breite des Fensters
WA_Height Zahlenwert vom Typ WORD Höhe des Fensters
WA_MinWidth Zahlenwert vom Typ WORD Minimale Breite des Fensters
WA_MinHeight Zahlenwert vom Typ WORD Minimale Höhe des Fensters
WA_IDCMP IDCMP_SIZEVERIFY
IDCMP_NEWSIZE
IDCMP_REFRESHWINDOW
IDCMP_MOUSEBUTTONS
IDCMP_MOUSEMOVE
IDCMP_GADGETDOWN
IDCMP_GADGETUP
IDCMP_REQSET
IDCMP_MENUPICK
IDCMP_CLOSEWINDOW
IDCMP_RAWKEY
IDCMP_REQVERIFY
IDCMP_REQCLEAR
IDCMP_MENUVERIFY
IDCMP_NEWPREFS
IDCMP_DISKINSERTED
IDCMP_DISKREMOVED
IDCMP_WBENCHMESSAGE
IDCMP_ACTIVEWINDOW
IDCMP_INACTIVEWINDOW
IDCMP_DELTAMOVE
IDCMP_VANILLAKEY
IDCMP_INTUITICKS
IDCMP_IDCMPUPDATE
IDCMP_MENUHELP
IDCMP_CHANGEWINDOW
IDCMP_GADGETHELP
 IDCMP (Intuition Direct Communications Message Port) Flags. Wenn ein solcher IDCMP Flag gesetzt ist, kann das Programm auf dem entsprechenden Message Port lauschen, ob dort eine Nachricht von Intuition ankommt.
WA_MaxWidth Zahlenwert vom Typ UWORD Maximale Breite des Fensters
WA_MaxHeight Zahlenwert vom Typ UWORD Maximale Höhe des Fensters
WA_Title String Text, der in der Titelzeile des Fensters angezeigt wird.
WA_ScreenTitle String Text, der in der Titelzeile des Screens angezeigt wird wenn das Fenster aktiviert ist.
WA_SizeGadget TRUE oder FALSE Size Gadget
WA_DragBar TRUE oder FALSE Ziehleiste
WA_DepthGadget TRUE oder FALSE Depth Gadget
WA_CloseGadget TRUE oder FALSE Close Gadget
WA_Backdrop TRUE oder FALSE Wenn WA_Backdrop TRUE ist, ist das Fenster ein Backdrop Window. Das bedeutet, daß das Fenster - unter allen anderen Fenstern - in den Hintergrund tritt. Ein Backdrop Window läßt sich nicht nach vorne holen.
WA_Borderless TRUE oder FALSE Wenn WA_Borderless auf TRUE gesetzt ist, wird das Fenster rahmenlos dargestellt.
WA_Activate TRUE oder FALSE Wenn WA_Activate auf TRUE gesetzt ist, wird das Fenster aktiviert.
WA_GimmeZeroZero TRUE oder FALSE Wenn WA_GimmeZeroZero auf TRUE gesetzt wird, wandert der Ursprung des fenstereigenen Koordinatensystems vom linken, oberen Eckpunkt des Fensters zum linken, oberen Eckpunkt der Zeichenfläche des Fensters - direkt unter der Titelleiste. Das verhindert das Übermalen der Titelleiste und Fenster Gadgets.

WORD ist in der AmigaOS Header Datei exec/types.h als Synonym für short definiert, kann also theoretisch Werte von -32768 bis 32767 annehmen. In der Praxis machen negative Werte für die Fensterbreite usw. keinen Sinn. UWORD ist ein Synonym für unsigned short.

Ob das Fenster bestimmte Gadgets hat wird davon bestimmt, ob das zugehörige Attribut den Wert TRUE (=wahr) oder FALSE (=falsch) hat. Falls das Fenster ein Size Gadget hat, bekommt es automatisch auch ein Zoom Gadget.

Zum Thema IDCMP Flags gibt es eine extra Lektion.

Das oben abgebildete Fenster läßt sich mit folgendem Aufruf von OpenWindowTags öffnen:

    // Fenster mittels Tags öffnen
     Fenster = OpenWindowTags(NULL,
                             WA_Left, 100,    // Abstand vom linken Rand
                              WA_Top, 100,     // Abstand vom oberen Rand
                              WA_Width, 400,   // Breite
                              WA_Height, 300,  // Höhe
                              WA_Title, "Fenster mit allen Gadgets",  // Fenstertitel
                              WA_CloseGadget, TRUE,              // Close-Gadget
                              WA_IDCMP, IDCMP_CLOSEWINDOW,
                             WA_ScreenTitle, "Screentitel zu meinem Fenster...",
                             WA_DragBar, TRUE,                  // Ziehleiste
                              WA_DepthGadget, TRUE,              // Depth-Gadget
                              WA_SizeGadget, TRUE,               // Size-Gadget
                              WA_Activate, TRUE,                 // Fenster aktivieren
                              WA_MinWidth, 100,                  // minimale Breite
                              WA_MinHeight, 100,                 // minimale Höhe
                              TAG_DONE);
