Zeiger und Adressen


In diesem Abschnitt werden wir uns mit Zeigern (Pointer) und Adressen beschäftigen. Das Konzept der Zeiger ist für Anfänger wahrscheinlich etwas schwer vorstellbar, deshalb versuche ich es einmal mit einer bildhaften Beschreibung.

Stellen Sie sich einmal ein fiktives, automatisches Lagersystem vor, welches aus folgenden Komponenten besteht: Einer Lagerhalle (dem Speicher), welche in Planquadrate unterteilt ist (unsere Adressen), Kisten (unsere Bezeichner für Variablen und Funktionen), einem Roboter (unsere Zugriffsoperatoren) und einer Fernbedienung für den Roboter mit integriertem Laserpointer (unsere Zeiger). Der Benutzer zeigt mit dem Laser (dem Zeiger) auf ein Planquadrat bzw. eine Kiste in der Lagerhalle (eine Adresse im Speicher, bzw. ein Bezeichner). Darauf hin fährt der Roboter auf das gewünschte Planquadrat. Mit Knöpfen an der Fernbedienung (den Operatoren) kann der Benutzer dem Roboter verschiedene Anweisungen wie "hole den Inhalt der Kiste" oder "lege den Gegenstand in dieser Kiste ab" geben.

Automatische Lagersysteme, wie oben beschrieben, existieren auch real. Beispielsweise verwendet ein großes Stuttgarter Kaufhaus in der Sportschuhabteilung ein ähnliches System. Der Verkäufer muß nicht mehr selbst in das Lager gehen, um ein paar Schuhe zu holen. In der Fernbedienung des Roboters ist ein Strichcode-Scanner eingebaut, mit dem der Verkäufer den Barcode der Musterauslage einliest. Der Roboter weiß anhand des Strichcodes, wo sich der gewünschte Artikel im Lager befindet und holt diesen.

Ein Zeiger verweist auf eine Speicherstelle und gibt gleichzeitig an, wie diese Speicherstelle zu interpretieren ist. Je nach Datentyp spricht man von Zeigern auf int, char usw.

In der Programmiersprache C benutzen wir den Operator &, um die Adresse einer Variablen im Speicher zu ermitteln. Auf das Roboter-Lagersystem übertragen, wäre dies ein Befehl der Form "sag mir, wo dieser Artikel im Lager abgelegt ist".

Eine Zeigervariable wird mit einem * vor der Bezeichnung deklariert. Sie können einer Zeigervariablen einen beliebigen Namen gemäß der in C geltenden Regeln für Bezeichner geben. Häufig sieht man in Sourcecodes die Bezeichnung *p oder *ptr. In unserem Beispiel verwenden wir als Bezeichner *Zeiger. Mit int *Zeiger; deklarieren wir eine Zeigervariable auf int (oder kurz: einen int-Zeiger) Namens Zeiger.

Listing 4.1 demonstriert die Verwendung von Zeigern und Adressoperatoren:

/*  Listing 4.1
 *  Zeiger und Adressen
 */
#include <stdio.h>

int main(void)
{
  int a,b;      // Zwei Variablen a und b
   int *Zeiger;  // Ein Zeiger vom Typ int
  a=3;
  b=5;

  Zeiger = &a;  // Zeiger zeigt jetzt auf a
  printf("Die Variable a befindet sich auf Speicherplatz ");
  printf("%x.\nDer Inhalt ist %d\n",&a,*Zeiger);

  Zeiger = &b;  // Zeiger zeigt jetzt auf b
  printf("Die Variable b befindet sich auf Speicherplatz ");
  printf("%x.\nDer Inhalt ist %d\n",&b,*Zeiger);

  printf("Der Zeiger zeigt im Moment auf b.\n");
  printf("Ich schreibe jetzt den Wert der Variablen a ");
  printf("in die Speicherstelle, auf die der Zeiger ");
  printf("gerade zeigt.\n");

  *Zeiger = a;  // Den Wert der Variable a in die Speicherstelle
                 // schreiben, auf die der Zeiger gerade zeigt.
  printf("Der Wert der Variable b ist jetzt %d.\n",b);

  return 0;

}

Der Ausdruck *Zeiger = a; bewirkt in diesem Beispiel das selbe, wie wenn wir b = a; schreiben würden, denn vor dieser Zeile lassen wir die Zeigervariable Namens Zeiger mit Zeiger = &b; auf die Adresse zeigen, wo die Variable b gespeichert ist.
