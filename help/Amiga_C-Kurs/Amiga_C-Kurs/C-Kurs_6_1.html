Standard-Definitionsdateien


Die Standard-Bibliothek stellt eine Reihe von Funktionen, Datentypen und Makros zur Verfügung.

Die Header-Datei stdio.h haben wir bereits in unserem allerersten Beispiel kennengelernt. Sie stellt eine Reihe von Funktionen zur Ein- und Ausgabe bereit. Wie man eine solche Header-Datei mit der Präprozessor-Direktive #include einbindet, haben wir schon gelernt. Um weitere Funktionen, Datentypen und Makros der Standard-Bibliothek benutzen zu können, müssen wir lediglich die entsprechenden Header-Dateien einbinden.

Nach ANSI-C Standard sind folgende Header-Dateien definiert:

Standard-Definitionsdateien
<assert.h> <ctype.h> <errno.h>
<float.h> <limits.h> <locale.h>
<math.h> <setjmp.h> <signal.h>
<stdarg.h> <stddef.h> <stdio.h>
<stdlib.h> <string.h> <time.h>

Diese Header-Dateien werden bei jedem C-Compiler, der dem ANSI-C Standard entspricht mitgeliefert. Jede dieser Header-Dateien liefert uns Definitionen zu einem bestimmten Aufgabengebiet. Wir wollen hier nicht alle Header-Dateien besprechen, denn dies würde den Rahmen unseres Kurses sprengen. Eine Referenz zu allen Header-Dateien der Standard-Bibliothek finden Sie in einschlägig bekannter Literatur, wie z.B. in "C - kurz & gut" von Peter Prinz und Ulla Kirch-Prinz. Selbst das Standardwerk "Programmieren in C" von Kernighan/Ritchie geht nicht auf alle Includes der Standard-Bibliothek ein.

Die am häufigsten verwendeten Header-Dateien sind:

häufig verwendete Includes
Name Inhalt
stdio.h Standard Ein/Ausgabe
stdlib.h Häufig verwendete Funktionen, wie z.B. rand
string.h Funktionen zur Verarbeitung von Strings
math.h Wurzel, Potenz, trigonometrische Funktionen usw.
time.h Datentypen und Funktionen für Zeit/Datum

Beachten Sie bitte, daß die Header-Datei locale.h nichts mit der Amiga-spezifischen Locale.library zu tun hat.

Um in unserem Kurs schnell voranzukommen, werden wir auf Details dieser Header-Dateien erst bei Bedarf eingehen.

Es folgt ein kleines Beispiel für die Verwendung mathematischer Funktionen aus math.h.

In Listing 6.1 wollen wir die Nullstellen eines Polynoms 2-ten Grades mit der quadratischen Gleichung (auch bekannt als "Mitternachtsformel") finden.


Zum Potenzieren benutzen wir die Funktion pow, und zum Ziehen der (reellen) Quadratwurzel die Funktion sqrt, die beide in der Header-Datei math.h definiert sind. Die Ein/Ausgabefunktionen scanf und printf sind in stdio.h definiert.

/*  Listing 6.1
 *  Quadratische Gleichung
 */
#include <stdio.h>
#include <math.h>

int main(void)
{
  float p,q;
  float Radikant,Wurzel;
  float x1,x2;

  printf("Programm zur Lösung der quadratischen Gleichung.\n\n");
  printf("Berechnet die Nullstellen eines Polynoms ");
  printf("2-ten Grades der Form x^2+px+q=0.\n\n");

  printf("p: ");
  scanf("%f",&p);
  printf("q: ");
  scanf("%f",&q);

  printf("\nx^2+%fx+%f=0\n\n",p,q);

  /*  Radikant berechnen:
   *  pow ist in math.h definiert
   *  und liefert Potenzen (x hoch y).
   *  pow(p,2) liefert das Quadrat von p.
   */
   Radikant = (pow(p,2)/4.0)-q;

  // Fallunterscheidung
  // Radikant negativ
   // -> keine reelle Nullstelle
   if (Radikant < 0.0)
  {
    printf("Keine reelle Nullstelle.\n");
  }
  // Radikant = 0
   // -> Doppelte Nullstelle
   else if (Radikant == 0)
  {
    x1 = -p/2.0;
    x2 = x1;
    printf("Doppelte Nullstelle bei %f.\n",x1);
  }
  // Radikant größer 0
   // -> zwei verschiedene Nullstellen
   else
  {
    /*  sqrt ist in math.h definiert
     *  und liefert die Quadratwurzel
     */
     Wurzel = sqrt(Radikant);
    x1 = -p/2.0 + Wurzel;
    x2 = -p/2.0 - Wurzel;
    printf("Nullstellen bei %f und %f.\n",x1,x2);
  }

  return 0;

}
