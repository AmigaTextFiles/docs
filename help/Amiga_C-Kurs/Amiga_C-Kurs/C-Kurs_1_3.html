Formatierte Textausgabe mit printf


In unserem HelloWorld ist uns schon die Funktion printf, die in der Header-Datei stdio.h definiert ist, begegnet. Es wurde als Argument der String "Hello, World!" übergeben.

Die Funktion printf ermöglicht formatierte Textausgabe und wird wie folgt verwendet:

int printf(const char *form, ... /*arg1, ...,argn*/ );

printf gibt den Formatstring form auf der Standardausgabe (in der Regel die Console) aus, wobei darin enthaltene Formatelemente durch die Werte aus der Argumentliste arg1, ..., argn ersetzt werden.

In unserem HelloWorld Beispiel hätten wir die Textausgabe auch so formulieren können:

printf("%s","Hello, World!");

In diesem Fall ist der Formatstring "%s", wobei das s für Stringargumente benutzt wird. Nach dem Formatstring folgt, durch Komma getrennt das Argument. In unserem Beispiel ist dies der String "Hello, World!". Der Formatstring wird bei der Ausgabe durch das Argument ersetzt.

Als nächstes schauen wir uns ein kleines, vollständiges Programm an:

/*  Listing 1.1
 *  Einlesen eines Strings von der Standardeingabe
 *  und formatierte Ausgabe.
 */

#include <stdio.h>

int main(void)
{
  char name[80];

  printf("Bitte geben Sie Ihren Namen ein: ");
  gets(name);
  printf("Hallo %s.",name);

  return 0;
}

Wir deklarieren ein Array namens "name" der Größe 80 vom Typ char. Stellen Sie sich der Einfachheit halber ein kariertes Blatt oder ein Formular vor, auf dem wir 80 Kästchen zur Verfügung haben, um unseren Namen einzutragen.

Mit gets(name) holen wir uns den String, den der Benutzer eingibt und schreiben Ihn in die Variable "name". Nachdem der Benutzer den Text eingegeben und die Return-Taste gedrückt hat, steht dieser Text also in der Variable "name".

Nun kommen wir zur formatierten Textausgabe mittels printf. Der Formatstring lautet diesmal "Hallo %s.". Unser Argument ist die Variable "name". Bei der Ausgabe wird der Platzhalter %s durch den Inhalt der Variable "name" ersetzt. Nach %s geht es im Formatstring mit normalem Text weiter, nämlich mit einem Punkt.

Wenn Sie also z.B. als Namen "Max Mustermann" eingeben, gibt unser kleines Programm den Text "Hallo Max Mustermann." aus.

Wir haben jetzt den Platzhalter für Strings kennengelernt. Aber wie sieht es mit anderen Datentypen aus? Schauen wir uns unser nächstes Beispielprogramm an:

/*  Listing 1.2
 *  Ein weiteres Beispiel für formatierte Ausgabe.
 */

#include <stdio.h>

int main(void)
{
  int x,y;
  float rotationsdauer = 23.9345;
  unsigned long z = 3735928559;

  x=5;
  y=10;

  printf("Koordinaten: (%d/%d)\n",x,y);
  printf("Die Rotationsdauer der Erde beträgt %f Erdstunden.\n",rotationsdauer);
  printf("Die Hexadezimaldarstellung von %lu ist %x.\n",z,z);

  return 0;
}

Aus diesem Beispiel lernen wir schon eine Menge. Für Variablen von Typ integer verwenden wir den Platzhalter "%d". Wie wir sehen, können wir auch mehrere Argumente angeben, die dann bei der Ausgabe entsprechend dem Formatstring eingefügt werden. Die Zeichenfolge "\n" ist eine sogenannte Escape-Sequenz. Die Zeichenfolge "\n" wird nicht ausgegeben, sondern bewirkt einen Zeilenumbruch.

Für Fließkomma-Zahlen, wie unsere Variable "rotationsdauer" verwenden wir im Formatstring den Platzhalter "%f".

Unsere Variable z ist von Typ unsigned long. Deshalb schreiben wir, wenn wir diese Variable in Dezimaldarstellung ausgeben wollen im Formatstring "%lu". "%u" wird benutzt, wenn wir einen Datentyp ohne Vorzeichen (= unsigned) in Dezimaldarstellung ausgeben wollen. Für long-Typen schreiben wir noch ein "l" davor.

Wenn wir "%x" im Formatstring verwenden, wird das entsprechende Argument in Hexadezimaldarstellung ausgegeben.

Wie Formatstrings für andere Datentypen aussehen müßen, entnehmen Sie der Tabelle.

Typangaben bei der formatierten Ausgabe
Typ Argument-Typ Ausgabe
d, i int dezimal
u unsigned int dezimal
o unsigned int oktal
x unsigned int hexadezimal mit a,b,c,d,e,f
X unsigned int hexadezimal mit A,B,C,D,E,F
f float/double Gleitkommazahl, dezimal
e, E float/double Exponentialdarstellung, dezimal
a, A float/double Exponentialdarstellung, hexadezimal
g, G float/double Gleitpunktzahl/Exponentialdarstellung, was kürzer ist.
c char/int einzelnes Zeichen
s String Ausgabe des Strings bis '\0'.
N int * Speichert im entsprechenden Argument die Anzahl der bis dahin ausgegebenen Zeichen
P Zeiger Gibt die im Argument enthaltene Adresse in Hexadezimaler Form aus.
% keiner Das Zeichen % wird ausgegeben.
