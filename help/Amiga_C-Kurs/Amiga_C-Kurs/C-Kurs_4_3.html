Zeiger auf Vektoren


Arrays werden auch als Vektoren bezeichnet. Erinnern Sie sich noch an die while-Schleife aus Listing 1.9? Dort sind wir einen String Zeichen für Zeichen durchgegangen:

  // Schleife läuft solange, bis das Zeichen '\0' entdeckt wird.
   while (buffer[i] != '\0')
  {
    /*  i-tes Zeichen aus dem Array buffer plus
     *  Zeilenumbruch ausgeben
     */
    printf("%c\n",buffer[i]);

    i++;  // Zähler um 1 erhöhen
  } // Ende der while-Schleife

Dabei haben wir auf die einzelnen Elemente unseres Arrays buffer mit buffer[0], buffer[1] usw. zugegriffen.

Wir können aber einen Array mit Hilfe eines Zeigers durchlaufen. In Listing 4.3 zählen wir die Länge eines Strings, indem wir das Array mit Hilfe eines Zeigers durchlaufen:

/*  Listing 4.3
 *  Zeiger auf Vektoren
 */
#include <stdio.h>

/*  Funktion gibt Länge eines Strings zurück
 *  Eingabe:  String
 *  Rückgabe: Länge des Strings
 */
 int laenge(char *s)
{
  int n;  // Zählervariable
  /*  Die for-Schleife läuft so lange, bis das Zeichen
   *  '\0' gefunden wird.
   */
   for (n=0; *s !='\0'; s++)
      n++;  // Zähler um 1 erhöhen
  return n;  // Länge des Strings zurückgeben
 }

int main(void)
{
  char Eingabe[80];  // Puffer für maximal 80 Zeichen
  printf("Bitte geben Sie einen Text ein (maximal 80 Zeichen):\n");
  gets(Eingabe);  // String von der Standardeingebe holen
  printf("Länge: %d Zeichen.\n",laenge(Eingabe));

  return 0;

}

Als Eingabeparameter für die Funktion laenge definieren wir mit char *s einen Zeiger auf char. Indem wir der Funktion ein Array (ein String) übergeben, zeigt unser Zeiger s automatisch auf das erste Element (Index 0) dieses Arrays.

Wie bei unserer while-Schleife läuft auch unsere for-Schleife solange, bis wir auf das Zeichen '\0' treffen.

Beachten Sie bitte das s++ in der for-Schleife: Wir können auch Zeiger addieren und subtrahieren. In unserer for-Schleife inkrementieren wir mit s++ bei jedem Schleifendurchlauf den Zeiger. Das bedeutet, daß wir bei jedem Durchlauf den Zeiger um eine ein Index-Element nach vorne setzen.

Bei jedem Schleifendurchlauf erhöhen wir zusätzlich unseren Zähler n mit n++ um eins. Der Rückgabewert unserer Funktion laenge ist also die Länge des Strings, auf den unser Zeiger s zeigt.
