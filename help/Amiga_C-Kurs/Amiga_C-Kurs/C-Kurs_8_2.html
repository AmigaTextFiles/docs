Grafikprimitiven


Grafikprimitiven sind einfache grafische Elemente wie z.B. Punkte, Linen, Polygone usw. . Auch komplexe 3D-Grafiken setzen sich aus solchen Grafikprimitiven zusammen. Nachdem wir die einfachste aller Grafikprimitiven - nämlich WritePixel - kennengelernt haben, wollen wir uns nun weitere Grafikprimitiven der graphics.libray anschauen. Es folgt ein tabellarische Übersicht.

Einige Grafikprimitiven der graphics.library
WritePixel

Syntax: 

  Fehler = WritePixel([rp], [px], [py] );

Setzt einen einzelnen Pixel mit den Koordinaten px, py des RastPort rp. Der Rückgabewert -1, falls die angegebenen Koordinaten außerhalb des RastPort liegen, ansonsten 0.

Draw

Syntax: 

  Draw([rp], [px], [py] );

Zeichnet im RastPort rp eine Linie von der aktuellen Position des (unsichtbaren) Grafikcursors zu den Koordinaten px, py. Der Grafikcursor muß zuvor mit der Funktion Move an die Koordinaten des ersten Punktes verschoben werden.

Move

Syntax: 

  Move([rp], [px], [py] );

Bewegt den Grafikcursor des RastPort rp an die Koordinaten px, py.

PolyDraw

Syntax: 

  PolyDraw([rp], [Anzahl], [Zeiger] );

Zeichnet eine Folge von Linien im RastPort rp, deren Koordinaten in einem (eindimensionalen) Koordinatenarray abgelegt sind. Anzahl gibt dabei die Anzahl der Koordinatenpaare an, der Zeiger zeigt auf das erste Element des Koordinatenarrays.

RectFill

Syntax: 

  RectFill([rp], [p1x], [p1y], [p2x], [p2y] );

Zeichnet im RastPort rp ein gefülltes Rechteck. Der Punkt mit den Koordinaten p1x, p1y ist das Linke, obere Eck des Rechtecks. p2x, p2y sind die Koordinaten des Punktes im rechten, unteren Eck. Dabei müssen folgende Ungleichungen gelten: (p1x <= p2x) und (p1y <= p2y).

DrawEllipse

Syntax: 

  DrawEllipse([rp], [cx], [cy], [a], [b] );

Zeichnet im RastPort rp eine Ellipse mit dem Zentrum cx, cy und dem horizontalen Radius a und dem vertikalen Radius b. Dabei müssen a und b jeweils größer 0 sein.


Listing 8.2 demonstriert die oben gezeigten Grafikprimitiven. Wir erzeugen jeweils in einer for-Schleife Zufallswerte für die Koordinaten.

/*   Listing 8.2
 *   Grafikprimitiven der graphics.library
 */
#include <stdlib.h>

#include <exec/types.h>
#include <exec/exec.h>
#include <intuition/intuition.h>
#include <graphics/gfx.h>
#include <dos/dos.h>

#include <proto/intuition.h>
#include <proto/graphics.h>
#include <proto/dos.h>
#include <proto/exec.h>

/*  Makro zur Erzeugung einer Zufallszahl
 *  im Bereich min..max
 */
 #define random(min,max) ((rand() % ((max+1)-min))+min)

// Makro zum finden des Minimums zweier Zahlen a,b
 #define MIN(a,b) ((a) < (b) ? (a) : (b))

// Symbolische Konstanten
 #define WIDTH 400   // Breite des Fensters
 #define HEIGHT 300  // Höhe des Fensters
#define LOOPMAX 200
#define MAXPEN 256

struct Window *Fenster;        // Zeiger auf Window-Struktur
 struct IntuitionBase *IntuitionBase;  // Zeiger auf IntuitionBase-Struktur
 struct GfxBase *GfxBase;        // Zeiger auf GfxBase-Struktur
 struct RastPort *rp;            // Zeiger auf RastPort-Struktur
// Funktionsprototyp
 void ZeichneGrafik(void);

int main(void)
{
  // Intuition Library öffnen
   IntuitionBase = (struct IntuitionBase *) OpenLibrary("intuition.library",36L);

  if (IntuitionBase != NULL)
  {
    // Graphics Library öffnen
     GfxBase = (struct GfxBase *) OpenLibrary("graphics.library",0L);

    if (GfxBase != NULL)
    {

      // Fenster mittels Tags öffnen
       Fenster = OpenWindowTags(NULL,
                               WA_Left, 100,    // Abstand vom linken Rand
                                WA_Top, 100,     // Abstand vom oberen Rand
                                WA_Width, WIDTH,    // Breite
                                WA_Height, HEIGHT,  // Höhe
                                WA_Title, "Grafikprimitiven",         // Fenstertitel
                                WA_CloseGadget, TRUE,           // Close-Gadget
                                WA_DragBar, TRUE,               // Ziehleiste
                                WA_DepthGadget, TRUE,           // Depth-Gadget
                                WA_GimmeZeroZero, TRUE,         // Ursprung 0/0
                                WA_IDCMP, IDCMP_CLOSEWINDOW,
                               WA_Activate, TRUE,              // Fenster aktivieren
                                TAG_DONE);

      if (Fenster != NULL)
      {

        rp = Fenster->RPort;  // rp zeigt auf RastPort des Fensters
        SetRast(rp, 1L);      // Fensterinhalt löschen
        ZeichneGrafik();

        // Auf Close-Gadget warten
         Wait(1L << Fenster->UserPort->mp_SigBit);

        CloseWindow(Fenster);   // Fenster schließen
       } // end if
    } // end if
  } // end if
  // Libraries schließen
   if (GfxBase != NULL) CloseLibrary((struct Library *)GfxBase);
  if (IntuitionBase != NULL) CloseLibrary((struct Library *)IntuitionBase);

  return 0;

}

void ZeichneGrafik(void)
{
  int i;
  int x0,y0,x,y,cx,cy,a,b;

  SHORT Feld[LOOPMAX*2];  // Array für Koordinaten
   SHORT *FeldZeiger;      // Zeiger auf Koordinaten-Array
  LONG p;

  // Punkte
   for (i=0;i<LOOPMAX;i++)
  {
    x = random(0,WIDTH-1);
    y = random(0,HEIGHT-1);

    p = (LONG)random(0,MAXPEN);

    SetAPen(rp,p);

    // Amiga-Grafikprimitive
     WritePixel (rp,x, y);
  }

  Delay(150);
  SetRast(rp,1L);

  // Linien
   for (i=0;i<LOOPMAX;i++)
  {
    x0 = random(0,WIDTH-1);
    y0 = random(0,HEIGHT-1);
    x = random(0,WIDTH-1);
    y = random(0,HEIGHT-1);

    p = (LONG)random(0,MAXPEN);

    SetAPen(rp,p);
    Move(rp,x0,y0);  // Grafik-Cursor Bewegen
     // Amiga-Grafikprimitive
     Draw(rp,x, y);
  }

  Delay(150);
  SetRast(rp,1L);

  // Linienzüge mit PolyDraw
  FeldZeiger=Feld;

  for (i=0;i<LOOPMAX;i++)
  {
    // Zufallswerte erzeugen
     x = random(0,WIDTH-1);
    y = random(0,HEIGHT-1);

    // x Koordinate in das Array eintragen
     *FeldZeiger=(SHORT)x;
    FeldZeiger++;

    // y Koordinate in das Array eintragen
     *FeldZeiger=(SHORT)y;
    FeldZeiger++;

  }

  FeldZeiger = Feld;
  p = (LONG)random(0,MAXPEN);
  SetAPen(rp,2L);

  // Amiga Grafik-Primitive
   PolyDraw(rp,LOOPMAX, FeldZeiger);

  Delay(150);
  SetRast(rp,1L);

  // gefüllte Rechtecke
   for (i=0;i<LOOPMAX;i++)
  {
    x0 = random(0,WIDTH-1);
    y0 = random(0,HEIGHT-1);
    x = random(0,WIDTH-1);
    y = random(0,HEIGHT-1);

    p = (LONG)random(0,MAXPEN);

   if (x >= x0 && y >= y0)
   {

    SetAPen(rp,p);

    // Amiga-Grafikprimitive
     RectFill(rp,x0,y0,x,y);
   }

  }

  Delay(150);
  SetRast(rp,1L);

  // Ellipsen
   for (i=0;i<LOOPMAX;i++)
  {
    cx = random(0,WIDTH-1);
    cy = random(0,HEIGHT-1);
    a = random(0,MIN(WIDTH-cx,cx));
    b = random(0,MIN(HEIGHT-cy,cy));

    p = (LONG)random(0,MAXPEN);

    SetAPen(rp,p);

    // Amiga-Grafikprimitive
     DrawEllipse(rp,cx,cy,a,b);

  }

}

Die AmigaOS Funktion Delay hält das Programm jeweils für eine kurze Zeit an, damit wir die gerade gezeichnete Grafik betrachten können. Als Eingabeparameter erwartet Delay einen int Wert, der die Zeit in 1/50 Sekunden angibt. So wartet beispielsweise Delay(50) genau eine Sekunde, Delay(150) genau 3 Sekunden usw. .

Wenn Sie wollen können Sie, während das Programm wartet den Mauszeiger auf "busy" setzen, dann erscheint die bekannte Uhr als Mauszeiger. Fügen Sie dazu vor und nach dem Aufruf der Funktion Delay jeweils folgende Zeilen ein:

  // BusyPointer setzen
   SetWindowPointer(Fenster, WA_BusyPointer, TRUE, TAG_DONE);

  // Drei Sekunden warten
   Delay(150);

  // Wieder normaler Mousepointer
   SetWindowPointer(Fenster, TAG_DONE);

Die graphics.library bietet neben den hier gezeigten noch eine Vielzahl weiterer Grafikprimitiven für Polygone, Text usw. . Alle zu besprechen würde den Rahmen dieses Kurses sprengen.
