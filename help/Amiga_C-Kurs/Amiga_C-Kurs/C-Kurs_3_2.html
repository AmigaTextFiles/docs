Synonyme mit typedef


Mit dem Schlüsselwort typedef kann man einem Datentyp einen alternativen Bezeichner (ein Synonym) geben.

Typische Beispiele für die Verwendung von typedef befinden sich in der Amiga-Include-Datei exec/types.h. Hier ein Auschnitt dieser Header-Datei:

typedef long    LONG;       /* signed 32-bit quantity */
 typedef unsigned long   ULONG;      /* unsigned 32-bit quantity */
 typedef unsigned long   LONGBITS;   /* 32 bits manipulated individually */
 typedef short   WORD;       /* signed 16-bit quantity */
 typedef unsigned short  UWORD;      /* unsigned 16-bit quantity */
 typedef unsigned short  WORDBITS;   /* 16 bits manipulated individually */

Bindet man die Header-Datei exec/types.h mit der Präprozessor-Direktive #include ein, so kann beispielsweise man statt unsigned long auch ULONG schreiben.

ULONG Meine_Variable;  // äquivalent mit unsigned long Meine_Variable;


Hier wird eine Variable namens "Meine_Variable" vom Typ unsigned long deklariert. Dabei ist ULONG ein alternativer Bezeichner für den Datentyp unsigned long.

Natürlich können Sie auch selbstdefinierte Bezeichner für Datentypen verwenden.

typedef long int32;  // int32 ist jetzt ein alternativer Bezeichner für long

Den so deklarierten alternativen Bezeichner für long können wir wie folgt benutzen:

int32 Meine_Variable;  // äquivalent mit long Meine_Variable;
