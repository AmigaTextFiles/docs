Q) The mouse still doesn't work under Windows?

A)  Change the "mouse.ini" file in the Windows directory to read
	MouseType=SERIAL2 instead of
	MouseType=PS2


Q) I am using Cybergraphics but I don't get a cursor?

A) Make sure you are running a delayed video update in Advanced Video Options
   Immediate display.

Q) What are the minimum requirements for PC-Task 4.2

A) PCTask requires at least Kickstart 2.0, a 68020, and 2Mb memory.  The
   more memory the better.  In order to use FPU emulation, you will need to
   have an FPU.

Q) Can PC-Task access the ISA (PC-XT/AT) slots inside the Amiga
   2000/3000/4000 ?

A) It is impossible for any Amiga software to access anything plugged into
   the ISA slots without a bridge type card.


Q) Why is the graphics update slow in ... ? 

A) Graphics update speed depends on many things. Like did you have it in
   immediate update or after # of frames and what PC mode was it. Did you run
   it on a native Amiga mode or a card mode (if so what card mode & what RTG
   software is doing the work). All these things are important. On a OCS and
   ECS machines (A500/A600/A2000/A3000) any native mode in 16 colors (the
   default PC text and MS Windows modes) you are stuck with the fact that OCS
   and ECS in hires/4 bitplanes are a complete and utter dog because all chip
   RAM access (for all the display area of the frame) is already taken just
   for the display data fetching. AGA does not have this problem. When using
   graphics card modes OR PC-Task as a window the more recent the AmigaOS the
   faster (V3.1 is faster than V3.0 which is much faster than V2).


Q) How can I get faster graphics update in MS Windows ? 

A) Try changing the update method to update after # of frames (except for 256
   color modes). Try using the VGA mono MS Windows display driver. If using a
   graphics card mode see if the native mode is faster.


Q) I can't get PC-Task to come up as a window ? 

A) Make sure you are setting the specific PC mode to the window option in

Q) The partition I had setup under CrossDOS/CrossPC doesn't seem to work? 

A) You must use a mountlist/DOSDriver entry that has NO CrossDOSFileSystem
   entry.


Q) I have the demo and can't seem to create a harddisk for it ? 

A) Setting up a harddisk requires the use of the MSDOS programs FDisk and
   Format which need to write to the harddisk so you cannont do this with the
   demo.


Q) I tried to set up a real partiton but it doesn't work ? 

A) Did it appear next to Hardrive C/D: on the BIOS screen of PC-Task (the one
   after you press start on the setup screen) ? If not then make sure it is
   being mounted and you correctly entered it with colon on the setup screen.
   If you have to set mask values on your Amiga partitions because you
   controller cannot DMA to your 32bit RAM, see section 2.10 of the manual on
   how to do this for PC-Task partitions. If using a mountlist/DOSDriver
   entry make sure it has NO CrossDOSFileSystem entry.


Q) I want to read the harddisk I setup for PC-Task from AmigaOS ? 

A) Use PCTCrossMount that came with PC-Task V4.0. 


Q) I tried to set up CDROM access but it doesn't work ? 

A) Follow section 5.8 of the manual. Make sure you are using PC-Task V3.1 and
   PCTCDROM.SYS that came with it (not some old version). If you have to set
   mask values on your Amiga partitions because you controller cannot DMA to
   your 32bit RAM, see section 2.10 of the manual on how to do this for
   PC-Task. If it still doesn't work contact tech support.
