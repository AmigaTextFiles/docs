Note: It look like an ISA-Bus, but it is a 100 pin connector.
An A1200-Zorroboard is a Zorro-II Board for the A1200 CPU-port;
it has the same pinout like the A2000 Zorro, but no DMA.
Zorro is autoconfig! Adresselimit is at 16 MB with Zorro II.
